# 第十六章　残疾人社会工作服务

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_104_0.png) 向督导老张询问如何与残疾人建立专业关系。老张提供的下列建议中，符合与残疾人建立专业关系指导原则的是（ ）。

**选项：**
```
A.同情残疾人的处境
B.避免当面讨论残疾问题
C.为残疾人提供建议
D.设身处地地理解残疾人感受
```

> **正确答案：** D

**解析：**
社会工作者在与残疾人建立专业关系时，需要遵循一定的指导原则。选项A中的“同情残疾人的处境”虽然表达了对残疾人的关心，但同情可能带有居高临下的态度，不利于建立平等的专业关系。选项B中的“避免当面讨论残疾问题”可能忽视了残疾人的需求和意愿，不利于真正了解他们的感受和需求。选项C中的“为残疾人提供建议”可能过于主动，没有充分考虑残疾人的自主性和决策权。而选项D中的“设身处地地理解残疾人感受”强调了共情和理解，尊重残疾人的主体地位，符合专业关系建立的原则。因此，正确答案是D。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_104_0.png) 应邀到小伟家进行探访。老李了解到，7岁的小伟去年被诊断为中度智障，未能到普通小学就读。小伟母亲辞职在家照顾小伟，但她觉得自己只能照顾其生活，无法为他提供系统的知识和社会技能训练，因而非常担心影响小伟的顺利成长。老李评估后认为，小伟当下最急需的是（ ）康复资源。

**选项：**
```
A.职业
B.社区
C.教育
D.社会
```

> **正确答案：** C

**解析：**
A、职业：职业康复通常针对的是成年或接近成年的个体，帮助他们获得就业技能，而小伟目前只有7岁，还未到考虑职业康复的阶段。

B、社区：社区康复资源可能包括社区活动、支持小组等，但根据题目描述，小伟的母亲更担心的是知识和社会技能的系统训练，这更偏向于教育方面。
C、教育：教育康复资源包括特殊教育、融合教育等，能够为智障儿童提供适合其发展水平的教育，帮助他们获得知识和社会技能，这正是小伟当前最需要的。
D、社会：社会康复可能涉及社交技能训练、融入社会等方面，但与教育康复相比，教育康复更为直接地满足小伟当前的知识和技能需求。
综上所述，小伟最急需的是教育康复资源，因此正确答案是C。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_104_0.png) 助残疾人全面康复，实现回归社会生活的主流。

**选项：**
```
A.社会康复
B.社区康复
C.职业康复
D.教育康复
```

> **正确答案：** B

**解析：**
社区康复（B选项）通过整合医疗、教育、职业和社会资源，在社区层面为残疾人提供全方位支持，是实现其全面康复和回归社会主流的核心理念。而社会康复（A选项）更偏向宏观政策环境建设，与题干中“全面康复”的具体实践场景不符。因此，正确答案为B。


详细分析：
社区康复强调以社区为依托，整合资源为残疾人提供就近、便利的康复服务，旨在促进其身心功能恢复、社会融入及平等参与社会生活。根据世界卫生组织（WHO）的定义，社区康复是通过社区资源动员，帮助残疾人获得教育、医疗、职业和社会服务，最终实现全面康复和回归社会主流。
社会康复（A选项）：侧重于通过宏观政策、法律保障和社会环境改造消除歧视，为残疾人提供平等参与社会的机会（如无障碍设施建设、反歧视立法）。其作用更多是营造包容性社会氛围，而非直接提供康复服务。
社区康复（B选项）：直接聚焦于在社区层面实施具体的康复服务（如医疗训练、心理疏导、职业技能培训），是残疾人全面康复的实践载体。例如：福建新罗区通过“爱心助残驿站”提供康复理疗、辅具租赁和家庭支持服务，形成“15分钟康复圈”，推动残疾人就近融入社区。
职业康复（C选项）：聚焦于通过技能培训和就业支持帮助残疾人获得经济独立，属于社区康复的一部分，但并非全面康复的核心理念。
教育康复（D选项）：关注残疾人的教育权利与学习能力提升，是康复体系的重要环节，但同样无法涵盖“全面康复”的整体目标。

## 第 6 题 [单选题]

**题目：** 6.68 ![图](./images/q_6_104_0.png) 中风丧失了部分生活自理能力，且身边无子女照顾。社会工作者小马对张阿姨进行了需求评估，发现其需要医护、生活照料、营养以及心理等方面的服务。据此，小马计划协调并整合医生、护士、家政工、营养师、志愿者和心理咨询师等组成跨专业团队，对张阿姨进行全面和持续的服务。小马上述工作采用的方法是（ ）。

**选项：**
```
A.个案工作
B.个案管理
C.社区康复
D.社会康复
```

> **正确答案：** B

**解析：**
A、个案工作：这是一种直接服务的方法，社会工作者与服务对象建立一对一的关系，提供直接的服务和支持。

B、个案管理：这是一种协调和整合服务的方法，社会工作者作为协调者，负责评估服务对象的需求，并协调不同专业资源，为服务对象提供综合性的服务。
C、社区康复：这是一种在社区环境中进行的康复服务，旨在帮助服务对象重新融入社区生活，提高生活质量。
D、社会康复：这是一种更广泛的社会层面的康复服务，可能包括政策倡导、社会支持系统建设等。
根据题目描述，小马的工作重点在于协调和整合不同专业人员和资源，为张阿姨提供全面的服务，这更符合个案管理的定义。而个案工作更侧重于直接服务，社区康复和社
会康复则更侧重于社区层面或社会层面的服务提供。因此，正确答案是B.个案管理。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_104_0.png) 工作的发展，社会工作者常常基于社会建构理念开展服务。下列关于残疾人的观念中体现该理念的是（ ）。

**选项：**
```
A.残疾人的主要特征是个体差异很小
B.残疾人随时随地都需要特别的帮助
C.消除社会障碍是残疾人权利实现的关键
D.分析残疾的原因应该从个人层面入手
```

> **正确答案：** C

**解析：**
自从有人类，就有残疾人。残疾的原因可能是遗传和疾病，也可能是自然灾害、战争和意外事故，还可能是环境污染和科技水平限制等。在残疾人致残的原因分析上应该坚持以社会原因分析为主导，选项D错误。当某个残疾人遇到困难时容易简单地判断是由于残疾引起的。其实，残疾人的个体差异是非常大的，当人们把困难的核心界定为残疾的时候，因为残疾的事实是客观存在的，那就意味着直接否认了残疾人克服困难的可能性，也就否认了作为人具有改变自身现状的潜能，直接否认了残疾人的权利，选项A错误。“残疾人问题”的主要原因是外部障碍(无论是政策还是环境等)，使残疾人处于“不利”的社会境遇中，无法发挥正常作用。残疾人权利的实现和能力的发挥需要消除外部的障碍，政府也应该在立法上制定针对残疾人的保障政策，推进发展残疾人事业。要认识到，残疾对个体的发展而言，既是一种挑战也是一种机遇，选项C正确。历史上，残疾人经常处于与世隔绝的状态。在现代社会中，残疾人与健全人一样，具备良好的社会适应能力，能够发展替代性需要并且相对独立地使用社会性资源。当然，这并不否认残疾人需要特别的帮助，关键问题是要评估残疾人需要“什么”帮助以及在“什么”处境下需要帮助。社会赋予残疾人以同等的发展机会才是最重要的，社会帮助的目标不是让残疾人“依赖”社会，而是建构残疾人的新“能力体系”，选项B错误。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_104_0.png) 十四五”残疾人保障和发展规划》中“强化残疾预防”的任务要求，某社会工作服务机构承接政府购买服务，与社区卫生服务中心合作，在社区开展残疾早期筛查和早期医疗干预服务。上述服务属于我国残疾人预防工作体系中（ ）的内容。

**选项：**
```
A.一级预防
B.二级预防
C.三级预防
D.四级预防
```

> **正确答案：** B

**解析：**
残疾人预防工作体系通常分为三个级别，即一级预防、二级预防和三级预防。根据题目描述，该社会工作服务机构与社区卫生服务中心合作，在社区开展残疾早期筛查和早期医疗干预服务。早期筛查属于二级预防的范畴，因为二级预防旨在早期发现疾病或残疾，及时进行治疗和干预，以防止残疾的发生或减轻残疾的程度。而一级预防主要是针对病因的预防，如遗传咨询、疫苗接种等，三级预防则是针对已经发生的残疾，提供康复服务和辅助器具等。因此，正确答案是B.二级预防。

D.帮助阿泰和有同样遭遇的人士建立互助小组
E.请阿泰同学为其提供学习和情感上的支持
```

> **正确答案：** DE

**解析：**
本题考察社会支持网络中“非正式社会支持”的识别。非正式社会支持主要指来自个人社交网络（如家人、朋友、邻居、同辈群体等）基于情感和人际关系提供的支持。与之相对的是正式社会支持，即由专业机构或制度（如政府、医院、学校、社会组织）基于规章制度提供的服务。
A 属于正式支持。康复机构是提供专业服务的正式组织。
B 属于正式支持。联系学校为其营造环境，是动员正式组织（学校）提供支持。
C 属于正式支持。申请康复补贴是链接政府的社会福利制度资源。
D 属于非正式支持。互助小组是基于共同经历和相互理解形成的同辈支持网络，属于典型的非正式支持。
E 属于非正式支持。同学的帮助是基于同学关系（个人社交网络）提供的情感与学习支持。
因此，属于建立非正式社会支持网络的选项是D和E。

```

> **正确答案：** BCD

小张采用深度访谈、焦点小组和非参与观察等定性研究方法，收集了精神障碍人士及其家属的资料，旨在分析家属参与对康复的作用并提出服务优化方案。
