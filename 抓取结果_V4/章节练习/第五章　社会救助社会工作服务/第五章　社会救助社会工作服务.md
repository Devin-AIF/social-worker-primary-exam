## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_102_0.png) 大多数是“老、弱、病、残、鳏、寡、孤、独”以及收入较低的困难群体，他们的衣食住行要靠政府或者社会给予帮助，要靠政府和社会帮助他们提供基本生活条件、疾病治疗以及办理丧葬事宜等。因此，（ ）是救助对象最基本也最迫切的需求。

**选项：**
```
A.生理需求
B.安全需求
C.尊重需求
D.社交需求
```

> **正确答案：** A

**解析：**
社会救助的对象主要是“老、弱、病、残、鳏、寡、孤、独”以及收入较低的困难群体。这些人群的衣食住行需要依靠政府或社会的帮助，以提供基本生活条件、疾病治疗以及办理丧葬事宜等。因此，他们的最基本且最迫切的需求应该是满足生存和基本生活条件的需求，即生理需求。

生理需求是马斯洛需求层次理论中的最低层次，包括食物、水、住所、衣物等基本生存条件。对于这些困难群体来说，由于自身能力有限或经济条件差，他们无法满足这些基本需求，因此需要政府和社会的帮助来保障他们的生理需求得到满足。而安全需求、尊重需求和社交需求虽然也很重要，但它们建立在生理需求得到满足的基础上。因此，在这种情况下，生理需求是最基本也是最迫切的需求。
综上所述，正确答案是A.生理需求。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_102_0.png) 地震中坍塌，造成多名师生伤亡。宁宁虽成功逃生，但在震后几天一直沉默不语，惊恐不安。在对宁宁进行危机干预的过程中，社会工作者小林采用“讲故事”的方式，引导宁宁讲述逃生的经历和感受，以对其问题和需要进行评估。小林开展的上述工作是为了（ ）。

**选项：**
```
A.定义宁宁的危机问题
B.向宁宁提供持续支持
C.确保宁宁的生命安全
D.讨论宁宁问题的多种可能
```

> **正确答案：** A

**解析：**
危机干预的目标是迅速识别和处理危机情况，以减轻其负面影响。在这个情境中，宁宁经历了地震这一创伤事件，表现出明显的心理应激反应，如沉默不语和惊恐不安，这些都是危机干预需要关注的症状。

小林采用“讲故事”的方式，让宁宁讲述自己的经历和感受，这实际上是一种心理评估的方法。通过这种方式，小林能够更好地了解宁宁的心理状态、创伤经历以及当前的感受，从而为后续的干预措施提供依据。这一步骤主要是为了明确宁宁面临的具体问题和需要，即定义她的危机问题，因此选项A是正确的。
选项B“向宁宁提供持续支持”虽然也是危机干预的一部分，但在这个阶段，小林的主要目的是评估宁宁的问题，而不是立即提供持续支持。选项C“确保宁宁的生命安全”通常是在危机干预的早期阶段，如在紧急情况下确保个体的安全，但在这个情境中，宁宁已经成功逃生，生命安全可能已经得到保障，因此选项C可能不是当前的主要目的。选项D“讨论宁宁问题的多种可能”可能涉及到后续的干预计划，但根据题目描述，小林当前的工作重点是评估宁宁的问题和需要，而不是讨论多种可能的解决方案。
综上所述，小林开展的工作是为了定义宁宁的危机问题，因此正确答案是A。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_102_0.png) 指对需要急救但身份不明或者无力支付急救费用且符合规定的急危重伤病患者，社会救助管理部门给予疾病应急救助。以下选项中符合疾病应急救助的是（    ）。

**选项：**
```
A.将重病中的流浪人员送到医院救治
B.给无力支付医疗费用的失独老人免除医疗费用
C.为社区居民提供公益性体检
D.在联系上正在急救的男孩家人的情况下仍然为男孩免除急救费用
```

> **正确答案：** A

**解析：**
疾病应急救助的适用对象是“需要急救但身份不明或者无力支付急救费用的急危重伤病患者”，核心是针对急危重症的急救阶段费用救助。

选项A：重病中的流浪人员属于身份不明、无力支付费用的急危重症患者，符合疾病应急救助的条件，因此正确。选项B：失独老人免除医疗费用不属于急救阶段的救助，若属于困难群体的常规医疗救助，应纳入医疗救助范畴，而非疾病应急救助。选项C：公益性体检属于公共卫生服务，与应急救助无关。选项D：已联系上家人的男孩，其急救费用应由家人承担，不符合应急救助条件。
流浪人员救助属于社会救助中的流浪乞讨人员救助范畴，其中急救阶段的费用由疾病应急救助基金承担。

**题目：** 11. ![图](./images/q_11_102_0.png) 长期失业，靠最低生活保障金生活，与父母关系紧张。社会工作者老马入户了解到陈先生常常感到自己与社会格格不入，烦心的时候找不到人倾诉，十分苦恼。老马对陈先生进行情绪疏导，与就业服务中心联系，为他申请免费职业技能培训，并帮他模拟面试，邀请他参加街道社工站组织的各种活动，结识新朋友。除上述服务外，针对陈先生的需求，老马还应开展的服务是（ ）。

```
A.提供心理支持
B.调节家庭关系
C.开展能力建设
D.促进社会融入
```

**解析：**
题目描述了陈先生的现状：离异、长期失业、靠最低生活保障金生活，与父母关系紧张。社会工作者老马已经为他提供了情绪疏导、就业服务、技能培训、模拟面试、参加社区活动等服务，帮助他结识新朋友，改善社会融入情况。

选项A（提供心理支持）已经在情绪疏导中得到了体现；选项C（开展能力建设）包括了职业技能培训和模拟面试；选项D（促进社会融入）则通过参加社区活动来实现。因此，这些选项中的服务已经被老马提供，而家庭关系的调节是尚未涉及的领域，因此正确答案是B。

## 第 16 题 [单选题]

**题目：** 16. ![图](./images/q_16_102_0.png) 家对符合规定标准的住房困难的最低生活保障家庭、分散供养的特困人员给予住房方面的帮助，满足其基本住房需求。住房救助的形式在农村主要是（ ）。

**选项：**
```
A.发放住房租赁补贴
B.开展危房改造
C.配租公共租赁住房
D.允许“城中村”的出现
```

> **正确答案：** B

**解析：**
住房救助是指国家对符合规定标准的住房困难的最低生活保障家庭、分散供养的特困人员给予住房方面的帮助，以满足其基本住房需求。根据题目中的描述，住房救助的形式在农村主要是开展危房改造。这是因为农村地区的住房条件可能较为简陋，危房改造能够直接改善这些家庭的居住环境，提供安全、舒适的住所。而发放住房租赁补贴、配租公共租赁住房等措施可能更适合城市地区的住房救助，因为城市中的公共租赁住房体系较为完善，租赁市场也更为发达。至于选项D，“允许‘城中村’的出现”，这显然不符合住房救助的宗旨，因为“城中村”往往指的是城市中的城乡结合部，其住房条件可能较差，且这一选项与住房救助的目标相悖。因此，正确答案是B.开展危房改造。

## 第 17 题 [单选题]

**题目：** 17. ![图](./images/q_17_102_0.png) 在走访社区困境家庭时发现，周大哥因病致残，被鉴定为三级肢体残疾，他有两个上学的孩子，家庭重担都落在妻子一人身上。小陈在为周大哥服务过程中，陪同他办理了残疾证，并给他讲解医疗救助政策，帮助他整理申请医疗救助所需材料。小陈的上述做法属于医疗救助社会工作中的（ ）服务。

急难性（C）：指困难情况紧急且严重，如家庭成员突发重大疾病，导致基本生活立即陷入困境，需要立即采取措施缓解。临时性（D）：指困难是暂时的，可能通过一次性救助或其他短期措施解决，而不是长期的经济援助需求。群体性（A）：通常指多个家庭或个人同时面临困难，但题目中的描述更侧重于个体或家庭的突发情况，因此群体性可能不是主要特点。间歇性（E）：指困难情况时有时无，但题目中的情形强调的是基本生活暂时出现严重困难，可能更符合临时性而非间歇性。

综上所述，正确答案是B、C、D。
