# 第十四章　老年社会工作服务

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_119_0.png) ，由媳妇照顾。媳妇和李奶奶以前有很多矛盾，现在照顾李奶奶时经常谩骂她，李奶奶稍有不从，媳妇就威胁她，有时还会阻断李奶奶与外界的接触，不让李奶奶见别人，更不让李奶奶出门。李奶奶遇到的这种虐待属于（ ）。

**选项：**
```
A.身体虐待
B.性虐待
C.心理虐待
D.经济虐待
```

> **正确答案：** C

**解析：**
A、身体虐待通常涉及物理伤害，如打、踢等，而题目中没有提到媳妇对李奶奶有身体上的暴力行为，所以A选项不正确。
B、性虐待是指任何形式的非自愿的性接触或性行为，题目中也没有提到这方面的情况，所以B选项也不正确。
C、心理虐待包括言语上的侮辱、威胁、孤立等行为，这些都符合题目中的描述，所以C选项是正确的。
D、经济虐待涉及控制受害者的财务资源，题目中没有提到媳妇控制李奶奶的经济情况，所以D选项不正确。
综上所述，正确答案是C.心理虐待。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_119_0.png) 完全自理，一直由儿子小王照顾。最近，王大爷经常责骂儿子照顾不周，小王很苦恼，社会工作者安慰小王，并和他一起分析老年人的心理特点，使他更好地理解了王大爷。此案中，社会工作者的工作属于老年社会工作中的（）服务。

**选项：**
```
A.身体健康
B.处理认知与情绪问题
C.精神健康
D.维护社会支持网络
```

> **正确答案：** B

**解析：**
（考点:处理认知与情绪问题） 本题中，社会工作者的核心工作是：通过帮助主要照顾者（小王）理解老年人（王大爷）责骂行为背后的心理特点，来解决因误解产生的家庭情绪问题。
B（处理认知与情绪问题）正确：这项服务直接针对由王大爷的认知（如可能因身体依赖产生的失控感）和情绪（如挫败、焦虑）所引发的家庭矛盾。社会工作者虽未直接服务王大爷，但通过分析和解释老年人典型的心理特点，赋能小王，使其能正确理解并应对父亲的情绪行为，从而在根本上处理和缓解了问题。这是处理老年人认知与情绪问题的一种重要且专业的间接介入方式。
A不正确：未涉及具体的疾病治疗或康复训练。
C不正确：通常指抑郁症、焦虑症等临床诊断的精神障碍，本题中王大爷的行为更接近于由情境引发的心理反应，尚未达到此程度。
D不正确：侧重于链接外部资源（如社区、亲友）构建支持系统，而本题焦点是改变小王个人的认知和理解。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_119_0.png) 力照顾她，儿子将她送至养老院，赵奶奶不希望搬到养老院居住，但又不敢和儿子说。社会工作者了解赵奶奶的想法后，协助赵奶奶和她儿子表达自己的想法，决定自己的住处。上述社会工作者提供的服务，满足了老年人的（ ）。

**选项：**
```
A.知情权
B.自决权
C.居住权
D.隐私权
```

> **正确答案：** B

**解析：**
题目描述的是社会工作者帮助赵奶奶表达自己的想法，并与儿子沟通，最终决定赵奶奶的住处。这里的关键在于赵奶奶有权自主决定自己的居住安排，而不是由儿子单方面决定。社会工作者的做法尊重了赵奶奶的意愿，确保她能够自主选择住处，这正是自决权的体现。自决权意味着个人有权自主做出决定，而不受他人不当干涉。因此，正确答案是B.自决权。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_119_0.png) 特别是失能、失智等有需要的老年人及家庭申请政府与社会资助"是老年社会工作（）方面的服务内容。

**选项：**
```
A.家庭辅导
B.救助服务
C.照顾安排服务
D.适老化环境改造
```

> **正确答案：** D

**解析：**
适老化环境改造：帮助老年人，特别是失能、失智等有需要的老年人及家庭申请政府与社会资助，改造室内照明、防滑措施、安装浴室扶手等，减少老年人跌倒等意外风险。
【易混点提示】
B 救助服务：通常指向老年人提供经济援助、物资帮助或紧急救援，解决基本生存需求（如医疗费、生活费）。
D 适老化环境改造：特指通过改造物理环境（如安装扶手、防滑设施）来提升老年人生活安全，属于专项服务，申请资助是达成环境改造的手段。
题干中“申请政府与社会资助”的目的是用于环境改造（解析明确提到“改造室内照明、防滑措施”等），而非直接经济救助。因此，资助的用途决定了选项归属。在社工服务分类中，同一服务手段（如申请资助）可能归属不同类别，取决于其服务目标。此题将资助用于环境改善，故归为适老化改造（D），而非一般性救助（B）。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_119_0.png) 务中心的社会工作者，她觉得新入住中心的半失能老人张奶奶与自己过世的母亲在长相和性格上都有几分相像，且又是同乡，因此对张奶奶格外关照，什么事情都主动替张奶奶做。这种情形属于社会工作者在服务过程中产生的（ ）问题。

**选项：**
```
A.移情
B.工作倦怠
C.反移情
D.过度保护
```

> **正确答案：** C

**解析：**
A、移情（Transference）：是指服务对象将对他人的情感转移到社会工作者身上的现象，例如，服务对象可能将社会工作者视为父母或其他重要人物，从而产生相应的情感反应。

B、工作倦怠（Burnout）：是指长期从事社会工作导致的身心疲惫状态，表现为情绪耗竭、去人格化和成就感降低等症状。
C、反移情（Countertransference）：是指社会工作者将自己对他人的情感转移到服务对象身上的现象，例如，社会工作者可能将自己对母亲的情感投射到服务对象身上，从而影响专业判断和服务质量。
D、过度保护（Overprotection）：是指社会工作者对服务对象的过度照顾，可能限制了服务对象的自主性和成长机会。
根据题目描述，李姐因为张奶奶与自己母亲相似而特别关照她，这显然是一种社会工作者将自己的情感投射到服务对象身上的现象，即反移情。而移情是服务对象对社会工作者的情感投射，工作倦怠是社会工作者自身的心理状态问题，过度保护则是行为表现，但题目中的情况更符合反移情的定义。
因此，正确答案是C.反移情。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_119_0.png) 年人的获得感和幸福感，某养老服务机构运用“互联网+”技术打造了“一站式”服务平台，为社区老年人提供“助浴”“助餐”“助医”“精神慰藉”等服务。社会工作者小蔡运用这一平台，通过链接卫生服务中心、送餐服务机构和相关志愿服务组织，为社区高龄独居老人李奶奶提供服务。小蔡在此服务过程中扮演的角色是（ ）。

**选项：**
```
A.个案管理者
B.产品经理人
C.专业辅导者
D.平台打造者
```

> **正确答案：** A

**解析：**
- 个案管理者：个案管理是一种协调和整合服务的方法，旨在满足个人的特定需求。个案管理者通常负责评估服务对象的需求，制定服务计划，并协调各种资源来实现这些计划。在这个情境中，小蔡通过链接不同的服务机构为李奶奶提供综合服务，这符合个案管理者的角色。

- 产品经理人：产品经理人的主要职责是负责产品的规划、设计和开发，确保产品满足市场需求。题目中的服务平台是由养老服务机构打造的，而不是小蔡个人负责的产品开发，因此小蔡不太可能扮演产品经理人的角色。- 专业辅导者：专业辅导者通常提供直接的心理或行为辅导服务。虽然题目中提到“精神慰藉”服务，但小蔡的角色更多的是协调和整合资源，而不是直接提供辅导服务。- 平台打造者：平台打造者负责设计和开发服务平台。题目中提到的平台是由养老服务机构打造的，小蔡只是使用这个平台来提供服务，而不是参与平台的开发和建设。

综上所述，小蔡在此服务过程中扮演的角色最符合个案管理者的定义，因为他负责协调和整合各种资源，为服务对象提供综合服务。因此，正确答案是A.个案管理者。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_119_0.png) 在某社区开办了老年人慢性病管理学习小组，拟采取基线测量方法评估服务成效。小李的下列做法中，属于建立基线的是（    ）。

**选项：**
```
A.选择测量慢性病管理行为的指标
B.改变老年人的慢性病管理行为
C.比较老年人慢性病管理行为变化
D.测量老年人对小组的满意度
```

> **正确答案：** A

**解析：**
