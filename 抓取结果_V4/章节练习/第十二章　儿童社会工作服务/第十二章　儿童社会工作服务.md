# 第十二章　儿童社会工作服务

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_132_0.png) 有位10岁脑瘫儿童，父母双双下岗，家境贫困，无钱支付高昂的康复医疗费，也无力送孩子读书，长期封闭的生活环境造成孩子性格孤僻、内向，存在较严重的心理问题。社会工作者小刘经过需求评估，认为（）需要不是该残疾儿童当前的主要需要。

**选项：**
```
A.医疗康复
B.基本生存
C.接受教育
D.价值实现
```

> **正确答案：** D

**解析：**
（考点:儿童的需要）本题考查了残疾儿童的基本需求。此案例中，医疗康复、 基本生存和接受教育都是该残疾儿童当前的主要需要，只有价值实现不属于当前的主要需要, 因此选D。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_132_0.png) 人员，丈夫因工伤死亡，独自带着女儿在城市打工。邻居向社会工作者小张反映，小王自丈夫去世后，精神状态一直不好，说女儿是“丧门星”，时常把女儿打得遍体鳞伤。根据《中华人民共和国反家庭暴力法》，小张首先要做的是（ ）。

**选项：**
```
A.向有关部门报告，确保孩子安全
B.寻找合适成年人，协助照料孩子
C.联系小王的家人，为小王提供帮助
D.组织社区志愿者，为小王提供帮助
```

> **正确答案：** A

**解析：**
根据《中华人民共和国反家庭暴力法》，当发现家庭暴力行为时，社会工作者有责任采取措施保护受害者。在本案例中，小王的女儿遭受了身体虐待，这显然是一种家庭暴力行为。因此，社会工作者小张首先应该向有关部门报告，确保孩子的安全。选项A直接指向了这一法律要求，即及时报告并保护受害儿童。其他选项虽然可能有助于解决问题，但它们并不是法律规定的首要行动。例如，寻找合适成年人或联系家人可能有助于提供支持，但不能替代向有关部门报告的法律责任。因此，正确答案是A。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_132_0.png) 在为一名困境儿童服务时，先与儿童一起梳理他身边的亲友、邻里和学校等潜在支持系统，并教他如何自主求助；随后把社区里闲置的绘本、企业捐赠的电脑及大学生志愿者名单集中起来，按优先级配送给该儿童。上述两项工作依次对应儿童社会工作者的核心角色是（    ）。

**选项：**
```
A.个案社会工作服务管理者和资源整合者
B.资源整合者和个案社会工作服务管理者
C.使能者和倡导者
D.教育者和治疗者
```

> **正确答案：** A

**解析：**
儿童社会工作者的核心角色包括使能者、教育者、个案管理者、资源整合者等。题干中的两项工作分别对应不同角色：
第一项工作：与儿童一起梳理潜在支持系统，教他如何自主求助，属于个案社会工作服务管理者的角色，即统筹整个个案服务的进程，评估服务对象的需求和资源，提升服务对象的能力，规划服务的整体方向。
第二项工作：把社区闲置绘本、企业捐赠的电脑、大学生志愿者名单集中起来按优先级配送，属于资源整合者的角色，即链接、整合各类分散的资源，匹配给服务对象，满足其需求。
各选项分析：
选项A：个案社会工作服务管理者和资源整合者，与两项工作的先后顺序完全对应，因此A正确。
选项B：将两个角色的顺序颠倒，第一项工作是统筹个案服务的管理角色，第二项是整合资源的角色，因此B错误。
选项C：倡导者角色是指向政策层面提出建议，推动政策完善，题干中没有体现相关内容，因此C错误。
选项D：治疗者角色是指为服务对象提供心理疏导、行为矫正等治疗服务，题干中没有体现相关内容，因此D错误。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_132_0.png) 救助后转学到城区小学，老师发现他总把午餐里的鸡腿、水果偷偷塞进书包，放学后不直接回救助站，而是绕到桥洞下把食物分给几个流浪伙伴。浩浩在作文里写道：“我希望明天他们还在，这样我就不是一个人了。”学校社会工作者此时首先应该做的是（    ）。

**选项：**
```
A.与浩浩建立信任关系，评估其情感支持系统
B.教育浩浩不要与陌生人接触，避免安全风险
C.向救助站报告，要求限制浩浩放学后外出
D.为浩浩申请物资补助，满足其基本生活需要
```

> **正确答案：** A

**解析：**
本题考查儿童社会工作的实务原则。社会工作介入的首要原则是与服务对象建立专业信任关系，优先评估服务对象的需求，而非直接采取干预措施。

选项A：浩浩的行为背后是对同伴支持的情感需求，他害怕孤独，希望通过分享食物获得归属感。社会工作者首先需要与浩浩建立信任关系，充分了解其内心感受和需求，评估其现有的情感支持系统，才能进一步制定针对性的服务方案，这是介入的首要步骤，因此该选项正确。选项B：直接教育浩浩不要与陌生人接触，没有共情浩浩的情感需求，反而会引发其抵触情绪，破坏专业关系的建立，因此错误。选项C：向救助站报告并限制其外出，会切断浩浩仅有的情感支持来源，加剧其孤独感，甚至可能导致更严重的心理问题，违背了儿童利益最大化原则，因此错误。选项D：浩浩本身已经接受救助站救助，基本生活需要已经得到满足，其核心需求是情感支持而非物资补助，因此该选项没有回应核心需求，错误。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_132_0.png) 的夫妇收养了一个被遗弃的1岁女孩。3年后，养父因车祸去世，女孩被诊断为智力发育迟缓，养母因无太多技能，收入也不高，无力安排女孩进行早教治疗，遂向社会工作者求助。下列服务计划中，社会工作者最宜采取的是（ ）。

**选项：**
```
A.向养母提供经济、康复等方面的帮助
B.安排女孩到经济条件较好的家庭寄养
C.将女孩送到福利机构抚养
D.为女孩寻找新的收养家庭
```

> **正确答案：** A

**解析：**
选项A是向养母提供经济、康复等方面的帮助。这符合社会工作的目标，即帮助服务对象解决实际困难，提供必要的支持。养母目前面临经济困难，无法为女孩提供早教治疗，社会工作者可以协助她申请经济援助，或者提供康复资源，帮助女孩得到必要的治疗和支持。

选项B是安排女孩到经济条件较好的家庭寄养。虽然寄养可能提供更好的经济条件，但寄养家庭是否愿意承担女孩的特殊需要（如智力发育迟缓的治疗）是一个问题。此外，寄养可能对女孩的情感稳定产生负面影响，因为她需要适应新的家庭环境，这对她的心理发展可能不利。
选项C是将女孩送到福利机构抚养。福利机构可能提供基本的生活保障，但是否能够提供个性化的康复治疗和教育支持存在疑问。此外，福利机构的环境可能不如家庭温暖，对女孩的心理健康可能有不利影响。
选项D是为女孩寻找新的收养家庭。寻找新的收养家庭可能需要时间，而且新的家庭是否愿意接受一个有特殊需要的孩子也是一个未知数。此外，频繁更换家庭环境可能对女孩的心理造成伤害。
综上所述，选项A是最直接、最实际的解决方案，能够直接解决养母的经济困难，并为女孩提供必要的康复治疗，符合社会工作的干预原则，即尽量保持家庭的完整性，提供必要的支持，而不是轻易改变女孩的生活环境。因此，正确答案是A。

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_132_0.png) 近学习成绩骤降，还常偷拿同学东西。班主任将小雨转介给社会工作者希望给她做一对一心理辅导。社会工作者初步了解后发现：小雨母亲患癌，父亲长期出差；小雨家里已拖欠医药费，社区无亲人可借力；小雨每天放学后独自照顾母亲和年幼的弟弟，无暇完成作业。此时，社会工作者应选择的服务方法及开展的具体工作是（    ）。

**选项：**
```
A.个案社会工作服务方法——对小雨立即开始每周两次个体心理辅导，先处理小雨的偷窃行为
B.个案社会工作服务管理·先联合医院、民政、学校、社区组建团队，共同化解小雨医疗—经济—照护—学业多重困境
C.个案社会工作服务方法——马上联系寄宿学校，让小雨住校脱离压力源
D.个案社会工作服务管理——先替小雨申请低保，然后结案
```

> **正确答案：** B

**解析：**
本题考查个案工作与个案管理的区别：

普通个案社会工作服务针对的是服务对象单一或相对简单的问题，由社工单独提供服务，聚焦于问题的直接干预。个案社会工作服务管理针对的是服务对象面临多重、复杂、交织的问题，需要跨专业、多部门的资源整合，社工的角色是资源协调者，链接多方主体共同解决问题。
对选项A和B的举例说明：如果小雨只是单纯有偷窃行为、家庭没有其他困难，那可以用普通个案方法，每周做心理辅导纠正行为（对应A选项）；但现在小雨的问题是多重的，母亲患癌需要医疗资源、家里欠医药费需要经济救助、弟弟没人照顾需要照护支持、自己没时间写作业需要学业支持，这些问题社工一个人解决不了，需要联合民政部门给低保、医院给医疗救助、学校给学业支持、社区给临时照护，所以需要用个案管理的方法，把这些资源整合起来共同解决问题（对应B选项）。
综上本题选B。

## 第 70 题 [单选题]

**题目：** 70. ![图](./images/q_70_132_0.png) 张经常打骂女儿，于是向社会工作者小邓反映，希望小邓进行干预。小张认为打骂孩子是自己的家事，不用外人管。小邓告诉小张，不能用打骂的方式管教孩子，如果管教不当造成严重伤害将承担相应的法律责任。小邓上述说法依据的基本价值理念是（ ）。

**选项：**
```
A.家长对儿童具有权威性
B.儿童权利的保护和实现
C.保障家庭利益的最大化
D.儿童教育发展的最优化
```

> **正确答案：** B

**解析：**
选项A：“家长对儿童具有权威性”这一观点可能强调家长在家庭教育中的权威地位，但忽视了儿童作为独立个体的权利。在现代社会，家长的权威不应凌驾于儿童的基本权利之上，特别是在存在虐待的情况下，家长的权威不应被过度强调。

选项B：“儿童权利的保护和实现”直接对应了社会工作的核心价值理念，即保护儿童免受虐待、忽视和其他形式的伤害，确保儿童的基本权利得到尊重和实现。小邓的干预行为正是基于这一理念，旨在保护儿童免受暴力侵害。
选项C：“保障家庭利益的最大化”可能意味着在家庭内部寻求利益平衡，但在这个情境中，家庭利益的最大化可能与儿童的权益保护相冲突。如果家长的行为对儿童造成伤害，单纯追求家庭利益的最大化可能忽视了儿童的权益。
选项D：“儿童教育发展的最优化”虽然关注儿童的发展，但题目中的问题更侧重于保护儿童免受暴力，而不是教育发展的优化。因此，这一选项可能不完全符合题意。
综上所述，小邓的干预行为最直接体现了儿童权利保护的理念，因此正确答案是B。

## 第 77 题 [单选题]

**题目：** 77. ![图](./images/q_77_132_0.png) 小刚，因父母入狱，没有亲戚照顾，无人抚养。为解决小刚目前的问题，社会工作者正确的做法是（ ）。

**选项：**
```
A.联系民政部门，帮助小刚寻找领养家庭

```

综上所述，选项C是最符合法律规定和社会工作专业伦理的做法，能够为小刚提供及时、有效的帮助。
