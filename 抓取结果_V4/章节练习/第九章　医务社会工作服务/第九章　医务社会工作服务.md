## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_102_0.png) 为糖尿病。通过网络检索和与病友交流，他知道自己未来要长期服药，还要严格控制饮食，生活质量将大幅下降。老张因此变得非常沮丧，情绪低落。针对老张的情况，医务社会工作者首要的做法应是（ ）。

**选项：**
```
A.走访老张的家庭了解其家人是否有患病史
B.与医生讨论老张的病情及可能的治疗方案
C.链接多种资源为老张提供及时的医疗救助
D.使用专业方法对老张的消极情绪进行干预
```

> **正确答案：** D

**解析：**
糖尿病服务内容包括：①在医疗适应方面，医务社会工作者需要协助患者了解和适应病情并配合治疗；鼓励患者接受病情和治疗措施；鼓励患者向医护人员了解病情和治疗措施；同时向医护人员反映患者的需求，以此提高患者的治疗依从性。②在疾病认知方面，医务社会工作者需要对患者进行健康教育，包括疾病、治疗、康复及饮食运动等知识介绍；同时对患者疾病及治疗的认知不足及偏差进行矫正。③在心理情绪支持方面，医务社会工作者需要为患者因疾病及治疗而引起的心理情绪困扰提供情绪支持(选项D正确)。④在家庭支持网络方面，医务社会工作者可以为患者家属提供资源，解决家庭因疾病而引起的各类情感困扰或家庭关系障碍；为患者家庭提供社会心理服务。⑤在出院康复照护方面，医务社会工作者可以协助患者提升疾病自我管理能力；协助家属学习照护患者的技巧；链接社区及医疗资源，为患者制订出院计划，使患者出院后依然能得到良好的身心照护。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_102_0.png) 中，社会工作者要为因疾病而陷入危急处境的病人及其家属进行危机干预。在进行危机干预时，要强调介入的（ ）。

**选项：**
```
A.服务性
B.预防性
C.及时性
D.长效性
```

> **正确答案：** C

**解析：**
危机干预是指在危机发生时，社会工作者迅速介入，提供必要的支持和帮助，以减轻危机的影响并促进恢复的过程。在医务社会工作中，面对因疾病而陷入危急处境的病人及其家属，及时性尤为重要。因为危机往往具有紧急性和不确定性，需要社会工作者能够迅速响应，提供及时的支持和资源链接，以防止情况进一步恶化。及时性意味着在危机发生的第一时间采取行动，这有助于稳定情绪、提供必要的信息和支持，从而有效地应对危机。其他选项如服务性、预防性和长效性虽然在社会工作中也很重要，但它们并不是危机干预中最强调的特性。预防性更多关注于危机前的准备工作，长效性则强调长期的支持，而服务性则是提供各种服务的总称，但都不如及时性在危机干预中更为关键。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_102_0.png) 意外事故，一名伤员被送至医院急诊部。因该伤员伤势较重，陪同人员心情急躁，觉得医护人员行动迟缓、救治不力，与医护人员发生争吵，出现过激行为，妨碍了救治工作。此时，急诊部的医务社会工作者最适宜的做法是（ ）。

**选项：**
```
A.协助医院保安，对患者陪同人员暂时实施隔离
B.提供现场陪伴，对患者陪同人员进行情绪疏导
C.引导医护人员，避免与患者陪同人员直接接触
D.劝导陪同人员，并探究其出现过激行为的原因
```

> **正确答案：** B

**解析：**
选项A建议协助保安对陪同人员实施隔离。虽然隔离可以防止进一步的冲突，但这种方法可能过于激进，容易加剧陪同人员的不满情绪，不利于问题的解决。

选项B提出提供现场陪伴并对陪同人员进行情绪疏导。这符合医务社会工作者的专业职责，通过陪伴和倾听，帮助陪同人员缓解情绪，理解医护人员的工作压力，从而减少过激行为，为救治工作创造良好的环境。
选项C建议引导医护人员避免直接接触陪同人员。虽然这可以在一定程度上减少冲突，但医护人员需要与陪同人员沟通病情，避免直接接触可能会影响信息的传递和治疗的连续性。
选项D提到劝导陪同人员并探究其过激行为的原因。这虽然有助于了解问题的根源，但在紧急情况下，可能需要更直接的干预措施来迅速稳定局面。
综上所述，选项B是最适宜的做法，因为它既体现了医务社会工作者的专业技能，又能有效地缓解紧张局势，保障救治工作的顺利进行。

## 第 33 题 [单选题]

**题目：** 33. ![图](./images/q_33_102_0.png) 为乳腺癌术后患者开展支持性小组社会工作服务。在小组服务过程中，小王邀请康复者向组员分享顺利度过术后康复期的经验。小王此举的目的是帮助组员（    ）。

**选项：**
```
A.缓解紧张焦虑情绪
B.建立社会支持网络
C.探索疾病应对策略
D.提升科学康复意识
```

> **正确答案：** B

**解析：**
本题考察医务社会工作中支持性小组的功能，需准确区分支持性小组各项功能的主次关系。
支持小组一般由具有某一共同性问题的小组组员组成。通过小组组员彼此之间提供信息、建议、鼓励和情感支持，达到解决某一问题和成员改变的效果。在支持小组中，最重要的是小组组员的关系建构、相互交流和相互支持。
A：虽然支持性小组客观上能够起到舒缓情绪的效果，但这是整个过程可能带来的良性结果，而非康复者介绍经验这一具体行为的最主要目的。情绪舒缓更多是依托于整个小组环境和组员之间的情感支持，单纯靠康复者介绍经验并不能直接达成情绪舒缓这一目标。故A错误。

C：建立社会支持网络是小组工作长期运行后的累积效果，是一个系统的建设过程。康复者单次的经验介绍虽然有助于促进组员互动（患者听、康复者讲），但这种单向分享很难在短期内实质性地构建出一种网络化的人际支持和资源连接网络。社会支持网络的建立通常需要小组组员的多轮交流和深度互动，而非康复者介绍经验这一动作本身所能直接实现的。故C错误。
D：提升科学养生意识属于教育小组或健康教育活动的主要目标，本题中的经验分享虽然包含养生信息，但并非最主要目的。故D错误。
