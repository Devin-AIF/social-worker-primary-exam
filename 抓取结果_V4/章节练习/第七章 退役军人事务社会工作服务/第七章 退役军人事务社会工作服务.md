## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_48_0.png) 部队服役时因训练伤导致腰椎间盘突出，退役后求职时多次因身体条件不符合标准被企业拒绝，妻子因家庭经济压力与他频繁争吵，李某逐渐出现失眠、自责、不愿社交的症状。社会工作者介入后，首先应采取的服务措施是（    ）。

**选项：**
```
A.为李某联系残疾人就业专场招聘会
B.协助李某申请退役军人医疗补助与伤残等级评定
C.为李某夫妻开展婚姻家庭辅导
D.采用认知行为疗法缓解李某的抑郁情绪
```

> **正确答案：** B

**解析：**
案例中李某的核心问题源于训练伤导致的身体障碍，进而引发就业困难、家庭矛盾和心理问题。社会工作者需遵循“根源解决优先”原则开展工作，伤残等级评定是申请医疗补助、就业扶持等后续服务的基础，且属于退役军人法定权益，优先协助办理能从根本上缓解其经济压力和化解就业困境。选项A未解决身体障碍的核心问题，选项C、D属于衍生问题的干预，应在权益保障基础上开展。综上，故选B。
