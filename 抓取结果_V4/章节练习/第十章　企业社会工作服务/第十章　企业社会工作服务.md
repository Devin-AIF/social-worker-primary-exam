## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_105_0.png) 策略是（ ）。

**选项：**
```
A.争取职工的职业福利
B.协助监督企业落实涉及职工权益的法律法规
C.因地制宜开展和设计有特色的服务项目
D.兼顾公共性、公益性与多方共赢
```

> **正确答案：** D

**解析：**
A争取职工的职业福利：这个选项关注的是职工的直接福利，如工资、保险、假期等。虽然这是企业社会工作的一部分，但它可能过于狭隘，没有涵盖更广泛的社会责任和多方共赢的概念。

B协助监督企业落实涉及职工权益的法律法规：这个选项强调的是监督企业遵守法律法规，确保职工权益得到保护。这也是企业社会工作的重要方面，但它可能更偏向于合规性监督，而没有提到更积极的策略，如服务项目的设计或多方共赢。
C因地制宜开展和设计有特色的服务项目：这个选项强调根据企业的具体情况设计服务项目，具有一定的灵活性和创新性。这确实是企业社会工作的一个重要策略，因为它可以根据企业的特点提供定制化的服务。然而，这个选项可能更侧重于服务项目的实施，而没有全面涵盖公共性、公益性和多方共赢的理念。
D兼顾公共性、公益性与多方共赢：这个选项强调的是在企业社会工作中，不仅要关注职工的福利和权益，还要考虑公共利益和社会责任，同时寻求多方共赢的局面。这种策略更加全面和长远，体现了企业社会工作的核心价值，即在满足职工需求的同时，促进社会的整体福祉和发展。
综合以上分析，选项D不仅涵盖了职工权益的保护，还强调了公共性、公益性和多方共赢，这更符合企业社会工作的整体策略。因此，正确答案是D。

```
A.个人中心取向规划技巧
B.心理动力取向规划技巧
C.认知行为取向规划技巧
D.特质因素取向规划技巧
```

> **正确答案：** D

**解析：**
A个人中心取向规划技巧：这种取向强调个人的自我实现和自我探索，注重个人的内在动机和自我导向。虽然与职业生涯规划有一定关联，但题目中的方法更侧重于评估需求和明确路径，而不是自我探索。

B心理动力取向规划技巧：这种取向关注个体的心理动力和潜意识动机，如弗洛伊德理论中的本我、自我、超我等。这与题目中的实际操作不符，因为大姚的方法更注重现实的职业需求评估。
C认知行为取向规划技巧：这种取向强调认知和行为的相互作用，通过改变思维模式来影响行为。虽然在职业生涯规划中有应用，但题目中的方法更侧重于评估和明确路径，而不是认知重构。
D特质因素取向规划技巧：这种取向认为个人的职业选择应基于个人的特质、兴趣、价值观和能力。它强调将个人特质与职业环境相匹配，以实现职业满意度和个人成长。这与题目中大姚的做法高度契合，因为他通过评估需求、展望理想、明确路径，正是在帮助女工将个人特质与职业发展相结合。
综上所述，正确答案是D.特质因素取向规划技巧。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_105_0.png) 工作者小赵计划开展货车司机疲劳管理支持小组，帮助司机掌握疲劳缓解与安全驾驶技巧。下列关于该小组社会工作服务的正确流程是（    ）。

**选项：**
```
A.需求评估—招募组员—确定目标—制定小组计划书
B.制定小组计划书—确定目标—招募组员—需求评估
C.需求评估—确定目标—制定小组计划书—招募组员
D.招募组员—需求评估—确定目标—制定小组计划书
```

> **正确答案：** C

**解析：**
本题考查小组工作的通用流程。小组工作的标准流程是：
需求评估：首先评估目标群体的需求，明确服务对象的问题和需要，这是小组设计的基础。
确定目标：基于需求评估的结果，设定小组工作的具体目标，明确小组要解决的问题和达到的效果。
制定小组计划书：根据目标制定详细的小组计划书，包括小组性质、时间安排、活动内容、招募标准、评估方法等。
招募组员：根据小组计划书里的招募标准，筛选和招募符合条件的组员。

## 第 14 题 [单选题]

**题目：** 14. ![图](./images/q_14_105_0.png) 个4岁的女儿，之前一直由阿莲的母亲在老家抚养。近期，由于母亲患病住院，阿莲将女儿接到自己身边。因要照顾年幼的女儿，阿莲经常迟到早退，企业负责人非常不满，阿莲面临被辞退的风险。为此，阿莲很焦虑，经常失眠，情绪低落，向社会工作者小芳求助。小芳为其开展了个案服务。在介入阶段，小芳为阿莲进行了心理辅导，舒缓其情绪后，她还应（ ）。

**选项：**
```
A.直接为阿莲的孩子提供临时托管服务
B.与医院联络，为阿莲提供健康医疗服务
C.链接社会资源，为阿莲解决孩子照顾问题
D.与企业负责人沟通，维护阿莲的劳动权利
```

> **正确答案：** C

**解析：**
A直接为阿莲的孩子提供临时托管服务：虽然直接提供托管服务可以解决孩子的照顾问题，但社会工作者通常不会直接提供服务，而是链接资源或建议服务。因此，这个选项可能不太合适。

C链接社会资源，为阿莲解决孩子照顾问题：这是社会工作者常见的介入方式，通过链接社区资源、儿童托管服务、家庭支持服务等，帮助阿莲解决孩子的照顾问题，从而减轻她的负担，让她能够更好地平衡工作和家庭，避免被辞退的风险。这个选项直接针对问题的核心，即孩子的照顾问题，因此是最合适的。
D与企业负责人沟通，维护阿莲的劳动权利：虽然维护劳动权利也很重要，但阿莲面临的主要问题是孩子照顾问题导致的工作表现不佳，如果孩子照顾问题得不到解决，即使暂时维护了她的劳动权利，她仍然可能因为无法妥善照顾孩子而继续面临困难。因此，这个选项虽然重要，但可能不是最直接的解决办法。
综合考虑，选项C是最全面和直接的解决办法，能够从根本上帮助阿莲解决孩子照顾问题，从而改善她的工作表现和情绪状态。因此，正确答案是C。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_105_0.png) 小张发现，有一些员工工作几年后工作热情下降，纪律涣散，不能安心工作，且有跳槽念头。针对这种情况，小张最应开展的小组工作类型是（ ）。

**选项：**
```
A.娱乐小组
B.支持小组
C.成长小组
D.治疗小组
```

> **正确答案：** B

**解析：**
A娱乐小组：这类小组的主要目的是提供娱乐和休闲活动，帮助成员放松和减压。虽然娱乐活动可以缓解压力，但它们可能不足以解决员工深层次的职业问题和动机问题。

B支持小组：支持小组旨在为成员提供情感支持和资源分享的平台，帮助他们应对共同的挑战。在这种情况下，员工可能需要相互支持，分享经验，共同寻找解决问题的方法，以提高工作满意度和归属感。
C成长小组：成长小组通常关注个人发展和技能提升，帮助成员设定职业目标并实现它们。虽然这有助于员工的职业发展，但可能不太直接解决当前的工作热情下降和纪律问题。
D治疗小组：治疗小组通常针对有心理或行为问题的成员，提供专业的治疗和支持。然而，题目中的员工可能并不一定有严重的心理问题，而是面临职业适应和动机方面的问题。
综上所述，支持小组（B）最能直接回应员工当前的需求，通过提供情感支持和资源分享，帮助他们应对职业倦怠和工作满意度下降的问题，从而提高他们的工作积极性和纪律性。因此，正确答案是B.支持小组。

**题目：** 24. ![图](./images/q_24_105_0.png) 织一员的身份而得到的，与职工个人的工作绩效无直接关系的是（）。

**选项：**
```
A.职工福利
B.职工奖励
C.企业福利
D.企业奖励
```

> **正确答案：** A

**解析：**
职工福利是指工资以外的各种福利，是职工基于企业组织一员的身份而得到的奖励，与职工个人的工作绩效并无直接关系。

 A 职工福利：直接对应题目描述。它是指企业为职工提供的非货币性报酬和服务（如社会保险、带薪休假、健康体检等），只要职工在职即可享有，通常不与个人绩效直接挂钩。
B 职工奖励：通常与个人或团队的工作绩效直接相关，如绩效奖金、销售提成等，因此不符合“无直接关系”的要求。
C 企业福利：此术语在人力资源管理中不常用。它通常指政府或社会给予企业的福利政策（如税收减免、补贴），或泛指企业整体的福利制度，而非职工个人直接获得的福利。即使有时被笼统地指代企业为职工提供的福利，但规范术语和题意更精准地指向“职工福利”。
D 企业奖励：通常指针对企业整体或部门的表彰或激励，与职工个人身份无直接必然联系。

## 第 38 题 [单选题]

**题目：** 38. ![图](./images/q_38_105_0.png) 在某工业园区开展企业社会工作服务时了解到，有的工厂没有向员工提供足够的安全防护用品；有的员工受伤后才意识到工作岗位存在安全隐患；有的员工心存侥幸，对职业安全缺乏足够重视。针对上述情况，小刘计划开办职业安全与健康教育小组，帮助员工学习相关知识，提升自我保护意识和能力。小刘开展该小组工作的正确流程是（ ）。

**选项：**
```
A.需求评估—招募组员—确定目标—制订小组计划书
B.招募组员—需求评估—确定目标—制订小组计划书
C.需求评估—确定目标—招募组员—制订小组计划书
D.制订小组计划书—确定目标—招募组员—需求评估
```

> **正确答案：** C

**解析：**
A需求评估—招募组员—确定目标—制订小组计划书

这个顺序存在问题。在招募组员之前，应该先确定小组的目标，因为目标会直接影响到招募什么样的组员。而需求评估是了解服务对象的需求，这一步应该在确定目标之前进行，以便更好地满足需求。
B招募组员—需求评估—确定目标—制订小组计划书
这个顺序同样不正确。招募组员之前，应该先进行需求评估，了解员工的具体需求，然后根据需求确定小组的目标，最后制订计划书。因此，招募组员应该在需求评估之后，而不是之前。
C需求评估—确定目标—招募组员—制订小组计划书
这个顺序是合理的。首先进行需求评估，了解员工在职业安全与健康方面的需求；然后根据这些需求确定小组的目标；接下来招募符合条件的组员；最后制订详细的小组计划书，包括活动内容、时间安排等。
D制订小组计划书—确定目标—招募组员—需求评估
这个顺序明显错误。制订计划书需要明确小组的目标，而确定目标又需要基于需求评估的结果，招募组员则是在目标确定之后进行的。因此，这个顺序是倒置的。
综上所述，正确的流程应该是先进行需求评估，然后确定目标，再招募组员，最后制订小组计划书。因此，正确答案是C。
