# 第八章　学校社会工作服务

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_106_0.png) 同学怀疑李同学偷走了自己的课外书，放学后，二话不说就将其暴打一顿，还当众辱骂其是“三只手”,致使李同学第二天不敢到学校上课。学校社会工作者得知此事后，首先应做的是（ ）。

**选项：**
```
A.为初一学生开办社交技巧训练小组
B.帮助孙同学认识到暴力行为的后果
C.通过个案工作，为李同学提供心理支持
D.与学校领导共同制定学校暴力的惩罚制度
```

> **正确答案：** C

**解析：**
结合题干，李同学因被暴打和辱骂而不敢上学，是直接受害者，其心理受到了较大创伤，此时社会工作者首先要做的就是对其进行心理安抚和支持，帮助他缓解恐惧和焦虑情绪，恢复心理平衡，使其能够重新回到学校正常学习，选项C符合满足遭遇伤害学生保护性需要的原则，其他选项虽然也可能是后续工作，但不是首要任务。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_106_0.png) 学生，因沉迷网络游戏曾被父母送到一家以严厉惩罚为手段的专门机 构戒除网瘾。这段不愉快的经历让小强对父母产生了很深的误解，认为他们根本不爱他,才让他 去受折磨。小强的父母也感到十分委屈和懊悔，觉得白花钱没效果，还给孩子造成心理创伤，干 脆不管小强了。小强的班主任将小强转介给学校社会工作者小方。小方首先帮助小强及其父母 觉察他们的亲子关系问题。根据以上描述，小强与父母的亲子关系目前存在的问题是（）。

**选项：**
```
A.纠缠
B.忽视
C.疏离
D.对立
```

> **正确答案：** C

**解析：**
（考点:亲子关系困境与社会工作介入）本题考查亲子关系困境与社会工作介 入。有问题的亲子关系主要表现为父母与孩子的关系过于纠缠或过于疏离，以及由于父母对孩 子过高的学业期望而产生的冲突对立。纠缠的亲子关系是指,父母与孩子之间的关系没有界限， 对孩子控制过于严格或过度溺爱,不给孩子自主发展的空间。疏离的亲子关系则走向了另一极端,即父母和孩子之间缺少正常的沟通交流,父母对孩子过于放任不管，这种亲子关系存在的风 险是，孩子感受不到父母的关注和爱。题干中小强因戒除网瘾事件与父母产生了很深的误解，父 母现在直接放任不管，双方缺乏必要的沟通和交流，属于疏离的亲子关系。故本题答案为C。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_106_0.png) 小孟发现很多高三学生十分迷茫，对自己的未来既缺乏清晰的目标，又不知道应该如何进行升学和就业的抉择。为满足这部分学生的发展性需要，小孟最适宜的做法是（ ）。

**选项：**
```
A.举办思想政治教育讲座，帮助学生树立远大的理想信念
B.开展特殊学生个案管理，帮助学生建立良好的自我认知
C.举办学生生涯规划小组，培养学生职业生涯规划的能力
D.开展励志教育专题培训，协助培养学生竞争意识和能力
```

> **正确答案：** C

**解析：**
A举办思想政治教育讲座，帮助学生树立远大的理想信念。思想政治教育虽然重要，但主要目的是提高学生的政治觉悟和道德水平，对于直接解决学生在升学和就业抉择中的迷茫作用有限，可能不是最直接的解决办法。

B开展特殊学生个案管理，帮助学生建立良好的自我认知。个案管理通常针对有特殊需要的学生，而题目中的情况是普遍存在的高三学生，所以这个选项可能针对性不强，且建立自我认知虽然重要，但可能不足以解决他们在升学和就业抉择上的迷茫。
C举办学生生涯规划小组，培养学生职业生涯规划的能力。生涯规划小组可以系统地帮助学生了解自己的兴趣、能力，探索职业方向，学习决策技巧，直接针对学生在升学和就业抉择中的迷茫，提供实用的指导和支持，因此这个选项非常符合需求。
D开展励志教育专题培训，协助培养学生竞争意识和能力。励志教育和竞争意识的培养虽然有助于学生积极面对未来，但可能更多是激发动力，而不是直接帮助他们明确职业规划和做出抉择，因此可能不是最合适的解决办法。
综上所述，选项C最直接、最有效地满足了学生的发展性需要，帮助他们明确职业规划，做出合理抉择，因此正确答案是C。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_106_0.png) 小唐拟开展“认识情绪”主题班会，运用讲授、角色扮演等方法协助学生更好地认识和处理情绪问题。根据主题班会的工作流程，在掌握学生的基本情况并与班主任沟通确定主题后，小唐接下来应该做的是（ ）。

**选项：**
```
A.准备班会需要的物资资料
B.设计班会的活动策划书
C.熟练掌握班会的活动环节
D.开展班会的满意度评估
```

> **正确答案：** B

**解析：**
首先，根据题目描述，小唐是一名学校社会工作者，计划开展“认识情绪”主题班会。在掌握学生基本情况并与班主任沟通确定主题后，接下来的步骤应该是设计班会的活动策划书。这是因为，在确定主题之后，需要详细规划班会的具体内容，包括活动目标、活动内容、活动流程、所需物资、时间安排等，这些都是活动策划书应该包含的内容。而选项A准备物资资料、C熟练掌握活动环节、D开展满意度评估，这些步骤通常是在活动策划书设计完成之后进行的。因此，正确的顺序应该是先设计活动策划书，再准备物资，然后实施活动，最后进行满意度评估。所以，答案应该是B。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_106_0.png) 小王发现四年级某班有些学生存在不尊重他人、不善团结合作、遇到矛盾冲突也不能很好沟通交流等问题。为改变这种状况，小王为该班学生开设了社交技巧训练课程。小王下列做法中，有助于提升学生社交技巧的是（ ）。

**选项：**
```
A.引导学生探讨诚信守法的议题
B.教授学生提升自尊自信的方法
C.指导学生学习赞赏别人和正确回应别人的赞赏
D.引导学生运用不同的思维方式处理日常生活中的问题
```

> **正确答案：** C

**解析：**
选项A：“引导学生探讨诚信守法的议题”。诚信守法是社会行为规范的重要组成部分，虽然对于学生的道德教育非常重要，但它主要涉及的是行为规范和法律意识，与直接提升社交技巧的关联性可能不是最强的。因此，虽然这是一个好的教育方向，但可能不是最直接的社交技巧训练。

选项B：“教授学生提升自尊自信的方法”。自尊自信对于个人的心理健康和社会适应非常重要，但自尊自信的提升更多是个人层面的心理建设，虽然有助于社交，但并不是直接的社交技巧训练。因此，这个选项可能不是最佳选择。
选项C：“指导学生学习赞赏别人和正确回应别人的赞赏”。赞赏别人是一种积极的社交行为，能够增进人际关系；而正确回应别人的赞赏则体现了对他人的尊重和认可。这两个方面都是社交技巧的重要组成部分，直接涉及到如何与他人进行正面互动，因此这个选项非常符合提升社交技巧的目标。
选项D：“引导学生运用不同的思维方式处理日常生活中的问题”。思维方式的多样性确实有助于解决问题，但这个选项更偏向于认知层面的训练，而不是直接的社交技巧训练。虽然有助于学生的全面发展，但与题目中提到的社交技巧问题的直接关联性可能不如其他选项。
综上所述，选项C最直接地针对了社交技巧中的正面互动和尊重他人这两个方面，因此是最佳答案

```
A.引导参加服务的学生想象引起紧张的情景
B.与参加服务的学生共同商定行为奖惩清单
```

选项D：“促使参加服务的学生互相给予积极反馈”，这可能涉及团体动力学或社会支持，但不是系统脱敏疗法特有的操作。

**题目：** 48. ![图](./images/q_48_106_0.png) 个案成长陪伴服务模式，下列说法正确的是（）。

```
A."2"是指两名服务对象
B."1"是指一名志愿者
C.志愿者配对形式主要为随机配对

```

**解析：**
拆解“2+1”的含义（这是排除A、B、D选项的关键）：
“2”：指的是两名志愿者。
“+”：代表“共同服务”或“配对组合”。
“1”：指的是一名服务对象。
因此，整个“2+1”模式可以解读为：由两名志愿者组成一个小组，共同为一名服务对象提供成长陪伴服务。

A错误。如上所述，“2”指的是两名志愿者，而非服务对象。如果服务对象是两名，就变成了“2对2”或小组工作，不符合该模式“个案”（针对单个对象）深度陪伴的初衷。

**题目：** 49. ![图](./images/q_49_106_0.png) 放学后在操场遇到英语老师，她对老师微笑，老师却没有理会她。小丽觉得老师一定是不喜欢自己，找学校社会工作者小秦诉说自己的苦恼。小秦引导小丽辨识出“老师不理会即等于不喜欢我”的这种非理性想法，从提升自我效能感、增加自信的角度，让小丽认识到上述想法属于（ ）。

**选项：**
```
A.非黑即白
B.灰色情境
C.以偏概全
D.透视心意
```

> **正确答案：** D

**解析：**
题目中的情境是小丽因为老师没有理会她的微笑，就认为老师不喜欢自己。这里涉及到认知行为疗法中的非理性思维模式。小秦引导小丽辨识出这种想法，即老师不理会等于不喜欢我，属于透视心意的非理性想法。

透视心意（Mind Reading）是指个体未经证实就认为自己知道他人在想什么，即错误地解读他人的意图或感受。小丽没有直接询问老师，而是主观臆断老师不喜欢她，这正是透视心意的表现。
其他选项的解释：
A非黑即白（All-or-Nothing Thinking）是指个体用绝对化的思维来看待事物，非此即彼，没有中间地带。例如，认为自己必须完美，否则就是失败。这与题目中的情境不符。
B灰色情境（Gray Area）这个表述可能不太准确，可能是指模糊不清的情况，但认知行为疗法中的非理性思维模式中没有直接对应的术语。可能是指过度概括或模糊化思维，但与题目中的情境不直接对应。
C以偏概全（Overgeneralization）是指基于单一事件或少量信息做出广泛的一般性结论。例如，一次考试失败就认为自己永远无法成功。题目中的情境是基于一次老师没有理会就认为老师不喜欢自己，这可能更接近以偏概全，但根据认知行为疗法的标准分类，透视心意更准确地描述了小丽的行为。
D透视心意（Mind Reading）如前所述，是最符合题目情境的选项。
因此，正确答案是D.透视心意。
