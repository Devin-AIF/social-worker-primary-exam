# 第十三章　青少年社会工作服务

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_116_0.png) 运用“父母、成人、儿童”自我状态觉察训练方法协助高一新生提高自我觉察能力。训练开始时，全体组员站在“儿童”圈内。当组员与其他组员沟通，说话语气从情绪化转向成熟理性，行为表现从抱怨转向理性地看待和解决问题后，该组成员就可以（ ）。

**选项：**
```
A.从父母圈跳到成人圈
B.从成人圈跳到儿童圈
C.从儿童圈跳到成人圈
D.从儿童圈跳到父母圈
```

> **正确答案：** C

**解析：**
A、从父母圈跳到成人圈：父母自我状态通常表现为权威、指导性强，但题目中组员是从儿童状态转变，直接跳到成人圈更符合逻辑。

B、从成人圈跳到儿童圈：这显然与题意相反，因为组员是从儿童状态进步到成人状态。
C、从儿童圈跳到成人圈：这正是题目所描述的转变过程，符合题意。
D、从儿童圈跳到父母圈：父母状态虽然也有指导性，但题目中强调的是理性解决问题，更贴近成人状态。
综上所述，正确答案是C。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_116_0.png) 家住农村，学习成绩优异，父母长期在外打工，身边的小伙伴也陆续进城打工，感到十分孤单。临近毕业，小杰不知自己该选择继续升学还是进城打工，十分迷茫，心情焦虑。社会工作者计划开展的下列服务中，有助于小杰进行抉择的是（ ）。

**选项：**
```
A.协助小杰学习压力管理技巧
B.协助小杰多与父母沟通交流
C.协助小杰学习情绪管理方法
D.协助小杰绘制“自画像”，增加自我觉知
```

> **正确答案：** D

**解析：**
小杰面临的是一个关于未来选择的决策问题，他感到孤单、迷茫和焦虑。社会工作者需要帮助他做出合适的决定。选项A和C虽然可以帮助他管理压力和情绪，但这些技能更多是应对当前的情绪问题，而不是直接帮助他做出升学或打工的决策。选项B建议与父母沟通，但父母长期在外打工，可能难以提供即时的支持。而选项D通过绘制“自画像”来增加自我觉知，可以帮助小杰更好地了解自己的兴趣、优势和价值观，从而更清晰地认识到自己的需求和愿望，为做出决策提供依据。因此，D选项是最有助于小杰进行抉择的。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_116_0.png) 会工作服务要特别关注那些好打抱不平、认同并采取激进行动以追求某些理想或社会取向的（ ）。

**选项：**
```
A.问题青年
B.正直青年
C.文化叛逆青年
D.政治偏激青年
```

> **正确答案：** D

**解析：**
A、问题青年：这个术语通常指的是存在行为问题或心理问题的青少年，但并不特指那些因为政治或社会理想而采取激进行动的群体。

B、 正直青年：这个选项显然不符合题意，因为正直青年通常指的是品行端正、遵守社会规范的青少年，与题干中的激进行动相矛盾。
C、文化叛逆青年：这类青年可能对主流文化持批判态度，但他们的行为更多体现在对文化的反抗上，不一定涉及政治或社会取向的激进行动。
D、政治偏激青年：这个选项最符合题干的描述，因为政治偏激青年往往对政治和社会问题持有强烈的立场，容易采取激进的行动来追求他们认为的理想或社会变革。
因此，正确答案是D.政治偏激青年。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_116_0.png) 级辩论队的队长，在准备校际比赛时，他先收集每位队员对辩题的看法，整理成逻辑框架，再组织大家模拟演练，并用“我认为”“你觉得怎样”等语句与队员交流。根据艾瑞克·伯恩的沟通分析理论，小吕在互动中主要处于（    ）状态。

**选项：**
```
A.成人式
B.儿童式
C.师长式
D.父母式
```

> **正确答案：** A

**解析：**
根据艾瑞克·伯恩（Eric Berne）的沟通分析理论，人有三种自我状态：
父母式（Parent） ：表现为权威、批评、控制或保护、照顾，常使用“你应该”“你必须”“不许”等语句。
成人式（Adult） ：表现为理性、客观、重事实、重逻辑，善于收集信息、分析问题、解决问题，常用“我认为”“我的看法是”“你觉得怎样”等语句。
儿童式（Child） ：表现为情绪化、冲动、依赖、任性、好奇，常用“我不管”“我要”“好棒”等语句。
本题中，小吕作为辩论队队长，先收集每位队员对辩题的看法，整理成逻辑框架，再组织大家模拟演练，体现的是理性分析、客观处理信息的能力；同时使用“我认为”“你觉得怎样”等语句，体现的是平等沟通、尊重他人意见的成人式互动方式。因此，小吕主要处于成人式状态。
故正确答案为A。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_116_0.png) 者针对青少年开展社会参与项目。依据赫胥社会连接理论，下列服务设计中最能提升青少年参与感的是（    ）。

**选项：**
```
A.组织青少年参与社区垃圾清理志愿服务
B.为青少年开展职业生涯规划主题讲座
C.邀请青少年代表参与社区公共空间改造方案讨论
D.为青少年提供免费的兴趣技能培训
```

> **正确答案：** B

**解析：**
本题考察赫胥社会连接理论的核心概念。该理论认为，个人与社会连接的纽带越强，发生越轨行为的可能性就越低。该理论提出的社会连接主要由四个要素构成：依附感、承担感、参与感和信念。
题目中考察的“参与感”，在赫胥社会连接理论中特指“花费时间和精力参加传统活动”。赫胥认为，个人若较深入地参加传统活动，就会全力以赴地忙于各种传统事务，从而缺少从事越轨活动的时间和精力。个人全力以赴参加的传统活动主要分为两类：第一类是传统的工作、运动、娱乐、业余爱好；第二类是与学校有关的传统活动等。
A选项“组织青少年参与社区垃圾清理志愿服务”属于社区志愿服务，在赫胥社会连接理论中并未被明确划入“传统活动”的范畴，因此不是最贴合理论定义的选项。
B选项“为青少年开展职业生涯规划主题讲座”属于传统的工作范畴，职业生涯规划讲座引导青少年提前关注、了解和参与工作相关事务，符合赫胥理论中“参与感”的核心要求。
C选项“邀请青少年代表参与社区公共空间改造方案讨论”属于公共事务参与，在赫胥理论的定义里，同样没有被列入“传统活动”的清单，因此也不是理论直接支持的最佳选项。
D选项“为青少年提供免费的兴趣技能培训”虽然属于娱乐、业余爱好活动，但相比B选项，职业生涯规划讲座更直接地体现了“传统活动”中的工作范畴，与理论核心更为契合。
综上所述，依据赫胥社会连接理论对“参与感”的明确定义，最直接、最贴合理论的选项是B。

```
A.亲子关系紧张
B.家庭关系紧张
C.夫妻关系紧张
D.社区关系紧张
```

> **正确答案：** A

**解析：**

选项AB的区别：

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_116_0.png) 岁的初中生，最近沉迷手机游戏，经常为此顶撞父母，学习成绩下滑。小杰的父母非常焦虑，每天反复唠叨，没收了小杰的手机，甚至打骂小杰，亲子关系日益紧张。学校社会工作者计划为该家庭提供亲职教育辅导服务。以下做法中，最能体现亲职教育辅导“第二主要目标”的是（    ）。

**选项：**
```
A.帮助小杰的父母理解青春期孩子追求自主的心理需求
B.教小杰的父母用“我信息”表达担忧，减少指责
C.指导小杰的父母制定手机使用契约并设奖惩
D.提醒小杰的父母检查家庭氛围是否过于高压
```

> **正确答案：** B

**解析：**
本题考查亲职教育辅导的目标层次。亲职教育辅导通常包含三个主要目标：
第一主要目标：帮助父母增进对子女发展阶段特点与需求的理解，提升对子女行为的正确认知。
第二主要目标：帮助父母掌握有效的亲子沟通与管教技巧，提升亲职能力。
第三主要目标：协助父母改善亲子关系，建立正向互动模式。
题干中，小杰的父母已出现打骂、没收手机等不当管教行为，亲子关系紧张。此时最迫切的是帮助父母学习新的沟通与管教方式，而非仅停留在认知层面。
A选项：帮助理解青春期心理需求，属于第一主要目标（认知层面）。
B选项：教父母用“我信息”表达担忧、减少指责，是典型的沟通技巧训练，属于第二主要目标（技巧层面），且直接回应了父母当前“反复唠叨、指责”的问题。
C选项：指导制定手机使用契约并设奖惩，虽然也属于管教技巧，但更偏向行为控制，且在与父母已采取强硬手段的背景下，需先改善沟通方式才能有效实施契约。
D选项：提醒检查家庭氛围是否高压，属于环境反思，更接近第一目标的延伸，而非具体的亲职技巧传授。
综上，最能体现亲职教育辅导“第二主要目标”的是 B。


最差的做法就是像A选项那样，将青少年视为“问题”进行外部强制和矫正；而专业社工应像B、C、D选项那样，视青少年为有潜能的个体，通过协商、培养兴趣、发挥优势等内在引导的方式促成积极改变。核心区别在于：是把服务对象当成需要修理的“问题”，还是可以共同合作的“伙伴”。
详细解析如下：
A（容易造成负面影响）是典型的错误做法。这类训练营通常采用强制性、封闭式、军事化的管理，其核心逻辑是“矫正”和“惩罚”。负面影响：极易给青少年贴上 “有问题”、“瘾君子” 的负面标签，严重伤害其自尊和自我认知；粗暴的干预方式可能引发强烈的抗拒和逆反心理，破坏专业关系；甚至可能对青少年造成心理创伤。这完全违背了社会工作 “尊重”、“接纳”、“助人自助” 的核心原则。
B（相对合理）是一种基于协商的行为契约法。社工与小强共同商讨、设定规则，体现了对他的尊重，并将部分控制权交还给他本人。这种方法旨在帮助小强学习自我管理，而不是强行剥夺。它更温和，也更容易被青少年接受，为后续深入的辅导打下基础。
C（有一定合理性，需谨慎）积极意图是用健康、有趣的活动（如体育、艺术）来替代和填充小强的空闲时间，转移其对网络的过度专注。关键点：做法的成败取决于是否尊重小强的兴趣和意愿。如果课程是他感兴趣且自愿参加的，则是很好的正向引导；如果是父母单方面强加的“任务”，则可能变成另一种压力，引发反感。
D（积极正向）是优势视角和增能理论的典型应用。通过志愿服务，可以让小强在现实生活中获得 “成就感”、“价值感” 和 “社会连接”。这有助于他重建自信，发现自我能力，体验比网络游戏更真实、更有意义的社会认可，从而从内在动机上降低对网络的依赖。

## 第 89 题 [多选题]

**题目：** 10. ![图](./images/q_89_116_0.png) 家艾德沃特的观点，下列活动中可以作为青少年结束的标志的表现有（）。

**选项：**
```
A.青春萌动
B.做逻辑思考，具有解决问题及作决定的技能
C.进入个人、家庭与工作角色中
D.能够逻辑思考并自主作决定
E.生理与性达到成熟状态，如具有生育能力
```

> **正确答案：** DE

**解析：**
根据著名心理学家艾德沃特的观点，DE两项属于青少年结束的标志的表现，ABC三项属于青少年开始的标志的表现。艾德沃特对青少年阶段的划分，关注的不是某个单一事件（如第一次来月经），而是能力或状态从“初步具备”到“成熟稳定”的质变过程。他的“开始”指刚进入这个发展轨道，“结束”指在这个轨道上达到成熟、稳定的成人状态。
1、 B和D的本质区别：技能 vs. 自主
B指的是刚刚获得这些认知能力，开始学习使用。就像一个青少年开始学着像大人一样思考和处理问题，但可能不熟练、不稳定，容易受他人影响。这是认知发展的起点。定位：青少年开始的标志。
D指的是熟练、稳定、独立地运用这些能力。“自主”是关键词，意味着他的思考和决定是成熟、负责、不受他人支配的。这是认知发展的成熟终点。定位：青少年结束的标志。
简单说：B是“学着做”，D是“能独立做好”。这是从“量变”（有技能）到“质变”（能自主）的飞跃。
2、“生育能力”是结束标志
E“成熟状态”和“生育能力”不能理解为生理功能上的初次具备（如女孩初潮）。在理论中，它指的是生理发育的全面完成和稳定，即身体各个系统（包括生殖系统、内分泌系统、骨骼等）都达到了成年人的成熟水平。这是一个长期的发育终点，而不仅仅是起点。定位：青少年结束的标志。
3、“进入工作角色”是开始标志
C“进入”指的是开始尝试、探索和初步承担这些角色。例如，青少年开始思考“我是谁”（个人角色），在家庭中承担更多责任（家庭角色），进行兼职或思考未来职业（工作角色）。这是一个角色探索和学习的开端，而非稳定、完全的承担。定位：青少年开始的标志。

B 接纳与关爱青少年：无条件接纳困境，传递温暖支持。本题中小冉输送温暖和爱心，主动提供帮助。
C 注重青少年个别需求：识别差异需求，提供个性化服务。本题中小冉针对不同困境提供差异化服务：租住环境差→链接便宜房源；无收入→安排饭店工作；无处求助→生活信息咨询。
D 协助适应社会变化：提升青年应对环境的能力。本题中小冉提供北京生活信息咨询，帮助适应新城市；安排工作促进经济独立。
E 尊重隐私（未体现）：保护服务对象敏感信息。题干未涉及隐私处理（如公开个人信息），题目中所有服务均为公开召集、群体支持（如集体链接资源），未涉及保密或隐私保护场景（如单独访谈、隐藏身份等）。
