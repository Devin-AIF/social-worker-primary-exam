# 2026年初级社会工作者考试《社会工作实务》模拟试卷四

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 在接案时，服务对象老范称“儿子不履行赡养义务，自己生活困难”,但拒绝提供儿子的联系方式，也不愿让小园与儿子沟通。此时小园最恰当的做法是（    ）。

**选项：**
```
A.尊重老范的意愿，暂不联系其子，先为老范链接临时生活救助
B.坚持要求老范提供联系方式，与其子沟通后再提供服务
C.自行通过社区居委会获取老范儿子的联系方式并主动沟通
D.指责老范“不配合服务，无法解决问题”
```

> **正确答案：** A

**解析：**
接案的伦理原则

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) 在对社区青少年沉迷短视频问题进行预估时，通过绘制社会生态图发现，青少年沉迷短视频的原因有：家长忙于工作疏于管教、学校课后作业少且缺乏兴趣活动、短视频平台算法推荐成瘾内容，且青少年自我控制能力较弱。该预估方法的核心优势是（    ）。

**选项：**
```
A.快速发现青少年个人问题
B.强调外部环境责任
C.直观呈现多系统对问题的影响
D.简化问题成因，便于快速干预
```

> **正确答案：** C

**解析：**
预估的方法

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_80_0.png) 为因沟通不畅而引发矛盾的马先生夫妇制定介入计划。初期介入时，采用夫妻面对面沟通训练，但几次介入后，马先生反馈“每次沟通时都忍不住吵架，训练没效果”。此时小希最应优先调整的介入方法是（    ）。

**选项：**
```
A.暂停沟通训练，为马先生夫妇分别开展个体情绪疏导
B.延长沟通训练时长，耐心劝导马先生夫妇坚持下去
C.指明马先生“缺乏耐心，导致训练失败”
D.坚持服务，用行动改变夫妇的意愿
```

> **正确答案：** A

**解析：**
介入方法的动态调整
