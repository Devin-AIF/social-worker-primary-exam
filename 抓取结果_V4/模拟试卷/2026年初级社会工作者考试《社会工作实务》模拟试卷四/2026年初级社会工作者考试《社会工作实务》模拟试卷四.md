## 第 29 题 [单选题]

**题目：** 29. ![图](./images/q_29_80_0.png) 交地方安置后，因不适应社区养老环境，多次拒绝参与社区活动，表示“自己对地方社区不如部队熟悉”。社会工作者从微观层面推进军休干部与社会老人融合时，最适宜的做法是（    ）。

**选项：**
```
A.推动修订军休干部安置保障法规
B.链接社区资源为军休干部提供上门服务
C.引导王某认识到非理性信念对其生活的影响
D.组织军休干部与社区老人共同开展文体活动
```

> **正确答案：** C

**解析：**
（考点：军休社会工作的介入策略——微观层面）
军休社会工作的介入策略分为微观、中观、宏观三个层面。其中，微观层面的核心做法是：与服务对象分析当下的情境，引导服务对象明白，不是个人遭遇的事件而是个人的非理性信念造成了情绪和行为上的困扰，从而引导服务对象加强自我反思、树立豁达的人生态度。
本题中，军休干部王某因不适应社区养老环境，表示“自己对地方社区不如部队熟悉”，其核心问题在于其存在的非理性信念（如对新环境的过度排斥、对部队生活的过度固守），而非缺乏社区活动机会或服务资源。
A项“推动修订军休干部安置保障法规”属于宏观层面做法，不符合“微观层面”要求。
B项“链接社区资源为军休干部提供上门服务”属于中观或宏观层面资源整合，不是微观层面的直接介入。
C项“引导王某认识到非理性信念对其生活的影响”正是微观层面介入的核心方法，符合军休社会工作微观层面的专业要求。
D项“组织军休干部与社区老人共同开展文体活动”属于中观或宏观层面的社区融合活动，虽然有助于促进融合，但不是微观层面的直接介入策略。
故正确答案为C。

## 第 47 题 [单选题]

**题目：** 47. ![图](./images/q_47_80_0.png) 会工作者求助，称小强辍学后沉迷网络游戏，每天昼夜颠倒。社会工作者计划先对小强进行一对一心理辅导，再邀请母子一起参加亲子沟通小组，同时联系社区就业服务站为小强提供职业体验机会。这一做法最能体现青少年社会工作服务方法的（    ）特征。

**选项：**
```
A.社会工作服务专业的通才要求
B.社会资源的综合性
C.青少年需求的多元性
D.服务评估的系统性
```

> **正确答案：** B

**解析：**
本题考查青少年社会工作服务方法的特征，需结合题干中社会工作者的具体做法：
A 社会工作服务专业的通才要求：通才要求指社会工作者需要掌握多领域的专业知识与技能，能够应对不同服务对象的多元需求，这是对社会工作者自身能力的要求，并非青少年社会工作服务方法的特征，与题干描述的服务举措无关，因此A选项错误。

C 青少年需求的多元性：青少年需求的多元性是青少年群体的自身特点，是社会工作者设计多元服务方案的前提和依据，而非服务方法本身的特征。题干设问指向的是“服务方法的特征”，而非服务设计的依据，因此C选项错误。
D 服务评估的系统性：服务评估的系统性侧重服务开展后期对服务成效、服务过程的系统性评价与复盘，题干描述的是服务计划的设计与实施阶段，尚未涉及评估环节，因此D选项错误。


老年社会工作服务的基本原则包括：独立原则、参与原则、照护原则、自我实现原则、尊严原则等。

## 第 52 题 [单选题]

**题目：** 52. ![图](./images/q_52_80_0.png) 周爷爷曾是一名军人，最近创伤后应激障碍症状加重，每晚持拐杖在楼道站岗，并阻止邻居通行，居民联系居委会后，社会工作者介入。社会工作者首先应开展的服务是（    ）。

**选项：**
```
A.危机干预和精神慰藉
B.精神慰藉和社会支持网络建设
C.危机干预和政策倡导
D.家庭辅导和老年教育
```

> **正确答案：** A

**解析：**
本题考察老年社会工作在创伤后应激障碍（PTSD）复发时的优先介入服务策略。
各选项分析如下：
A选项正确。危机干预和精神慰藉是当出现PTSD症状加重情况时的首要服务选择。周爷爷晚上持拐杖站楼道、阻止邻居通行的行为，可能是在创伤状态下的预警反应，需要立即进行危机干预以防意外发生。同时，曾为军人的身份可能使回避的邻居之间产生更多隔离，叠加精神慰藉能缓解心理创伤。因此，A同时包含了“危机干预”和“精神慰藉”，是最完整的优先服务方案。
B选项错误。精神慰藉和社会支持网络建设属于稳定阶段的扩展服务，但题干中行为已出现执行功能和夜间异常行为，在未完成危机评估前不宜直接进入支持网络建设阶段。
C选项错误。危机干预和政策倡导在顺序上不匹配，政策倡导属于宏观层面、中长期工作，不适用于紧急状态的优先介入。
D选项错误。家庭辅导和老年教育属于中长期服务，题干中行为具有较高安全隐患，家庭辅导和老年教育无法解决当下的紧急安全问题。
综上，根据老年人社会工作服务中的安全第一原则，出现PTSD加重及异常行为时应优先进行危机干预（处理紧急安全隐患）+精神慰藉（稳定情绪状态），故A为正确答案。
