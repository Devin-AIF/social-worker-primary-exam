# 2026年初级社会工作者考试《社会工作实务》模拟试卷一

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_78_0.png) 一直没有找到合适的工作，整天闲着，不务正业，前几天他和几个好友在溜冰场溜冰时跟别人发生了冲突，溜冰场的老板好不容易才把他们拉开，张鸣的一个朋友受伤了，为了替哥们儿出口恶气，他趁别人不注意，拿起一块石头朝"仇人"头上砸去，不小心将对方打死。在司法判决前，社会工作者的服务对象除了张鸣之外，还有（）。

**选项：**
```
A.受害者的亲人
B.张鸣的亲友
C.张鸣的朋友
D.当天参与打架的人
```

> **正确答案：** B

**解析：**
矫正社会工作是司法矫正体系中的社会福利服务，贯穿于对罪犯进行司法矫正的各个方面以及整个过程，其主要内容包括司法判决前的服务、监禁场所中的服务、社区矫正中的服务以及刑满释放后的服务。司法判决前社会工作者的服务对象主要包括犯罪嫌疑人和犯罪嫌疑人的亲友。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_78_0.png) 火车票，这是对军人的（）。

**选项：**
```
A.优待
B.优抚
C.抚恤
D.特殊对待
```

> **正确答案：** A

**解析：**
优待一般是指国家、社会对优抚对象在政治上、经济上给予优先、优厚待遇的制度。优抚安置工作中的优待是指对现役军人、退役军人、军队文职人员及其家属等对象，在经济、政治、社会等方面给予物质照顾和优先优惠待遇，主要包括发放优待金，以及在医疗、交通、参观游览、教育、住房等方面给予优待。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_78_0.png) ，在社区活动中结识了学习遇到困难的学生小强，经过面对面的沟通之后，小强主动来到社会工作站向小李求助。这个过程称为社会工作的（）。

**选项：**
```
A.接案
B.接触
C.筛选
D.评估
```

> **正确答案：** A

**解析：**
本题考查接案的步骤及核心技巧——什么是接案。当服务对象遇到困难前来社会工作服务机构求助时，负责接待的社会工作者所做的工作就是社会工作过程中的第一个阶段，即"接案"。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_78_0.png) 的评估中，评估者事先不预设答案，只是听取参与服务的社区居民或服务对象的意见和看法，然后再进行归纳和总结的方法是（）。

**选项：**
```
A.标准评估法
B.定性评估法
C.定量评估法
D.整体评估法
```

> **正确答案：** B

**解析：**
社区服务方案的评估方法包括定量评估法和定性评估法。定性评估法的特点：一是用文字来表达评估结果；二是评估者事先不预设答案，只是听取参与服务的社区居民或服务对象的意见和看法，然后再进行归纳和总结。

B.具有冲动好斗的人格特征

D.具有困难重重的生活特征

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_78_0.png) 对象考核奖惩暂行办法》，社区社会工作站在学习和研究基础上为矫正对象开展了"当新人，做好事，多加分"的宣传服务活动，同时还要求社会工作者在服务开展过程中注意收集活动反馈信息，总结经验，以促进社区矫正制度的不断完善。上述工作采用的是（）方法。

**选项：**
```
A.社区社会工作
B.社会工作行政
C.社区综合治理
D.社会工作研究
```

> **正确答案：** B

**解析：**
社会工作行政是将社会政策变为社会服务的活动，宏观上，既要将社区矫正的政策法规变成具体的针对矫正对象的社会服务，又要将服务经验上升到理论层面以影响政策法规的修订。本题中，先根据政策法规开展宣传服务活动，同时又总结经验，以促进社区矫正制度的完善，正是采用了社会工作行政的方法。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_78_0.png) 现状的评估，通过4R指标来分析，即角色（Roles）、关系（Relationship）、（）、资源（Resources）。

**选项：**
```
A.理解
B.反应
C.尊重
D.关心
```

> **正确答案：** B

**解析：**
本题考查残疾人社会参与现状评估中的4R指标体系。4R指标是残疾人社会工作领域中用于评估残疾人社会参与状况的专业工具，具体包括以下四个维度：
各选项分析如下：
A选项错误。 “理解”不属于4R指标的评估维度，4R指标的四个维度中没有“理解”。
B选项正确。根据残疾人社会参与评估的4R指标体系，四个维度分别为：角色（Roles）、关系（Relationship）、反应（Reaction）、资源（Resources）。其中，“反应”指的是病人和家属对残疾及所面临问题的心理和行为反应，是评估社会参与的重要维度。
C选项错误。 “尊重”虽然体现了服务价值观，但并非4R指标体系的专用术语，不纳入4R评估维度。
D选项错误。 “关心”不属于4R指标的评估维度，4R指标的四个维度中没有“关心”。
【提示：4R指标为Roles(角色)、Relationship(关系)、Reaction(反应)、Resources(资源)】

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_78_0.png) 与社区事务积极性的因素是指（）。

**选项：**
```
A.居民参与社区活动需要精力
B.居民参与社区活动需要具备参与的动力
C.居民参与社区活动主要受环境因素的影响
D.居民愿意参加与自身利益相关的社区活动
```

> **正确答案：** D

**解析：**
社区居民是否积极参与社区事务，主要受三个因素的影响：对参与价值的肯定、参与意愿和参与能力。其中，社区居民参与社区事务的意愿关键在于所参与的社区事务是否与他们的生活或利益密切相关。因此，社会工作者在选择工作的目标和方向时，最好能与居民的利益挂钩。C项，居民参与意愿受参与者的主观因素控制，有时候也会受客观环境的影响，但并不是主要因素。

## 第 11 题 [单选题]

**题目：** 11. ![图](./images/q_11_78_0.png) ，已辍学多年。小明的父母靠给人打临工赚钱，家里比较贫困，也没有社会保障。因为家庭经济窘迫，小明没有读完初中就去打工赚钱了。由于他年龄小，在工作中，工友往往会欺负他，包工头也常常克扣他的工资。这些经历使得小明人际沟通不太理想，失去了对生活的信心，工作缺乏积极性。他总觉得自己低人一等。负面的自我评价，使他变得退缩，没有勇气接外面的世界。为了帮助小明，社会工作者为小明以及其他有相似背景的青少年设计了一套生活辅导计划。以下内容中，（）属于该计划初期的介入策略。

**选项：**
```
A.为他们提供就业培训，创造就业机会
B.进行历奇辅导，发掘内在潜力，提升生活动力
C.提供就业信息，帮助联系就业单位
D.协助重返校园，接受正规的教育，增强竞争力
```

> **正确答案：** B

**解析：**
历奇辅导通过有目的地把青少年带离安适区，进入适度冒险的挑战区，借助体验性活动促进自我探索、自我觉察与自我成长。其核心是帮助青少年在安全环境中突破心理障碍，提升自信与自尊，培养团队合作能力。该策略适用于小明当前因负面经历导致的自我评价低、退缩等问题，符合初期介入以心理支持与动力激发为重点的目标。其他选项（如就业支持、返校）需建立在自信心恢复的基础上，属于中后期介入措施。

## 第 12 题 [单选题]

**题目：** 12. ![图](./images/q_12_78_0.png) 社会工作者小李的鼓励下，开了一家杂货店。最初没有人来买东西，后来在小李的努力下，附近的居民都来小王的杂货店买东西，小王觉得，大家对自己的感觉变了，他的心里充满了温暖。社会工作者小李的做法属于（）。

**选项：**
```
A.社区教育
B.社区照顾
C.社区服务
D.社区安抚
```

> **正确答案：** A

**解析：**
本题考查社区工作方法在社区矫正中的作用一开展社区教育，培育社区居民接纳、尊重矫正对象的意识和习惯。社区矫正的功能发挥要靠社区居民的共同参与以形成有利于罪犯改过自新的社会氛围。矫正社会工作者的一个重要的工作任务，就是通过社区教育的途径，改变居民对矫正对象的偏见，培育社区居民接纳、尊重矫正对象的意识和习惯，使矫正对象顺利回归社会。
