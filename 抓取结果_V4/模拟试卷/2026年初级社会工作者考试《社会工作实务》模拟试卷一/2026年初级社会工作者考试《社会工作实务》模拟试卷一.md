B.张鸣的亲友
C.张鸣的朋友
D.当天参与打架的人
```

> **正确答案：** B

本题考查接案的步骤及核心技巧——什么是接案。当服务对象遇到困难前来社会工作服务机构求助时，负责接待的社会工作者所做的工作就是社会工作过程中的第一个阶段，即"接案"。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_78_0.png) 的评估中，评估者事先不预设答案，只是听取参与服务的社区居民或服务对象的意见和看法，然后再进行归纳和总结的方法是（）。

**选项：**
```
A.标准评估法
B.定性评估法
C.定量评估法
D.整体评估法
```

> **正确答案：** B

**解析：**
社区服务方案的评估方法包括定量评估法和定性评估法。定性评估法的特点：一是用文字来表达评估结果；二是评估者事先不预设答案，只是听取参与服务的社区居民或服务对象的意见和看法，然后再进行归纳和总结。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_78_0.png) 与社区事务积极性的因素是指（）。

**选项：**
```
A.居民参与社区活动需要精力
B.居民参与社区活动需要具备参与的动力
C.居民参与社区活动主要受环境因素的影响
D.居民愿意参加与自身利益相关的社区活动
```

> **正确答案：** D

**解析：**
社区居民是否积极参与社区事务，主要受三个因素的影响：对参与价值的肯定、参与意愿和参与能力。其中，社区居民参与社区事务的意愿关键在于所参与的社区事务是否与他们的生活或利益密切相关。因此，社会工作者在选择工作的目标和方向时，最好能与居民的利益挂钩。C项，居民参与意愿受参与者的主观因素控制，有时候也会受客观环境的影响，但并不是主要因素。

D.家庭生活需要
E.社会交往需要
```

> **正确答案：** ABDE

**解析：**
优抚医院社会工作主要服务于伤残军人、带病回乡退役军人等需要长期医疗康复的群体，其核心需求包括：
治疗康复需要（A）：因伤病导致的医疗及康复支持是首要需求；
基本生存需要（B）：保障基本生活条件（如医疗费用、生活补助）；
家庭生活需要（D）：协助家庭关系调适、解决家属照料问题；

术语更新说明：“复退军人”一词已逐步被“退役军人”取代（如《退役军人保障法》），本题中选项C的实质内容仍指向安置领域，与优抚医院需求无关，因此不选。

## 第 67 题 [多选题]

**题目：** 9. ![图](./images/q_67_78_0.png) 生活在一起，爷爷年迈体力不支，只能靠捡塑料瓶为生。因为家境贫寒，小霞在学校经常被其他同学嘲笑，变得越来越孤僻。作为一名学校社会工作者，面对处境不利的学生，你应该（    ）。

**选项：**
```
A.用包容、整合的视角来看待他们
B.重构“困境学生”的环境
C.和其他同学一样，一视同仁
D.给予更多的支持和鼓励
E.选择漠视，尽量少给自己找麻烦
```

> **正确答案：** ABD

**解析：**
题目要求选择作为学校社会工作者面对处境不利的学生应该采取的措施。根据社会工作的伦理和实践原则，社会工作者应该采取包容、整合的视角来看待处境不利的学生，即选项A。这意味着要理解学生的背景和面临的挑战，而不是简单地评判或忽视他们的情况。
重构“困境学生”的环境，即选项B，是必要的。这可能包括提供资源、支持或改变环境中的不利因素，以帮助学生更好地适应和成长。
选项C，“和其他同学一样，一视同仁”，虽然表面上看起来公平，但实际上可能忽视了处境不利学生所面临的特殊挑战，因此可能不够恰当。
选项D，“给予更多的支持和鼓励”，是直接帮助学生克服困难、建立自信的重要措施，因此是正确的。
选项E，“选择漠视，尽量少给自己找麻烦”，显然违背了社会工作的伦理原则，是不可取的。
综上所述，正确答案是A、B、D。
