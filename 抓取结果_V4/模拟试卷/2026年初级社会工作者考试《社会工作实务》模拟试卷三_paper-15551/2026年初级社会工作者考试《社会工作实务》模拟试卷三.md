# 2026年初级社会工作者考试《社会工作实务》模拟试卷三

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_78_0.png) 很好，有一个儿子，很乖巧听话，学习也很好。自从儿子离家上大学后，老高的妻子开始郁郁寡欢，感觉很孤独，于是向社会工作者求助。社会工作者分析老高妻子的情况后，向其妻子介绍每一个家庭发展阶段都有不同的任务和要求，家庭成员应及时调整自己的角色和分工。以上案例中社会工作者使用的理论解释是（    ）。

**选项：**
```
A.家庭系统理论
B.家庭生命周期理论
C.生态系统理论
D.结构式家庭治疗理论
```

> **正确答案：** B

**解析：**
A、家庭系统理论关注家庭作为一个系统如何运作，强调家庭成员之间的相互作用和系统内的变化。虽然这个理论可以解释家庭成员之间的互动，但题目中更强调的是家庭生命周期的不同阶段，因此A可能不是最佳选择。

C、生态系统理论关注个体与其环境之间的相互作用，包括微观、中观和宏观环境。虽然这个理论可以解释家庭与外部环境的关系，但题目中的重点在于家庭生命周期的不同阶段，因此C可能不太合适。
D、结构式家庭治疗理论关注家庭的结构和沟通模式，以及如何通过改变这些结构来解决问题。虽然这个理论可以用于解决家庭内部的问题，但题目中的情况更侧重于家庭生命周期的变化，因此D可能不是最佳选择。
综上所述，B.家庭生命周期理论最符合题目中的情境，因此正确答案是B。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_78_0.png) 种与人相处的技巧，良好的亲子关系和师生关系与学生的正向社交行为有正面的关系，良好的同辈关系与学生的负面行为则有反面的关系。因此，学校社会工作者提高学生社交能力的第一步是（    ）。

**选项：**
```
A.协助学生认识自己所在的城市或农村
B.协助学生认识自我
C.协助学生建立身份认同
D.协助学生培养社会责任感
```

> **正确答案：** B

**解析：**
学校社会工作者提高学生社交能力的第一步是B 协助学生认识自我。这为后续步骤（如建立身份认同或培养社会责任感）奠定了基础，确保学生能有效管理各种关系，促进正向社交行为并减少负面行为。
A 协助学生认识自己所在的城市或农村：这涉及环境认知，但与环境相关的知识（如社区文化）是社交能力的辅助因素，而非基础。它不直接解决自我理解或人际互动问题，因此不适合作为“第一步”。


B 协助学生认识自我：这是社交能力发展的基石。自我认知包括了解自己的情感、动机、优点、缺点、兴趣和价值观等，它使学生能够反思自己在社交中的行为和反应，从而更有效地沟通、同理他人和管理冲突。
C 协助学生建立身份认同：身份认同（identity formation）涉及个体对社会角色、群体归属（如家庭、文化、性别）的认知和接纳。虽然它影响社交行为（例如，强烈的身份认同可能增强自信），但它不是最基础步骤：身份认同是建立在自我认知之上的更高级过程。
D 协助学生培养社会责任感：这涉及道德发展和社会义务，属于更高层次的社交能力（如公民意识）。它需要以自我认知和身份认同为基础，否则学生可能缺乏内在动机去承担责任，因此不适合作为“第一步”。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_78_0.png) 妈妈要求他的考试成绩一定要在班里名列前茅，但爸爸认为不应给孩子这么大的压力，为此，夫妻俩经常争吵，小强觉得爸爸妈妈的争吵是因为自己而引起的，变得沉默寡言，成绩也下降了。根据家庭系统理论，小强出现上述问题的原因可能是（    ）。

**选项：**
```
A.学习不够努力
B.妈妈对小强要求过高
C.家庭沟通不良
D.爸爸对小强要求过低
```

> **正确答案：** C

**解析：**
根据家庭系统理论，家庭是一个相互影响的系统，家庭成员之间的互动和沟通模式会对每个成员产生影响。在这个案例中，小强的父母因为对他的期望不同而经常争吵，这种争吵可能让小强感到压力和不安，从而导致他变得沉默寡言，成绩下降。这说明家庭内部的沟通存在问题，缺乏有效的沟通和理解，导致了家庭系统的失衡。因此，小强的问题可能源于家庭沟通不良，而不是仅仅因为妈妈要求过高或爸爸要求过低，或者小强自己学习不够努力。家庭系统理论强调的是家庭作为一个整体的功能和互动模式，所以正确答案是C。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_78_0.png) 产期，最近总是很沮丧，觉得生育子女是一件很恐怖的事情，产生了焦虑，为此她求助于医务社会工作者。小林的问题属于医务社会工作者要处理的（    ）。

**选项：**
```
A.心理调适问题
B.疾病适应问题
C.情绪问题
D.生育问题
```

> **正确答案：** C

**解析：**
正确答案为 C（情绪问题）。小林的问题本质是情绪困扰（焦虑、恐惧），而非适应过程的心理重构。医务社会工作者在此类案例中优先处理情绪问题，再逐步介入心理调适。
 C（情绪问题）小林因生育临近产生焦虑、恐惧、沮丧等负面情绪，属于直接的情绪困扰，医务社会工作者需帮助其缓解情绪压力，建立积极心态。此为最精准的归类。


A（心理调适问题）心理调适侧重个体适应环境变化或角色转变（如成为母亲），但题干未提及小林因角色转换产生心理冲突，核心是情绪反应本身，故排除。
B（疾病适应问题）与疾病无关，小林未患生理疾病，仅因生育引发情绪困扰，排除。
D（生育问题）该选项过于宽泛，生育问题可涵盖生理、医疗或社会支持，但题干聚焦于焦虑情绪，而非生育本身的技术性困难，排除。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_78_0.png) 儿童服务是指（    ）的方法。

**选项：**
```
A.以建立有效的沟通模式为目标
B.以家庭和睦为目标
C.以建立严格的家庭规则为目标
D.以儿童获得最佳照顾和保护为目标
```

> **正确答案：** D

**解析：**
选项A提到建立有效的沟通模式为目标，这虽然是家庭功能中的一个重要方面，但并不是以家庭为中心服务的核心目标。选项B强调家庭和睦，这虽然有助于儿童的福祉，但同样不是该服务的主要目标。选项C提出建立严格的家庭规则，这可能与某些家庭管理有关，但并不符合以家庭为中心服务的宗旨。而选项D明确指出以儿童获得最佳照顾和保护为目标，这正是以家庭为中心服务的核心所在，即确保儿童在家庭环境中得到适当的照顾和保护，促进其全面发展。因此，正确答案是D。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_78_0.png) 独自抚养14岁的女儿，面对青春期女儿的叛逆表现，她认为孩子不听话，无法沟通，感到无能为力，向社会工作者小谭求助，在了解情况后，小谭拟开展亲职教育辅导服务。下列服务目标中，恰当的是（    ）。

**选项：**
```
A.协助女儿耐心倾听王女士的想法
B.协助女儿更好地扮演家庭角色
C.协助王女士纠正女儿的不当行为
D.协助王女士了解青春期的特征
```

> **正确答案：** D

**解析：**
选项A：“协助女儿耐心倾听王女士的想法”，这个目标侧重于女儿的行为改变，但忽略了母亲可能也需要学习如何更好地与女儿沟通。因此，这个选项可能不够全面。

选项B：“协助女儿更好地扮演家庭角色”，这个目标同样侧重于女儿的行为改变，而没有直接针对母亲的需求，即如何更好地理解和应对女儿的青春期行为。
选项C：“协助王女士纠正女儿的不当行为”，这个目标可能过于侧重于惩罚或纠正孩子的行为，而没有提供积极的沟通和理解的方法，可能不利于建立健康的亲子关系。
选项D：“协助王女士了解青春期的特征”，这个目标直接针对母亲的需求，帮助她理解青春期孩子的特点和发展需要，从而更好地与女儿沟通和相处。这有助于母亲调整自己的期望和行为，减少冲突，建立更有效的沟通渠道。
因此，选项D是最恰当的服务目标，因为它不仅关注母亲的需要，还为她提供了理解和应对女儿行为的基础，有助于改善亲子关系。其他选项虽然可能在一定程度上有助于解决问题，但它们的侧重点不够全面或可能过于偏向行为纠正，而忽略了理解和发展性的一面。
最终答案是D。

B.开展能力建设
C.服务对象识别

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_78_0.png) 交通事故暂时失去劳动能力，申请临时救助。社区居委会社会工作者小马对其开展评估工作。小马观察到小郭居住在狭窄的平房里，家具简陋，生活用品摆放凌乱。小马采取的上述评估方式是（    ）。

**选项：**
```
A.直接询问
B.家庭探访
C.使用量表
D.观察身体语言
```

> **正确答案：** B

**解析：**
A、直接询问：这通常是指通过对话直接获取信息，但题目中提到的是小马观察到的环境情况，而不是通过对话获得的。

B、家庭探访：符合题意，因为小马实地访问了小郭的住所，观察了他的生活环境。
C、使用量表：这通常是指通过标准化的问卷或评估工具来收集数据，但题目中没有提到使用任何量表。
D、观察身体语言：这指的是通过观察人的非言语行为来评估，但题目中提到的是对居住环境的观察，而不是对人的行为的观察。
因此，正确答案是B.家庭探访。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_78_0.png) 孩子去上哪所学校发生争吵，情急之下还动手打了起来，为此两人闹得不可开交。沟通中社会工作者问他们：“根据刚才你们讲的情况，你们觉得问题出在哪里?”社会工作者这样问是为了（    ）。

**选项：**
```
A.了解问题
B.了解需求
C.界定问题
D.界定需求
```

> **正确答案：** C

**解析：**
社会工作者在与小王夫妻沟通时，提出了一个问题：“根据刚才你们讲的情况，你们觉得问题出在哪里?” 这个问题的目的是引导他们识别和明确他们之间存在的问题所在。在社会工作实践中，界定问题是帮助服务对象明确他们面临的主要问题，以便后续制定解决方案。而了解问题和界定问题虽然听起来相似，但界定问题更侧重于明确问题的范围和性质，而了解问题可能更侧重于收集信息。因此，正确答案是C.界定问题。
