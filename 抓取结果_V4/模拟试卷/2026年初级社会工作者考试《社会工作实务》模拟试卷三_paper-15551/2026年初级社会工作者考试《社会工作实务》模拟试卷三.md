# 2026年初级社会工作者考试《社会工作实务》模拟试卷三

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_78_0.png) 很好，有一个儿子，很乖巧听话，学习也很好。自从儿子离家上大学后，老高的妻子开始郁郁寡欢，感觉很孤独，于是向社会工作者求助。社会工作者分析老高妻子的情况后，向其妻子介绍每一个家庭发展阶段都有不同的任务和要求，家庭成员应及时调整自己的角色和分工。以上案例中社会工作者使用的理论解释是（    ）。

**选项：**
```
A.家庭系统理论
B.家庭生命周期理论
C.生态系统理论
D.结构式家庭治疗理论
```

> **正确答案：** B

**解析：**
A、家庭系统理论关注家庭作为一个系统如何运作，强调家庭成员之间的相互作用和系统内的变化。虽然这个理论可以解释家庭成员之间的互动，但题目中更强调的是家庭生命周期的不同阶段，因此A可能不是最佳选择。

C、生态系统理论关注个体与其环境之间的相互作用，包括微观、中观和宏观环境。虽然这个理论可以解释家庭与外部环境的关系，但题目中的重点在于家庭生命周期的不同阶段，因此C可能不太合适。
D、结构式家庭治疗理论关注家庭的结构和沟通模式，以及如何通过改变这些结构来解决问题。虽然这个理论可以用于解决家庭内部的问题，但题目中的情况更侧重于家庭生命周期的变化，因此D可能不是最佳选择。
综上所述，B.家庭生命周期理论最符合题目中的情境，因此正确答案是B。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_78_0.png) 种与人相处的技巧，良好的亲子关系和师生关系与学生的正向社交行为有正面的关系，良好的同辈关系与学生的负面行为则有反面的关系。因此，学校社会工作者提高学生社交能力的第一步是（    ）。

**选项：**
```
A.协助学生认识自己所在的城市或农村
B.协助学生认识自我
C.协助学生建立身份认同
D.协助学生培养社会责任感
```

> **正确答案：** B

**解析：**
学校社会工作者提高学生社交能力的第一步是B 协助学生认识自我。这为后续步骤（如建立身份认同或培养社会责任感）奠定了基础，确保学生能有效管理各种关系，促进正向社交行为并减少负面行为。
A 协助学生认识自己所在的城市或农村：这涉及环境认知，但与环境相关的知识（如社区文化）是社交能力的辅助因素，而非基础。它不直接解决自我理解或人际互动问题，因此不适合作为“第一步”。


B 协助学生认识自我：这是社交能力发展的基石。自我认知包括了解自己的情感、动机、优点、缺点、兴趣和价值观等，它使学生能够反思自己在社交中的行为和反应，从而更有效地沟通、同理他人和管理冲突。
C 协助学生建立身份认同：身份认同（identity formation）涉及个体对社会角色、群体归属（如家庭、文化、性别）的认知和接纳。虽然它影响社交行为（例如，强烈的身份认同可能增强自信），但它不是最基础步骤：身份认同是建立在自我认知之上的更高级过程。
D 协助学生培养社会责任感：这涉及道德发展和社会义务，属于更高层次的社交能力（如公民意识）。它需要以自我认知和身份认同为基础，否则学生可能缺乏内在动机去承担责任，因此不适合作为“第一步”。
