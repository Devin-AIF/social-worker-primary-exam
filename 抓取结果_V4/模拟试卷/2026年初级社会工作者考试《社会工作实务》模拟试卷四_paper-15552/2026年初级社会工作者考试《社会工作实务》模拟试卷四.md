## 第 3 题 [单选题]

**题目：** 3.![图](images/q_3_79_tit_0.png)骗罪被判有期徒刑2年，缓期执行。杨某一直认为自己运气不好，对法院判决耿耿于怀，经常无故缺席社区服务活动。针对杨某的情况，社会工作者为其提供帮助，宜采取的方法是（ ）。

**选项：**
```
A.加强监督管理
B.培养劳动习惯
C.开展风险评估
D.认知行为治疗

## 第 4 题 [单选题]

**题目：** 4.![图](images/q_4_79_tit_0.png)了解到，某企业部分员工对如何与青春期的子女沟通感到束手无策。为此，小英设计了以学习亲子沟通知识为主要内容的培训课程，回应员工的需要。在此培训中，小英扮演的角色是（ ）。

**选项：**
```
A.促进者
B.联络者
C.教育者
D.倡导者
```

> **正确答案：** C

**解析：**
A、促进者：促进者通常是指那些鼓励和支持他人，帮助他们实现目标的人。虽然小英可能在培训中起到一定的促进作用，但她的主要角色并不是促进者。

B、联络者：联络者主要是负责沟通和协调工作，确保各方之间的联系畅通。小英在这个情境中并没有主要负责联络工作，而是专注于培训内容的设计和实施。
D、倡导者：倡导者是指那些为某个特定的事业或目标发声，争取支持和资源的人。小英虽然可能在某种程度上倡导亲子沟通的重要性，但这并不是她在这个情境中的主要角色。
综上所述，小英在这个培训课程中的主要角色是教育者，因此正确答案是C。

B.机构转介的服务对象

> **正确答案：** D

**解析：**

C、由他人介绍的服务对象：指的是通过他人介绍而来的服务对象，但题目中是小徐在走访中发现的，而不是他人介绍的，所以C不正确。
D、外展发现的服务对象：指的是社会工作者通过主动外展工作发现的服务对象，这符合题目中的情况，因为小徐是在走访中主动发现小强的，所以D正确。

## 第 7 题 [单选题]

**题目：** 7.![图](images/q_7_79_tit_0.png)业特别拖拉，经常边写作业边玩手机，针对这种状况，学校社会工作者小王计划运用“行为契约法”帮助小亮养成良好的行为习惯。根据该方法的具体要求，小王首先应该帮助小亮（ ）。

**选项：**
```
A.确定目标行为
B.商定奖惩清单
C.书写行为契约
D.举行承诺仪式
```

> **正确答案：** A

**解析：**
行为契约法是一种通过明确目标、制定计划、实施监控和反馈来改变不良行为的方法。在这个过程中，首先需要明确的是小亮需要改变的具体行为，也就是确定目标行为。只有明确了需要改变的行为，才能进一步商定奖惩清单、书写行为契约和举行承诺仪式。因此，小王首先应该帮助小亮确定他需要养成的良好行为习惯，比如专心写作业，减少玩手机的时间等。这一步是整个行为契约法的基础，后续的步骤都是基于这个目标展开的。所以正确答案是A。

```
A.协助小明分析害怕上学的原因
B.帮助小明处理结案时的离别情绪
C.帮助小明回顾会谈过程中的成长
D.鼓励小明把会谈学到的知识用于生活中
```

> **正确答案：** C

**解析：**
选项A：协助小明分析害怕上学的原因。这可能与题目中的问题不直接相关，因为题目提到的是注意力不集中，而不是害怕上学，所以A选项可能不太合适。

综上所述，最适宜的做法是C，即帮助小明回顾会谈过程中的成长，以强化已有的转变成效。

**题目：** 10.![图](images/q_10_79_tit_0.png)多是无儿无女、无亲无故的孤老优抚对象。前些天，与老曾一起住了20多年的老曲过世了，老曾十分悲伤，情绪低落，不愿见人。社会工作者组织光荣院院友们坐在一起，表达对老曾的理解，分享对生命的感悟。社会工作者组织的小组是（ ）。

**选项：**
```
A.教育小组
B.成长小组
C.治疗小组
D.支持小组
```

> **正确答案：** D

**解析：**
A、教育小组：这类小组的目的是通过教育和学习，帮助成员获得知识和技能，以解决个人或社会问题。题目中的情境更侧重于情感支持和心理调适，而不是教育和学习，因此A选项不太合适。

B、成长小组：成长小组通常关注成员的个人成长和发展，帮助他们克服挑战，实现自我提升。虽然老曾可能需要成长，但题目中的主要目的是提供情感支持，而不是促进个人成长，所以B选项也不太合适。

D、支持小组：支持小组的目的是为成员提供情感支持和鼓励，帮助他们应对生活中的困难和挑战。题目中的情境正好符合这一特点，院友们坐在一起表达理解，分享感悟，就是为了给老曾提供情感支持，帮助他度过难关。因此，D选项是最合适的。

```

> **正确答案：** A

**解析：**
通过动员社区内部的人力资源（骨干、志愿者）和意识资源（环保理念），机构实现了资源整合的目标，因此正确答案为 A资源整合。

资源整合是指将社区内分散或未被充分利用的资源（如人力资源、组织资源、意识资源）进行挖掘、组织和协调，使其形成有机整体，提升资源利用效率。
题目中的实践：培育“新乡贤”：挖掘社区内有影响力的骨干力量（人力资源整合）；发展“绿色志愿者”：组织居民参与志愿服务（组织资源整合）；倡导环保活动：提升居民环保意识（意识资源整合）。
B资源共享：强调不同主体之间共享资源（如跨社区合作），而题目未涉及外部资源共享，仅聚焦内部资源动员。
C资源配置：指根据资源特性进行分配和优化组合（如物资调配、资金分配），但题目中的做法更侧重于资源的组织与协调，而非单纯的分配行为。


D资源互补：需不同资源之间相互补充短板（如技术与资金互补），而题干未体现资源间的互补性，而是资源的内生整合。【提示1】资源配置：通常指根据需求对资源进行分配（如资金划拨、物资分配），但题干中的实践更强调资源的组织与系统化利用（如动员人力、凝聚意识），属于整合而非分配。
【提示2】资源整合强调对社区内部资源的挖掘与组织（如将居民转化为志愿者、乡贤转化为骨干）；资源共享强调不同主体间的资源互通（如跨社区合作）。题目中所有措施均围绕激活社区内部资源展开，未涉及外部资源引入或共享，答案A无误。
