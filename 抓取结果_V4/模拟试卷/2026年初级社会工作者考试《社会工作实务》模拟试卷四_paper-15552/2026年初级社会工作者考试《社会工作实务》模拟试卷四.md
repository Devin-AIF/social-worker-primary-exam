# 2026年初级社会工作者考试《社会工作实务》模拟试卷四

## 第 79 题 [多选题]

**题目：** 19. ![图](./images/q_79_79_0.png) 企业未能有效履行社会责任的是（    ）。

**选项：**
```
A.裁减工人
B.不尽社区义务
C.搞不正当竞争
D.违章经营作业
E.制假售假
```

> **正确答案：** CDE

**解析：**
AB选项的争议点：
A 裁减工人：若企业因经营困难依法裁员（如提前通知、支付法定补偿），属于合法行为，不违反社会责任。但若裁员程序违法（如暴力裁员、不补偿），则属于责任缺失。原题表述未区分合法与非法情形，需补充条件。
B 不尽社区义务：企业社会责任中的“社区义务”并非强制性法律责任，而是倡导性要求。若企业仅未参与社区活动但未违反法律，可能存在伦理争议，但不必然构成“未能有效履行”。需明确“完全拒绝参与”或“长期忽视”等具体行为。
CDE均符合题意：
C 搞不正当竞争（如虚假宣传、商业贿赂）直接违反《反不正当竞争法》，属于明确的法律责任缺失。
D 违章经营作业（如违规排污、忽视安全生产）违反《环境保护法》《安全生产法》，属于违法且危害公共利益的典型行为。
E 制假售假违反《产品质量法》《消费者权益保护法》，直接侵害消费者权益，属于法律与伦理双重失责。
