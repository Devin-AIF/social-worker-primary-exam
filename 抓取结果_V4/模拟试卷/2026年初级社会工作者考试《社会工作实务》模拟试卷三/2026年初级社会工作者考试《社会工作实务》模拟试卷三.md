# 2026年初级社会工作者考试《社会工作实务》模拟试卷三

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 接待了因孩子沉迷手游拒绝学习而前来求助的吴女士，吴女士称已没收孩子的手机但孩子绝食抗议，多次沟通无果。在接案会谈中，小唐首先邀请吴女士详细描述没收手机后孩子的具体反应和此前尝试过的沟通方式，小唐做法的核心目的是（    ）。

**选项：**
```
A.快速制定孩子沉迷手游的干预方案
B.收集问题客观信息以明确成因
C.评判吴女士教育孩子的方式
D.积极联系学校共同干预
```

> **正确答案：** B

**解析：**
接案阶段的核心任务

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) 机构接收了街道转介的服务对象老王。老王因独居且高血压并发症发作，无人照料被转介。初次会谈时，老王反复说“我老了没用了，没人要我”,对服务表现出抗拒。此时社会工作者最恰当的回应是（    ）。

**选项：**
```
A.“街道帮你转介了，说明还是有人关心你”
B.“我会帮你联系护工，你不用怕”
C.“你觉得没人管你，能具体说说最近遇到的困难吗”
D.“你要积极配合服务，才能改善现状”
```

> **正确答案：** C

**解析：**
接案会谈技巧

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_80_0.png) 社区青少年犯罪预防制定服务计划时，关注对象明确为青少年本人、家长、学校德育老师、社区民警。该关注对象的设定核心是为了（    ）。

**选项：**
```
A.聚焦青少年自身改变
B.明确介入需改变的多系统主体
C.依赖外部环境支持
D.简化服务流程，降低工作难度
```

> **正确答案：** B

**解析：**
服务计划的构成

---

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_80_0.png) 在为因失恋而出现自伤倾向的大学生王明开展服务时，在介入前记录王明的自伤频率、情绪状态等基础数据，介入1个月后再次测量这些指标，对比发现王明自伤次数从每周5次减少至每周2次，情绪量表得分从90分降至53分。小玲通过介入前后数据对比判断服务有效的做法，体现了评估阶段的哪项核心功能?（    ）

**选项：**
```
A.验证服务对象的主观感受
B.为调整介入计划提供依据
C.通过数据对比，客观衡量介入成效
D.直接确定后续服务是否终止
```

> **正确答案：** C

**解析：**
评估的核心功能
