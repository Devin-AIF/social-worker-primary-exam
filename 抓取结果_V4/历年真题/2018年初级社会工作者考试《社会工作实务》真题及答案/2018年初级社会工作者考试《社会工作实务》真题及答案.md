# 2018年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 发动村里的妇女成立了“巧娘草编合作社”。近期，镇政府将举办手工艺大赛，妇女们的参与积极性很高，社会工作者小王运用增能的方法，协助妇女们为大赛做准备，其适宜的做法是（    ）。

**选项：**
```
A.让“巧娘”们自己商量制作参赛作品
B.由村里的能人帮助改进“巧娘”们的作品
C.请工艺美术大师设计产品，由“巧娘”们制作
D.由小王从以往“巧娘”们的作品中挑选有特色的去参赛
```

> **正确答案：** A

**解析：**
本题考查妇女社会工作的方法。妇女增能是通过社会工作者和妇女之间的共同合作，让妇女有能力认识到自身的真实处境，引发她们对形成这种境况的社会因素进行思考，寻找解决的途径，并且通过采取具体的行动来改善处境。妇女增能的主要方法有：
(1)接纳服务对象对问题的看法，让服务对象感到被尊重，平等对待以及可自我决定；(2)将工作过程透明化，破除神秘色彩；(3)帮助服务对象整合与动员资源，提供获得资源的机会；(4)帮助服务对象分析权利关系，两性不平等是导致问题产生的根源；(5)帮助服务对象相信自己具有改变的能力，协助服务对象产生自尊、责任感、能力感、自我概念；(6)意识觉醒。通过阅读、小组讨论、经验分享及观看影片等方式观察和了解妇女的弱势社会地位，为进一步改变建立必要的思想基础。A项的做法体现了妇女增能方法的宗旨。(考点：妇女社会工作)

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) 新员工，许多应聘者不了解招聘岗位存在的职业风险，社会工作者小周为此协调人事部门设计制作了岗位说明书，详细介绍了招聘岗位的操作规程、潜在的职业危害与职业病防护措施，并与前来应聘的人员进行了面对面交流。小周在上述工作中扮演了（    ）角色。

**选项：**
```
A.利益协调者
B.资源联结者
C.矛盾调解者
D.咨询辅导者
```

> **正确答案：** D

**解析：**
本题考查企业社会工作者的角色。作为咨询辅导者，企业社会工作者主要是为企业职工就职业和生活问题提供咨询辅导，对员工的需求、工作环境和工作条件进行明确，协助其分辨困扰问题，并挖掘各种方法和手段帮助其解决问题，以提高员工适应环境的能力。题干中社工人员的做法就鲜明地体现了“咨询辅导者”角色。(考点：企业社会工作)

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_80_0.png) 对象。老范想依靠自己的努力，减少对低保救助的依赖。因要照顾卧病在床的妻子，他只能就近打些零工，与邻居几乎没有往来。社会工作者小桃协助老范与邻居结成帮扶伙伴，彼此互帮互助。小桃的这种做法属于救助社会工作中的（    ）服务。

**选项：**
```
A.提供心理支持
B.开展能力建设
C.调节家庭关系
D.促进社会融入
```

> **正确答案：** D

**解析：**
本题考查社会救助社会工作的内容。题干中的“与邻居几乎没有往来”是解题的题眼。促进社会融入是指社会工作者要为低保家庭创造参与社区活动的机会，鼓励他们参与社区的公益和文娱活动，帮助他们建立和其他社区居民的联系，互帮互助，承担一定的责任和义务，增加他们的成员感、归属感和自信心。社会工作者要积极帮助困难群体修复社会关系，建立社会支持网络，尤其是邻里支持。(考点：社会救助社会工作)

**题目：** 5. ![图](./images/q_5_80_0.png) 时，因丈夫有外遇而与其离婚，她获得了孩子的监护权，出于怨恨，小张不让前夫见孩子，也拒绝了前夫支付给孩子的抚养费，以及孩子奶奶帮忙照看孩子的请求，但小张既要工作又要照顾孩子，心力交瘁，向社会工作者小王寻求帮助。根据性别视角的工作原则，小王应该做的是（    ）。

**选项：**
```
A.让小张认识到单亲家庭对孩子的负面影响
B.动员小张接受孩子奶奶帮忙照顾孩子的建议
C.与小张分析现状.让其明白自己弱势的状况和原因
D.告诉小张.前夫有见孩子的权利和支付抚养费的义务
```

> **正确答案：** C

**解析：**

## 第 14 题 [单选题]

**题目：** 14.49 ![图](./images/q_14_80_0.png) 去世后经常失眠，精神恍惚，总觉得有人要害他，在家中自言自语、破口大骂，甚至砸坏家具，殴打家人。女儿将陈某送到某综合医院的精神科治疗，陈某在诊室里大吵大闹。医生初步治疗后，请社会工作者小吴来协助工作，小吴以下的做法中，最适宜的是（    ）。

**选项：**
```
A.鉴于陈某病态的暴力倾向，建议将陈桌单独隔离
B.向陈桌隐瞒病情。避免陈某得知病情后产生病耻感
C.帮助陈某分析发病原因。鼓励他表达自己内心的想法
D.稳定和安抚陈某情绪，促进其适应医院环境并配合检查
```

> **正确答案：** D

**解析：**
A不适宜。单独隔离是医疗处置措施，应由医护人员根据病情判断后实施，社会工作者不能擅自提出此类建议。且隔离并非首选的心理社会干预手段。
B错误。向服务对象隐瞒病情违背了社会工作尊重、诚信和知情同意的专业价值观，也不利于后续治疗的有效配合。
C不适宜。在服务对象处于精神科诊疗初期、情绪极不稳定的状态下，直接帮助分析发病原因为时过早。应在先稳定情绪、建立信任关系后再逐步开展。
D正确。医生已对陈某进行初步治疗，此时陈某最需要的首先是情绪稳定和医院环境适应。小吴作为医务社会工作者，首要任务是安抚其情绪、帮助其适应住院环境、建立对医护人员的信任感，为后续的心理支持和治疗干预奠定基础。这与医务社会工作中“住院适应”和“心理支持”的核心服务内容完全吻合。
结论：正确答案为D。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_80_0.png) 机构在对口扶贫村开展精准脱贫工作。该机构在当地政府支持下，协助村民成立了养殖合作社，合作社理事会成员有一半是贫困妇女，并形成了妇女参与合作社事务决策的相关制度。这一做法（    ）。

**选项：**
```
A.满足了妇女的实用性社会性别需求
B.满足了妇女的战略性社会性别需求
C.既满足了妇女的实用性社会性别需求，又满足了战略性社会性别需求
D.既没满足妇女的实用性社会性别需求，又没满足战略性社会性别需求
```

> **正确答案：** C

**解析：**
本题考查妇女社会工作的方法。实用性社会性别需求是指在社会生活中的基本需求，如健康、就业、学习育儿经验、改善生活条件等。战略性社会性别需求主要是指获得平等权利和社会认可的需求，如参与和管理社区事务、享受更高教育等。题干中，“协助村民成立了养殖合作社，合作社理事会成员有一半是贫困妇女”，满足了妇女的实用性社会性别需求；“形成了妇女参与合作社事务决策的相关制度”，满足了妇女的战略性社会性别需求。(考点：妇女社会工作)

## 第 18 题 [单选题]

**题目：** 18. ![图](./images/q_18_80_0.png) 服务的社区有一些初中毕业后待业在家的青少年，他们常成群结伙在社区内闲逛，喝酒嬉闹。为预防这些青少年发生越轨行为，小李应（    ）。

**选项：**
```
A.加强对他们的管理和控制
B.协助司法机关对他们进行社会调查
C.以合适成年人的身份为其提供帮教服务
D.设立“人生导师计划”为其提供社会支持
```

> **正确答案：** D

**解析：**
正确答案为 D，设立支持性计划更契合预防目标。
选项C：合适成年人制度通常针对已涉案的未成年人（如被警方询问时需监护人到场），而题干中的青少年尚未发生越轨行为，仅是存在风险。提供帮教服务（C）属于司法矫正范畴，与预防性干预不符。错误。
选项D（人生导师计划）：①预防性干预：通过建立正向支持关系（如导师引导职业规划、兴趣培养），填补青少年社会支持缺失，减少因无聊、缺乏目标导致的越轨行为。②发展性社会工作理念：强调赋能而非管控，符合社工“助人自助”原则。正确。
选项A（加强管控）：易激化对立，违背信任建立原则。错误。
选项B（司法调查）：青少年未违法，司法介入无依据。错误。
