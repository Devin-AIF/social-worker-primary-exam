# 2017年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 父母前几年因车祸双亡，爷爷奶奶靠微薄的养老金抚养他。尽管生活艰辛，但爷爷奶奶非常疼爱小李，老师和同学也都很喜欢和关心他。小李有自己的理想，学习努力，成绩优秀。从抗逆力理论视角看，上述案例中，对小李健康成长最重要的影响因素是（    ）。

**选项：**
```
A.个体的坚强与忍耐
B.支持性人际与社会关系
C.个人较好的情绪管理能力
D.家人与老师的殷切期待
```

> **正确答案：** B

**解析：**
构成个体抗逆力的要素有很多，综合起来主要体现在三个方面：①外部支持因素。包括拥有正向的连接关系、坚定清晰的规范、关怀支持的环境、积极合理的期望、有意义的参与机会。②内在优势因素。包括完美的个人形象感、积极乐观感。③效能因素。包括人际技巧、解决问题能力、情绪管理及目标制定等。案例中，爷爷奶奶非常疼爱小李，老师和同学也都很喜欢和关心他，可以看出，对小李健康成长最重要的影响因素是抗逆力中的外部支持因素，即支持性人际与社会关系因素。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) 正在为心身疾病患者制订减压小组工作方案，督导者老张提醒小李，在小组中除了要安排转化患者不良情绪、改变他们的不合理认知等内容之外，还应安排（    ）加以配合。

**选项：**
```
A.果敢训练
B.敏感性训练
C.放松训练
D.沟通技巧训练
```

> **正确答案：** C

**解析：**
慢性疾病患者及长期照顾者因疾病而产生的压力包括来自于个人、家庭、工作（学业）和社会等方面。压力管理是社会工作的主要服务内容，具体包括：①社会工作者通过咨询，帮助患者及家庭辨明压力来源，认知压力的表现，以便于更好地管理压力；②根据压力的程度可以采取综合干预方法，如咨询、训练、心理治疗等综合干预；③由于患者和照顾者的压力互相影响，社会工作者需对患者和照顾者共同进行评估和干预；④压力管理要结合心理教育，为患者及家属提供心理情绪支持、连接社会支持，纾缓患者及家属的压力；⑤以冥想为基础的减压课程、心理疗法、锻炼和放松训练等也是比较常用的方法。故本题中，针对减压小组的工作方案，还应安排放松训练加以配合。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_80_0.png) 为孙子的教育问题经常争吵，婆媳关系紧张，李奶奶对此很苦恼，希望得到社会工作者小王的帮助。为了了解李奶奶的问题和需要，帮助李奶奶识别和善用环境中的积极因素，小王应做的是（    ）。

**选项：**
```
A.了解李奶奶的背景资料
B.明确李奶奶的问题所在
C.了解李奶奶曾经得到的帮助
D.分析李奶奶问题的成因
```

> **正确答案：** C

**解析：**
预估的主要任务有：①识别服务对象问题的客观因素。②识别服务对象问题的主观因素。③识别服务对象问题的成因及使问题延续的因素。④识别服务对象及环境的积极因素。社会工作者要注意，预估不仅要集中在服务对象的“问题”上，更重要的是要找出服务对象系统内外的资源，并运用这些资源去帮助他们。识别环境中的积极因素即是运用专业知识去思考、辨认情境中需要改变及可以成为改变资源的部分，是将着眼点放到服务对象的能力和优势上。⑤决定提供服务的方式和内容。题中，为了帮助李奶奶识别和善用环境中的积极因素，需要思考情境中可以成为资源的部分，即了解李奶奶曾经得到的帮助，帮助李奶奶获取过去的有用资源。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_80_0.png) 在为脑瘫儿童小强提供服务的过程中，整合了教育、护理、康复等不同专业的服务与资源，回应了小强的多重需要，增强了小强对社会环境的适应能力。小刘所运用的方法属于（    ）。

**选项：**
```
A.社区康复
B.个案管理
C.教育康复
D.需求评估
```

> **正确答案：** B

**解析：**
残疾人社会工作的主要方法有：①个案管理；②社区康复方法；③社会康复方法；④职业康复方法；⑤教育康复方法；⑥促进残疾人就业权利实现的方法。其中，个案管理的特点有：①由社会工作者整合协调不同专业服务和资源为某个或某种残疾人提供全面性的服务；②增强残疾人获取资源并运用资源网络的能力，提升对社会环境的适应能力。社会工作者在个案管理中的角色是“资源整合者、价值倡导者、服务咨询者”。故本题中，小刘所运用的方法属于个案管
理。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_80_0.png) 完成了小秦的个案服务并顺利结案，计划接下来的3个月内定期对小秦进行电话回访。小王的上述工作安排属于（    ）。

**选项：**
```
A.跟进服务
B.成效评估
C.情绪安抚
D.服务咨询
```

> **正确答案：** A

**解析：**
社会工作者要在服务结束后的一段时期内定期对服务对象进行回访和跟踪，了解他们的情况和服务需要，此即跟进服务。跟进服务有以下四个实施方法：①电话跟进。在结案后一段时间内，社会工作者用电话与服务对象继续联络，了解结案后服务对象的情况。②个别会面。在结案后一段时间内社会工作者根据约定在机构或者服务对象家里与他们会面，以了解他们的情况。③集体会面。这种方式适用于小组跟进。在集体会面中，组员可以共同回顾小组的经历，分享小组结束后的情况，交流各自经验，增强关系和促进相互的支持。④跟进服务对象的社会支持网络。社会工作者可以通过跟进服务对象的社会支持网络（包括家庭、邻居、朋友、单位或者学校老师等）来了解服务对象的情况，以便及时提供相应的服务和必要的支持。题中，小王计划进行定期的电话回访，这属于跟进服务中的电话跟进。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_80_0.png) 习生小丁第一次独立做老年小组工作，心里没底，向督导者张老师请教，张老师给了小丁一些建议。下列建议中，正确的是（    ）。

**选项：**
```
A.根据老年人的节奏调整小组进度
B.老年人参加小组一定要征得家人同意
C.为每位老年人配备一名志愿者，以确保其安全
D.确保老年人在小组内能够任意表达对其他组员的看法
```

> **正确答案：** A

**解析：**
老年人小组工作的特点如下：①老年人与健康的儿童、青少年或年轻成人不同，他们有各种身体上的不便和知觉方面的限制，因此在开办小组的时候就要有相应的调整。如在空间安排和使用辅助器具上要作特别的考虑。②在老年人小组中，带领者可能自始至终都要扮演一个比较积极的角色。小组带领者可能要投入额外的时间与小组成员建立个人关系，老人可能需要社会工作者的持续鼓励才能参加最初的小组聚会和以后的小组活动。③老年人由于其身心健康状况，可能在小组中的表现比较被动，其头脑中可以都是个人问题，这使得老年人小组工作的节奏比年轻人的小组慢许多。对小组带领者来说，要学会欣赏成员点滴的进步。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_80_0.png) 到老伴刚去世的张奶奶家探访，碰到张奶奶正与女儿聊天，聊天过程中，张奶奶表示要修改之前和老伴一起订的遗嘱，还要尽快将老伴留下的字画分给孩子们……张奶奶女儿还悄悄对社会工作者说，她收拾房间时，在张奶奶的枕头下发现不少安眠药。针对上述情况，小林首先要做的是（    ）。

**选项：**
```
A.请亲朋好友陪伴张奶奶
B.对张奶奶进行哀伤辅导
C.对张奶奶进行自杀风险评估
D.请法律志愿者协助张奶奶修改遗嘱
```

> **正确答案：** C

**解析：**
社会工作者需要意识到与老人自杀有关的风险因素，并通过直接的、间接的和行为上的线索加以评估。在评估老人的自杀风险时，可以从以下三方面进行评估：①直接线索；②间接线索；③行为线索。在行为线索中，国外学者提出下面一些线索表明老人会有自杀倾向：企图自杀或者过去自杀过；储存药物；出人意料地留遗嘱或修改遗嘱等。题中，张奶奶想修改遗嘱，并在枕头下藏有安眠药，社会工作者小林首先要对张奶奶进行自杀风险评估。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_80_0.png) 平时生活比较拮据，主要靠老刘的工资收入过日子。最近，老刘因车祸失去了左腿，无法正常工作，家庭陷入困境。老刘整天待在家里，觉得自己没用，情绪低落。针对老刘的情况，社会工作者首先应为其提供的服务是（    ）。

**选项：**
```
A.协助其申请低保
B.促进其社会交往
C.促进其身体康复
D.减轻其心理压力
```

> **正确答案：** A

**解析：**
社会救助的工作原则包括：①救急难。在一个人的生命和生活处于危急时刻，社会救助要在第一时间给予最直接、最有效的物质帮助，以保全生命、保障基本生活。②托底线。社会救助是对社会保险、社会福利等其他社会保障制度的补充，在整个社会保障体系中起着托底的作用。③可持续。对于陷入困境中的社会成员来说，需要政府和社会不间断地支持和帮助才能彻底改变境遇，提高生存能力，改善自身生活质量。故本题中，针对老刘的情况，社会工作者应首先为其提供的服务是协助其申低保，进行物质帮助，保障其基本生活。

## 第 11 题 [单选题]

**题目：** 11. ![图](./images/q_11_80_0.png) 住了6年，近期被诊断为晚期肺癌。原来乐呵呵的许大爷变得情绪低落，整天唉声叹气，对社会工作者小关说自己活不了多久，治疗也没用了，想放弃治疗。此时，为了帮助许大爷，小关适当的做法是（    ）。

**选项：**
```
A.与医生沟通其治疗方案
B.与其社区建立联系
C.鼓励其接受现实保持希望
D.帮助其做出院前的心理准备
```

> **正确答案：** C

**解析：**
社会工作者在机构照顾中扮演多重角色，从工作流程的角度看，在死亡或垂危阶段，由于老人的反应通常为极其恐惧和焦虑、绝望、否定和愤怒、压抑、放弃、讨价还价等，此时社会工作者应鼓励老人接受现实，让其保持希望。社会工作者在老人濒临死亡时，首先要做的工作是提供情感支持。协助老年人及其家人处理伴随濒临死亡而带来的多种复杂的情绪，敞开心扉，处理未了的事宜。

**题目：** 12. ![图](./images/q_12_80_0.png) 家住农村。学习成绩优异，父母长年在外打工，身边的小伙伴也陆续进城打工，感到十分孤单。临近毕业，小杰不知自己该选择继续升学还是进城打工，十分迷茫，心情焦虑。社会工作者计划开展的下列服务中，有助于小杰进行抉择的是（    ）。

```
A.协助小杰学习压力管理技巧

C.协助小杰学习情绪管理方法
D.协助小杰绘制“自画像”，增加自我觉知
```

> **正确答案：** D

**解析：**
协助青少年开展自我探索，是青少年社会工作的重要服务手法之一。自我认识与自我探索服务的主要目标如下：①帮助青少年更清楚地认识自己及未来发展的可能性；②协助青少年发掘内在的潜能，并使之得到充分的发挥；③通过青少年间的互动与分享，强化其自我表达的能力；④提升青少年自我觉察和觉察他人需要的能力；⑤强调青少年间彼此回馈和反应的重要性，不仅帮助了个人的自我成长，也帮助了他人成长；⑥协助青少年能够自我接纳、自我完成，终至自我实现。题中，小杰感到迷茫和焦虑，对未来无所适从，社会工作者应当开展自我探索服务，增强其自我觉知。

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_80_0.png) 骗罪被判有期徒刑2年，缓期执行。杨某一直认为自己运气不好，对法院判决耿耿于怀，经常无故缺席社区服务活动。针对杨某的情况，社会工作者为其提供帮助，宜采取的方法是（    ）。

**选项：**
```
A.加强监督管理
B.培养劳动习惯
C.开展风险评估
D.认知行为治疗
```

> **正确答案：** D

**解析：**
犯罪行为的实施有个人因素的影响。个人因素包括生理因素、心理因素、思想观念、行为特征、生活方式等。矫正社会工作者通过运用专业的理论、知识、方法和技巧，使犯罪者或具有犯罪倾向的违法人员得到生理上、心理上、思想上和行为上的矫正治疗，从而重新融入社会，成为正常成员。本题中，针对杨某的情况，社会工作者宜采取认知行为治疗的方法为其提供帮助，使其认清自己的行为。

## 第 14 题 [单选题]

**题目：** 14. ![图](./images/q_14_80_0.png) 中，社会工作者设置了提高社区青少年践行社会主义核心价值观能力的内容，此内容属于（    ）教育。

**选项：**
```
A.成人
B.健康生活
C.公民
D.家庭生活
```

> **正确答案：** C

**解析：**
从社区教育的服务功能角度看，主要内容如下：①家庭生活教育。通常是运用讲座、展览、小组及宣传活动，灌输家庭沟通和人际相处的态度和方法。②公民教育。主要强调要以社区为依托，以全体社区成员尤其是社区青少年为对象，以提高全民素质和培养“四有”新人为宗旨。③成人教育。主要以因各种原因失学或未能接受正规教育的人士为对象，为他们提供教育机会。④健康教育。主要是以社区为单位，向居民提供健康和预防疾病的知识。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_80_0.png) 释回到社区的顾某提供了就业、社会保障等政策咨询服务，并协助其申领了临时救助。社会工作者的上述做法体现了矫正社会工作的（    ）功能。

**选项：**
```
A.监管
B.矫正
C.服务
D.改造
```

社会工作实务的通用过程中的预估是依据既定情境中的事实与特点推论出有关服务对象问题含义的暂时性结论的逻辑过程。它是一个认识服务对象情况的过程，目的在于达到对服务对象、服务对象的问题以及他们所处环境的了解，并对之形成概念化的认识，从而建构一个计划去解决或消除问题。预估的主要任务有：①识别服务对象问题的客观因素。具体包括：a.服务对象的背景资料；b.服务对象所处的环境，与其生活有关的重要系统的资料；c.问题发生与持续的时间；d.服务对象为解决问题所作的努力，使用过的处理问题的方法等。②识别服务对象问题的主观因素。主观因素是指服务对象对问题的实际感受。③识别服务对象问题的成因及使问题延续的因素。④识别服务对象及环境的积极因素。⑤决定提供服务的方式和内容。故本题中，小侯所做的工作属于社会工作服务过程中的“预估”阶段应完成的任务。

## 第 61 题 [多选题]

**题目：** 1. ![图](./images/q_61_80_0.png) 活动中，组员表达了各自对生活的愿望，社会工作者将其归纳如下：广泛参与社会生活，保持健康的生活方式，享有和谐、安全、宜居的家庭氛围和社会环境。上述愿望体现了老年人对（    ）的需要。

**选项：**
```
A.婚姻家庭
B.居家安全
C.健康维护
D.就业休闲
E.社会参与
```

> **正确答案：** BCE

**解析：**
本题考查老年人的需要分类。根据老年社会工作实务理论，老年人的需要主要包括健康维护、经济保障、就业休闲、社会参与、婚姻家庭、居家安全、善终安排等。题干中组员表达了三个愿望，分别对应不同的老年人需求类别，分析如下：
B居家安全（正确） ：题干中"享有和谐、安全、宜居的家庭氛围和社会环境"中的"安全、宜居"直接指向老年人居住环境的物理安全（如防跌倒、无障碍设施）与社会环境的安全感，这正是老年人居家安全需要的核心内容。因此该选项正确。
C健康维护（正确） ：题干中"保持健康的生活方式"强调通过积极的生活方式维持身体健康，这属于最基础、最核心的老年人健康维护需求，即关注疾病预防、康复服务、健康促进等。因此该选项正确。
E社会参与（正确） ：题干中"广泛参与社会生活"强调老年人继续融入社会，参与各类社会活动，表达自身意愿，维护自身权益，这正是社会参与需求的直接体现。因此该选项正确。
其他选项不选的原因：
A婚姻家庭（错误） ：虽然题干提及"家庭氛围"，但其所强调的核心是"安全、宜居"，即侧重于物理环境和社区环境的安全感，并未具体指向夫妻关系、子女关系、代际支持等婚姻家庭需求的核心要素。老年人的婚姻家庭需要主要涉及亲情关系、配偶支持、子女赡养等内容，与题干描述不符，故不应选。
D就业休闲（错误） ：题干中"广泛参与社会生活"可能包含休闲活动，但该表述的核心仍在于"参与"本身，且题干并未明确指向"就业"或具体的休闲活动安排，因此"社会参与"是更全面、更准确的概括，故不应选。
