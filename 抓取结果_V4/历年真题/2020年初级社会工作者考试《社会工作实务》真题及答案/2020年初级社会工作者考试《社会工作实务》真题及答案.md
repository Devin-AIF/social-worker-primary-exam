# 2020年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_79_0.png) 接待了前来求助的王女士母子。王女士告诉小李，儿子沉迷网络游戏不爱学习，父母劝他也不听，还与父母顶撞，王女土希望社会工作者小李能教育儿子小强。小李在确认小强有接受服务的意愿后，与王女士母子签订了服务协议。小李的上述做法属于社会工作运用过程中的（）

**选项：**
```
A.接触
B.接案
C.预估
D.会谈
```

> **正确答案：** B

**解析：**
接案阶段的主要工作和步骤包括：了解服务对象的求助原因和求助过程、初步评估服务对象的问题、决定是否接案、订立初步协议。根据题干签订了初步服务协议，得知属于接案阶段。故答案为B。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_79_0.png) ，社会工作者小李和小刘对是否需要继续为外来务工青年大规模提供服务产生了分歧，小李认为大名已经返回原籍，个案就结案了。小刘则建议联络大名户籍所在地的社会工作服务机构，由他们继续为大名服务，以巩固大名改变的信心和成果，小刘上述建议依据的是对结案阶段（）的要求。

**选项：**
```
A.转介服务
B.跟进服务
C.时机选择
D.情绪处理
```

> **正确答案：** B

**解析：**
结案并不意味着社会工作服务就结朿了，社会工作者要在服务结束后的一段时期内定期对服务对象进行回访和跟踪，了解他们的情况和服务需要，这就是跟进服务。跟进服务是结案阶段的有机组成部分。根据题干小刘建议联络大名户籍所在地的社工服务机构，由他们继续服务，是对结案后的回访和跟踪，是对结案阶段跟进服务的要求。故本题答案为B

---

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_79_0.png) 后，张女士与丈夫的关系变得原来越冷漠，丈夫逐渐染上了酗酒恶习，对家里的事不闻不问，还多次动手打张女士。无奈之下，张女士向社会工作者求助。为了帮助张女土界定其夫妻间存在的问题，下列社会工作者的提问中，最适宜的事（）

**选项：**
```
A.“你希望我们提供什么帮助？”
B.“你认为自己目前遭遇的问题是什么？”
C.“你认为丈夫染上酗酒恶习的原因是什么？”
D.“你认为你们夫妻关系恶化的原因是什么？”
```

> **正确答案：** B

**解析：**
服务对象自己对问题的看法是界定问题时最重要的起点,因此,在使用沟通技巧与服务对象会谈时,服务对象关心的问题、他们的困惑即是界定问题的入手点。从询问服务对象对自己问题的看法开始来界定问题,其道理在于,服务对象对自己问题的看法可以让社会工作者了解服务对象希望改变的是什么

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_79_0.png) ，李女士与丈夫经常因是否接送孩子上下学发生冲突，李女土认为，孩子从小体质弱，比较敏感，即使上了初中，为安全起见还是应该接送，但丈夫不以为然，还以工作为由将接送的事甩给了李女士。李女士对丈夫的行为很不解，怀疑丈夫有了“外心”，非常苦恼、困惑，遂求助社会工作者小齐。为了识别李女士问题产生的主观因素，小齐应该了解（）

**选项：**
```
A.丈夫对夫妻问题的看法
B.李女士为处理夫妻问题所作的努力
C.李女士对夫妻问题的看法
D.丈夫为处理夫妻问题所作的努力
```

> **正确答案：** C

**解析：**
所谓主观因素是指服务对象对问题的实际感受。识别问题的主观因素即是要认识服务对象是如何看待自己的问题的，站在他的角度来理解这些问题对他的意义是什么、他为什么会有如此的主观理解、它们对服务对象现在的社会一心理影响是什么，即认识问题在服务对象处境中的意义。故答案C符合题意

```
A.顺序性
B.个体差异性
C.不均衡性
D.分化与互补性
```

> **正确答案：** B

**解析：**
个体差异性：儿童生长发育的个体差异性是指由于遗传和环境因素的影响，儿童在具有整体共同特征的基础上，在身心发展的表现形式、内容和水平等方面，都有不同于整体特征的个性化特点，具有不同于他人的成长轨迹。根据题干儿童成长发育有早有晚得知答案为B

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_79_0.png) 机构在社区招募中小学生夏令营营员。租住在本社区的外来务工人员老张觉得这是一个让孩子增长见识、提升能力的好机会，想给儿子乐乐报名，但他又担心孩子不是本地户籍，可能没有资格参加，在报名处徘徊犹豫。社会工作者小杜主动向老张询问了解情况，并协助老张完成报名流程，乐乐成为夏令营营员。小杜的做法体现了儿童社会工作中的（）原则。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_79_0.png) 在社区开展0-3岁婴幼儿父母亲职教育服务项目，在对朵朵家进行家访时，9个月大的朵朵见到小燕就大哭，朵朵妈妈告诉小燕，朵朵面对生人或进入陌生环境就会哭闹。小燕从积极人格培养的角度，建议妈妈对朵朵进行信任人格的培养。下列小燕给朵朵妈妈的建议中最适宜的是（）

**选项：**
```
A.让他想跟谁玩就跟谁玩
B.尽可能多带她到陌生的地方
C.尽可能多留出时间陪伴她
D.尽可能多带她到人多的地方
```

> **正确答案：** C

**解析：**
9个月大，属于埃里克森人格发展理论的婴儿前期，这个阶段的儿童最为孤弱，因而对成人依赖性最大，如果护理人能以慈爱和惯常的方式来满足儿童的需要，他们就会形成基本信任感。如果他们的母亲拒绝他们需要或以非惯常的方式来满足他们的需要，儿童就会形成不信任感。婴儿是否得到了充满爱的照料、他们的需要是否得到了满足、他们的啼哭是否得到了注意，这都是他们人格发展中的第一个转折点。需要得到了满足的儿童，会产生基本的信任感。所以最适宜的建议就是让妈妈尽可能的多留出时间陪伴她。故本题答案为C

**题目：** 28. ![图](./images/q_28_79_0.png) 患有多种慢性病。最近，张爷爷突发脑梗，导致日常生活起居、外出行动和就医等都出现了困难。社区居委会将张爷爷转介给社会工作者小袁。小袁初步了解了张爷爷的情况后，决定将张爷爷列为个案管理工作的服务对象。在与张爷爷建立了专业关系后，小袁下一步需要完成的工作是（）。

```
B.与张爷爷签订个案管理服务契约
C.遂请邻居作为志愿者照顾张爷爷
D.为张爷爷制订个案管理服务方案
```

> **正确答案：** A

**解析：**
张爷爷因脑梗导致行动困难，属于残疾人服务范畴。根据残疾人个案管理的工作模式，其步骤明确为：建立关系 → 评估需求 → 制定计划 → 整合资源 → 实施 → 结案。因此，建立专业关系后，下一步应是评估需求（选项A），而非直接签订服务契约。


残疾人个案管理的特殊性：①评估优先性：残疾人服务需全面了解其身体、心理、社会支持等多维度需求，评估是制定个性化服务方案的前提。②契约签订的时机：服务契约通常在评估后与计划阶段同步签订，以确保契约内容基于具体需求和目标，而非在接案时草率确定。
A 评估需求：符合残疾人个案管理流程的第二步，是制定有效服务方案的基础。
B 签订契约：在普通个案管理中可能是接案阶段的一部分，但在残疾人服务中需以评估为依据，故不优先。
C 邀请志愿者：属于介入阶段行动，需在评估和计划后实施。
D 制定方案：需基于评估结果，属于后续步骤。
【提示】若题干未明确为残疾人服务，普通个案管理可能选B（签订契约）。但本题中张爷爷的残疾属性（脑梗致残）适用残疾人个案管理流程，故评估需求（A）为正确答案。
