# 2021年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_76_0.png) 注意到每次说到爸爸妈妈离婚这件事时，小刚都会发火或者沉默。大张求教心理学专家魏教授与其讨论了有关青少年心理发展的问题，试图通过会谈帮助小刚认识父母离婚对自己的影响。大张在与小刚会谈时以“父母离婚时你有什么感受？来协助小刚表达自己的想法。上述大张收集有关小刚资料的方法是（）

**选项：**
```
A.询问
B.咨询
C.观察
D.调查
```

> **正确答案：** A

**解析：**
收集资料的方法中的询问的概念。询问是直接向服务对象询问，通过面对面的会谈收集资料，可以为决定介入与干预的方法提供依据。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_76_0.png) 控常态化以来，社会工作者小王发现，某社区的很多老年人由于不会使用智能技术在购物、出行和就医时遇到诸多不便。许多老年人说我们老了，跟不上时代了学不会这些新东西啊小王计划设计协助老人跨越ˆ数字鸿沟ˉ的服务项目饲以解决老年人的上述问题。为此，小王除了需要对造成老年人问题的客观因素进行预估外，还应对造成老年人问题的主观因素加以识别。小王下列做法中属于主观因素识别的是（）

**选项：**
```
A.了解老年人对自己问题的看法
B.分析老年人处理问题采用的办法
C.了解老年人为解法问题所做的努力
D.询问老年人问题发生的时间和地点
```

> **正确答案：** A

**解析：**
所谓主观因素是指服务对象对问题的实际感受。识别问题的主观因素即是要认识服务对象是如何看待自己的问题的，站在他的角度来理解这些问题对他的意义是什么、他什么会有如此的主观理解、它们对服务对象现在的社会一心理影响是什么，即认识问题在服务对象处境中意义.

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_76_0.png) 作行业协会将党史学习教育和专业服务相结合，积极践行社会工作为人民服务的专业价值理念组织部分高校社会工作专业师生对口高村开展社会工作，助力乡村振兴服务。在制订服务计划时，社会工作专业研究生小薛在村民会上说“现在大家都明确了咱村实施乡村振兴所存在的困难，齐心协力提出解决问题的具体办法和努力的方向来，这样才能一步步走向富强大伙说说、你觉得什么问题最重要，咱们先从哪里开始，好吧？”上述工作所运用的方法是（）

**选项：**
```
A.确定服务对象需要
B.向服务对象解释所设定的目标
C.确定服务对象问题
D.向服务对象解释所设定目标的目的
```

> **正确答案：** C

**解析：**
社会工作者可以通过征询服务对象对问题的认识和理解来确认问题，例如我们已经对影响问题的因素和问题本身进行了分析，这些问题包括....和…....以及你觉得这些问题是否就是我们下一步需要解决的？

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_76_0.png) 需要掌握有关儿童发展的理论知识，在幼儿身体快速生长和发育过程中，他们先学会抬头，后学会拍胸、坐立和行走，这一过程体现了儿童成长发展的（）特点

**选项：**
```
A.快速性
B.阶段性
C.顺序性
D.差异性
```

> **正确答案：** C

**解析：**
儿童身体的快速生长和发育的过程中，遵循从上到下、由近到远、从大到小、由低级向高级的顺序性。如婴幼儿先会抬头，后会抬胸再会坐、立、行，这是从上到下的顺序。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_76_0.png) ”儿童关爱服务体系试点项目建设过程中社区儿童主任小刘对辖区内的困境儿童群体摸排后发现，6岁的乐乐自出生就由祖父母照顾父母外出务工杳无音讯。因乐乐没有出生证明又缺乏相关材料无法获得户籍身份，影响其入学、就医和享受其他儿童福利。小刘了解情况后，走访了乐乐的出生医院和当地派出所，获得了其出生资料，协助其办理了户口。小刘的上述工作满足了乐乐（）

**选项：**
```
A.生存的需要
B.被监护的需要
C.受保护需要
D.社会化需要
```

> **正确答案：** A

**解析：**
生存的需要包括生命存在的需要和社会存在的需要两个方面（1）生命存在的需要，即获得基本生活照料，包括养育照料和可获得的最高水平健康医疗照料。（2）社会存在的需要，即获得社会身份，包括姓名、户籍和国籍。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_76_0.png) 机构在彩虹社区组织了夏令营活动，社会工作者小美策划的服务中有一项内容是联合父母，采用体验式方法，培养儿童人际交往、时间管理、理财和整理房间等技能、小美策划的上述服务是儿童社会工作中的（）

**选项：**
```
A.儿童问题辅导
B.儿童社会化引导
C.儿童支持服务
D.儿童娱乐和休闲
```

> **正确答案：** B

**解析：**
为处于不同年龄段的儿童提供与其社会化发展需要相符合的社会化引导服务，主要内容包括（1）自我认同；（2）技能学习。这里的技能学习是指儿童学业之外的生活技能学习，包含人际交往技能、时间管理技能、理财技能。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_76_0.png) 一起儿童保护案件召开审查建逮捕听证会，听取各方意见。社会工作者小蓝被邀请参加。该案件的主要事实是一名小学五年级的男同学因考试成绩不理想而被母亲体罚造成轻伤。听证会上，有人提出，案件中母亲对儿童造成了伤害。性质恶劣。应依法逮捕。小蓝则认为案件中母亲有明显悔改表现，且儿童处于身心发展的关键时期，家庭对其成长意义重大，应对其家庭提供支持服务，而不应简单将其逮捕。小蓝的观点体现了儿童社会工作实务的（）原则。

**选项：**
```
A.儿童优先
B.儿童利益最大
C.生态系统
D.儿童平等参与
```

> **正确答案：** C

**解析：**
社会工作者应重视家庭的作用，运用生态系统的观点，从儿童自身及其与家庭、朋辈群体、社区、学校、服务机构等的互动关系中分析儿童问题识别所需资源，提供专业服务，促进儿童发展。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_76_0.png) 机构为拓宽青少年社会参与渠道。组织青少年志愿者参与城市社区更新改造行动。包括参加社区居民意见征集、发挥所学专长参加学校邻近街区的改造方案设计等活动。上述服务属于青少年社会工作类型中的（）服务。

**选项：**
```
A.预防性
B.矫正性
C.发展性
D.治疗性
```

> **正确答案：** C

**解析：**
青少年社会工作大致可分为如下类一是针对青少年的生理、心理和社会发展需要，提供社会资源，协助正常发展而设计的发展性青少年社会工作服务：二是针对已经发生问题之青少年的个人、家庭社区环境的不良因素而提供的矫正性青少年社会工作服务三是针对青少年个人及其家庭、学校、社区现状而开展的预防性青少年社会工作服务。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_76_0.png) 多高中毕业生在填报高考志愿时，不知该如何选择合适的专业。学校社会工作者小董拟运用“生涯选择配合论”为他们开展生涯…在辅导过程中，小董首先应该帮助他们（）

**选项：**
```
A.了解专业高校的基本类别
B.了解自己的兴趣和追求
C.分析不同专业的就业前景
D.评估不同专业的预期收入
```

> **正确答案：** B

**解析：**
Wood的生涯选择配合论（1990）是生涯规划的重要理论基础⑩Wood认为生涯规划的先决条件是必须要先对自己有充分的认识与了解包括自己的能力、兴趣、人格和需求与价值观等，也就是要先能掌控自己的内在世界之后，才开始探索外在的工作世界、了解职业所需的能力、职业的分类与内容、职业所需的特质及各类职业的报酬率等。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_76_0.png) 友好型社区”创建活动，对全社区的道路、公共活动场所和公共卫生间等适老化改造，预防老人发生意外事故。上述措施满足了老年人的（）需要。

**选项：**
```
A.健康维护
B.居家安全
C.社会参与
D.就业休闲
```

> **正确答案：** B

**解析：**
答案B 居家安全正确。依据专业定义，社区公共设施改造符合居家安全的需要。
A 健康维护：涉及保持身体机能和预防疾病，但题目重点在预防意外，更偏向安全而非直接健康管理。
B 居家安全：通常包括居住环境的安全，但扩展至社区公共区域（如道路、卫生间）的适老化改造，属于广义的“居家安全”范畴。
C 社会参与：强调促进老年人参与活动，但题目措施主要针对安全，非直接促进参与。
D 就业休闲：明显不相关。
关键辨析：居家安全的定义在社工理论中常涵盖老年人日常活动的外部环境安全，包括社区公共设施。例如，防滑路面、无障碍卫生间等改造直接预防意外，属于安全需求。若严格限定“居家”为家庭内部，则可能出现争议，但题目中的改造属于老年人生活环境的整体安全，仍属居家安全范畴。
