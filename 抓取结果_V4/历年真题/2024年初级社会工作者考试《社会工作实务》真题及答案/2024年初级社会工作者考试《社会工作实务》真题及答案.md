# 2024年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_77_0.png) 在社区入户时发现，大海近期因交通意外造成重度残疾，家里只有上初中的儿子照顾他，大海希望得到支持，老张拟为其提供服务。从服务对象的来源看，大海一家属于( )。

**选项：**
```
A.主动求助的服务对象
B.机构转介的服务对象
C.外展发现的服务对象
D.他人介绍的服务对象
```

> **正确答案：** C

**解析：**
服务对象的来源通常有 3 种情况:(1)主动求助者。通常是一个人、一个家庭或团体带着超出他们能力之外、不能解决的问题主动前来寻求帮助。(2)由他人介绍或机构转介来的。(3)由社会工作者通过外展工作而成为服务对象的。这是指那些既不是自己主动求助，也不是由他人或机构转介而来的服务对象，而是由社会工作者主动接触并使他们接受服务的人。

题中社会工作者老张是在社区入户时发现的服务对象，而非服务对象主动前往机构寻求帮助，所以属于外展方式成为的服务对象。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_77_0.png) 在社区开办了“悦读悦成长”亲子绘本阅读小组，带领 10 个家庭共同阅读，促进亲子及家庭间的交流。在结案阶段，小李的恰当做法是( )。

**选项：**
```
A.协助家庭回顾小组过程中的改变
B.引导家长继续监督孩子阅读
C.要求孩子在小组结束后继续阅读
D.跟进小组结束后的阅读情况
```

> **正确答案：** A

**解析：**
结案阶段的任务：巩固已有改变。确保服务对象在社会工作助人过程中获得的经验能够巩固下来并能够用于日常生活中，是社会工作者的责任。社会工作者要尽力帮助服务对象保持在助人过程中取得的进步，巩固和增强他们的自我功能。下面一些方法能够帮助社会工作者达到这样的目的。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_77_0.png) 儿最近学习成绩下滑且总是闷闷不乐。王女士询问女儿学习遇到的困难，女儿很不耐烦，不想理她。王女士遂向社会工作者小刘求助。小刘在接案过程中向王女士仔细询问女儿的状况。下列提问中,属于治疗性沟通的是( )。

**选项：**
```
A.“您是什么时候发现女儿的变化的?”
B.“您是怎么知道女儿学习成绩下滑的?”
C.“您是否想过用其他方法了解女儿的情况?”
D.“您能否尝试让女儿感受到情绪上的支持?”
```

> **正确答案：** D

**解析：**
治疗性沟通需兼具认知引导与行动促进。通过推动王女士提供情绪支持，直接回应女儿的需求，选项D是正确答案。

详细分析各选项：
选项A（“您是什么时候发现女儿的变化的?”）属于非治疗性沟通。
· 定位：收集基本信息，了解问题发生的时间节点。
· 作用：属于接案阶段的事实性询问，帮助社工掌握情况，但未推动王女士改变现有互动模式。
选项B（“您是怎么知道女儿学习成绩下滑的?”）属于非治疗性沟通。
· 定位：探询问题发现的途径（如老师反馈、成绩单等）。
· 作用：仍为信息收集，未涉及亲子互动改善的引导。
选项C（“您是否想过用其他方法了解女儿的情况?”）部分符合治疗性沟通，但干预力度较弱。虽涉及反思，但未明确行动路径，干预效果有限。
· 定位：引导王女士反思当前沟通方式的局限性。
· 作用：属于认知调整，帮助王女士意识到现有方法的不足，但未明确如何行动。
· 局限性：停留在“是否想过”，未直接推动有效行动。
选项D（“您能否尝试让女儿感受到情绪上的支持?”）是最契合治疗性沟通的行动导向，帮助服务对象采取具体措施解决问题。
· 定位：直接建议王女士调整与女儿的互动策略。
· 作用：促进行动-鼓励王女士通过情感支持（如倾听、共情）改善母女关系，缓解女儿的抵触情绪。解决核心问题-女儿“闷闷不乐”可能源于情感需求未被满足，此提问直指问题根源。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_77_0.png) 种办法督促小强的日常学习，导致小强出现逆反心理，不仅成绩没有提高，而且父子关系紧张。为了协助他们改善父子关系，社会工作者小马为其开展家庭治疗。小马制订的下列具体目标中，适宜的是()

**选项：**
```
A.缓解小强与父亲之间的冲突
B.增进父亲与小强之间的相互理解
C.降低父亲对小强的学习要求
D.指导小强提高学习成绩
```

> **正确答案：** B

**解析：**
介入目标应采用积极正向的语言，重在促进服务对象的成长。家庭治疗以家庭为介入单位，探索青少年问题背后的家庭结构和互动关系，促进家庭内在系统的改变，优化青少年成长的家庭环境，主要适用于改善并重建青少年和家庭组员之间的关系，实现家庭组员之间的良性互动。家庭治疗的焦点主要集中在家庭本身的结构和互动关系上。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_77_0.png) 在某社区开办了老年人慢病管理学习小组，拟采取基线测量评估服务成效。小李的下列做法中，属于建立基线的是( )。

**选项：**
```
A.选择测量慢病管理行为的指标
B.改变老年人的慢病管理行为
C.比较老年人慢病管理行为变化
D.测量老年人对小组的满意度
```

> **正确答案：** A

**解析：**
建立基线的方法有 3 种。一是确定介入的目标，例如服务对象的行为、思想、感觉、社会关系或社会环境的变化及指标。二是选择测量工具，包括直接观察或使用标准化问卷或量表。三是对目标行为进行测量并记录目标行为(或者思想、感觉、社会关系或社会环境的情况)这个过程建立的是基线数据，此过程也称为基线期。A 选项选择测量慢病管理行为的指标。


## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_77_0.png) 对独居残疾老人张奶奶进行评估，发现她有日常生活照料、居家康复、精神慰藉等多重需要，拟采用个案管理方法提供服务。小王正确的做法是( )。

**选项：**
```
A.自己为张奶奶提供所有服务
B.为张奶奶提供整合性服务
C.将张奶奶转介给养老机构
D.鼓励张奶奶做力所能及的事
```

> **正确答案：** B

**解析：**
个案管理的关键在于评估需求后协调多方资源（如联系居家护理、康复师、志愿者提供精神关怀），确保服务无缝衔接，这正是选项B的内涵，正确答案为选项B。其他选项或偏离方法核心（A、C），或侧重具体技巧（D），均不符合题意。


详细分析如下：
个案管理的核心是整合资源：张奶奶存在日常生活照料、居家康复、精神慰藉等多重需求，单一服务无法满足其复杂需求。个案管理强调协调医疗、康复、社区服务等不同资源，形成系统支持网络，而非由社会工作者直接包揽所有服务（排除A）。
避免简单转介（排除C）：转介养老机构可能割裂现有社会支持，且题干未提及张奶奶有机构安置需求。个案管理需优先在案主熟悉环境中整合服务，保障服务的连续性和个性化。
“鼓励做力所能及的事”（D）属于增强自主性，属于具体干预手段，但并非个案管理方法的核心特征。题干明确问“采用个案管理方法”，应聚焦资源整合而非单一技巧。（排除D）

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_77_0.png) 天定时在社区广场组织活动。近期，社区居委会接到多起相关投诉，抱怨舞蹈队活动声音过大，影响家人休息。为此，社会工作者小董协调辖区某单位将其多功能厅在晚间向社区居民开放，并为舞蹈队安排训练空间。小董还组织社区居民骨干轮流值班，协助该单位管理多功能厅，投诉问题得到了顺利解决，居民们都很满意。小董的上述做法属于( )。

**选项：**
```
A.资源置换
B.资源整合
C.资源共享
D.资源分配
```

> **正确答案：** B

**解析：**
（1）资源整合：将分散的资源（如场地、人力、组织）通过协调与合作，形成功能互补的系统，共同达成目标。


（2）关键特征：不同主体（如社区、单位、居民）分工协作；资源功能互补（如单位提供场地 + 居民提供管理人力）；建立互惠关系（如单位获得社区支持，居民解决矛盾）。
（3）小董的做法是典型的 资源整合（B），因其：将单位场地、居民人力、社工协调能力等分散资源组合为协作系统；通过功能互补（场地+管理）实现共同目标（解决投诉）；符合题干解析中“整合既有资源，形成功能互补与互依”的核心定义。
（4）其他选项排除：
A 资源置换：双方交换资源（如舞蹈队帮单位宣传换取场地），但本题无资源交换行为。
C 资源共享：仅强调资源共同使用（如单位单纯开放场地），但本题涉及 资源功能互补（场地+人力）和 组织协作（居民参与管理），远超单纯共享。
D 资源分配：自上而下分配资源（如政府指定场地给舞蹈队），但本题通过 多方协调合作 实现，非单向分配。

## 第 11 题 [单选题]

**题目：** 11. ![图](./images/q_11_77_0.png) 在与社区矫正对象小夏的例行会谈中得知，小夏过去的“兄弟"来找他、拉他参与对一个团伙的报复行动。尽管小夏不想再次违法，但他迫于压力不得已答应了“兄弟”的要求，在为小夏提供服务时，大江恰当的做法是( )。

**选项：**
```
A.要求小夏反思自己的错误行为
B.妥善处理保密与社会安全的关系
C.鼓励小夏自行决定是否参与行动
D.建议小夏改变犹豫不决的个性
```

> **正确答案：** B

**解析：**
社会工作者应当保护服务对象的隐私，未经服务对象同意或允许，社会工作者不得向第三方透露涉及服务对象个人身份资料和其他可能危害服务对象权益的隐私信息在特别情况下必须透露有关信息时，社会工作者应向机构或有关部门报告，并告知服务对象有限度公开隐私信息的必要性及采取相关保护措施 如果在紧急情形下，必须打破保密原则而来不及提出报告时，社会工作者事后应当提供相关的证据并补办手续，以记录必要的工作程序。基于本题背景，应妥善处理保密与社会安全的关系。
