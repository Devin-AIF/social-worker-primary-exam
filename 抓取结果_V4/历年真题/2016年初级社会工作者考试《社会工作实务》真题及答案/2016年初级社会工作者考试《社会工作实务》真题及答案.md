# 2016年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 场所的女性开展“生殖健康服务月”活动，为她们进行了健康体检，发现很多人患有妇科病，还有一些人患有性病和艾滋病。针对这种情况，该机构的社会工作者可以采取的服务措施是（    ）。

**选项：**
```
A.公布艾滋病和性病人员名单，切断感染源
B.建议公安部门开展对娱乐场所的专项整治活动
C.联系医务人员为她们做健康教育和咨询
D.与患有性病、艾滋病的人个别交流，提醒其自重
```

> **正确答案：** C

**解析：**
针对妇女生殖健康的工作干预策略包括：①健全具有社会性别敏感性的妇女生殖健康的政策；②建立“以社区为基础，以妇女为中心”的服务策略；③建立妇女定期生殖健康检查的制度。C项，联系医务人员为她们做健康教育和咨询有助于建立与完善妇女定期生殖健康检查的制度。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) ，一些青少年因为缺乏与父母的有效沟通，亲子关系紧张而不想回家，以逃避父母的责问。为此，社会工作者计划举办“亲子并行小组”以改善青少年与父母间的关系。下列做法中，有助于实现上述目标的是（    ）。

**选项：**
```
A.组织青少年和家长分别参加为期三天的夏令营活动
B.为青少年和家长分别举办沟通技巧学习小组
C.运用分组和跨组交流的方式，协助青少年和家长学习沟通技巧
D.组织家庭教育专家，为青少年和家长举办亲子沟通讲座
```

> **正确答案：** C

**解析：**
开展亲职教育教辅服务时，应注意积极沟通，即训练父母学习如何与子女做积极沟通。为调和青少年家庭的亲子关系，亲子并行小组是社会工作者经常运用的服务方法，这种方法是将父母和子女分成两个平行小组，并运用分组和跨组交流的方式达到知己知彼的目的，以此来改善青少年与父母之间的关系。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_80_0.png) 动员和组织社区中待业在家的低保人员参与社区志愿服务，看护“四点半学堂”的孩子，上门探望独居老人。小赵上述工作的目的是（    ）。

**选项：**
```
A.为低保人员提供心理支持
B.调节低保人员的家庭关系
C.为低保人员提供就业援助
D.促进低保人员的社会融入
```

> **正确答案：** D

**解析：**
在社区层面，低保家庭对于社区的依赖程度很高，生活范围主要局限在社区内部，人际圈子同质性高，家庭支持不够。社会工作者要为低保家庭创造参与社区活动的机会，鼓励他们参与社区的公益和文娱活动，帮助他们建立和其他社区居民的联系，互帮互助，承担一定的责任和义务，增加他们的成员感、归属感和自信心。社会工作者要积极帮助困难群体修复社会关系，建立社会支持网络，尤其是邻里支持。题中，社会工作者小赵动员和组织社区中待业在家的低保人员参与社区志愿服务、看护“四点半学堂”的孩子、上门探望独居老人的措施增加了低保家庭的社会支持，为低保家庭创造了参与社区活动的机会，目的是促进低保家庭的社会融入。

**题目：** 4. ![图](./images/q_4_80_0.png) 在急诊室可提供的服务是（    ）。

**选项：**
```
A.鼓励患者使用自我管理的方法

C.配合医疗需要，帮助患者及家属获取资源
D.介入医疗过程，帮助医务人员舒缓压力
```

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_80_0.png) 经朋友介绍到一家公司给老板当司机，经过一段时间后，大李因工作强度大，不知不觉中又开始吸毒。在与社会工作者小林的一次会面中，大李告诉小林自己复吸的事，并表示如果老板和家人知道，不仅会丢掉工作，也会影响与家人的关系，请求林为他保密。对此小林合适的做法是（    ）。

**选项：**
```
A.明确告知大李毒驾的严重后果，鼓励他积极戒毒
B.遵守保密原则，协助大李保住这份工作，以维持与家人的关系
C.遵循服务对象自决的原则，由大李自行决定是否辞职
D.联系大李的家人，要求其加强对大李毒驾行为的监管
```

> **正确答案：** A

**解析：**
为戒毒人员提供戒毒治疗及康复服务的内容包括：①掌握戒毒人员的个人背景资料，建立戒毒人员档案；②定期不定期对戒毒人员进行个案辅导或家访，帮助其排解戒毒和康复过程中因生理和心理反应而产生的不良情绪，增强其抵制毒品诱惑的能力，协助其建立良好的人际关系，提高其社会适应的能力，从而降低复吸率；③与相关单位和部门合作，为戒毒人员提供生活上的关心和支持，帮助其解决生活、工作、学习、社会交往中的困难和问题，为其创造回归社会的良好环境和条件；④对戒毒人员进行职业技能培训，职业指导和就业援助，并鼓励其积极参加各种社会公益活动，成为自食其力和对社会有用的人，帮助其树立生活信心并融入社会。题中，大李因工作强度大，又开始复吸，在这种情况下，社会工作者应该鼓励大李积极戒毒，增强其抵制毒品诱惑的能力。

**题目：** 7. ![图](./images/q_7_80_0.png) 者为员工策划了幼儿照顾、家庭和婚姻辅导及闲暇娱乐等服务，上述服务可实现的目标是（    ）。

**选项：**
```
A.改善劳动关系
B.平衡职工的生活与工作
C.保障职工安全
D.帮扶职工困难群体
```

> **正确答案：** B

**解析：**
企业社会工作力求为职工创造条件和营造环境，使职工工作与生活更有幸福感。企业社会工作关注职工的婚姻家庭、职工在工作以外的生活、女职工的特殊权益、职工的家庭成员、职工生活中的不便和困难等。为了让职工生活得更好，社会工作者协助企业制定政策，提供设施，采取具体措施来为职工着想，解决职工的实际困难，解除后顾之忧，既节省职工的时间和精力、体力，又使职工能够充分享受生活，更使职工对企业有归属感，为职工献身企业发展，提高工作效率创造条件。题中，该企业社会工作者为员工策划的幼儿照顾、家庭和婚姻辅导及闲暇娱乐等服务可以起到平衡职工工作和生活的作用。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_80_0.png) 戒毒后偶有复吸。一次，小芳在与社会工作者小林的会谈中提到，为防止被抓，有时会带着儿子去买毒品以作掩护。针对此情况，遵循儿童社会工作原则，小林应该（    ）。

**选项：**
```
A.申请剥夺小芳对儿子的监护权
B.提议小芳将儿子长期寄养在亲戚家
C.评估小芳行为对儿子的潜在风险，尽量避免母子分离
D.关注小芳积极改变的一面，相信她不会伤害到自己的孩子
```

> **正确答案：** C

**解析：**
以家庭为中心的实务原则主要包括：①在服务过程中，包括制订和落实服务计划，儿童的安全和健康永远是第一需要考虑的。②儿童社会工作者必须通过相关服务了解儿童需要和儿童及其家庭情况。③在服务过程中，儿童和家庭不应因为语言、宗教、发育迟缓等原因受到不同对待。④凡有可能，家庭应获得服务以保持儿童在家庭中生活，避免发生亲子隔离创伤。⑤家庭寄养照料是临时安置方式，应尽量短暂。⑥凡有可能，家庭外照料儿童都应在最短的时间里与父母团聚，并在过程中获得社会工作者的支持。⑦如没有回归原生家庭的可能，儿童社会工作者应该为家庭外照料安置儿童提供永久性替代照顾服务，安排领养寄养。⑧儿童社会工作服务必须为家庭外照料安置儿童提供下列保障：a.能够维持儿童与其关系密切人士的联系，如父母、兄弟姐妹和有其他具有养育能力的亲戚，使其感受到稳定和持续的家庭氛围；b.安置环境对儿童应无不利于其成长的约束，能够满足儿童个性化需要；c.从临时安置到永久安置的决策应在符合法律规定的前提下，以有利于儿童长远发展为出发点。因此，小林的做法是评估小芳行为对儿子的潜在风险，尽量避免母子分离。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_80_0.png) 学后在操场遇到英语老师，她对老师微笑，老师却没有理会她。小丽觉得老师一定是不喜欢自己，找学校社会工作者小秦诉说自己的苦恼。小秦引导小丽辨识出“老师不理会即等于不喜欢我”的这种非理性想法，从提升自我效能感，增加自信的角度，让小丽认识到上述想法属于（    ）。

**选项：**
```
A.非黑即白
B.灰色眼镜
C.以偏概全
D.透视心意
```

> **正确答案：** D

**解析：**
自我效能感影响一个人的感受、思想、行为和做事的毅力。四种常见的扭曲思想包括：①非黑即白，用是或不是这样“绝对”的思想影响人；②灰色眼镜，使人看一件事情时只着眼于负面或令人感到沮丧的地方，而忽略其他好的方面；③以偏概全，使人把个别事件或贬低自我的想法无限放大；④透视心意，使人单凭直觉猜测别人的想法和用心，还信以为真。
