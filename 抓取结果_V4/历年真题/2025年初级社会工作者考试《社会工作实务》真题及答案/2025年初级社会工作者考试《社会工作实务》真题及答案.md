# 2025年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_80_0.png) 托，某社区工作服务机构组织社区退休党员为儿童教授书法，既丰富了党员的课余生活也让儿童有了优质的书法课，把居民所需和党员所长紧密结合，这种资源链接方式是（    ）。

**选项：**
```
A.资源整合
B.资源共享
C.资源开发
D.资源置换
```

> **正确答案：** A

**解析：**
资源整合强调将不同来源、不同层次、不同结构、不同内容的资源通过整理、组合等方式，实现有机融合来更好地服务。题中把社区退休党员（党员所长）的书法教学能力和儿童（居民所需）对书法课的需求结合在一起，属于资源整合。资源共享是让各方共同使用已有的资源。此情境重点不在共享现有资源。资源开发侧重于挖掘新的资源。这里并非开发新资源。资源置换是交换资源。题中不是资源相互交换的情况。答案：A

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_80_0.png) 现有些老年人对自己养老安排不清晰，对身边养老资源缺乏了解，为此老吴拟开设夕阳红老年人知识小组，为组员提供养老信息交流机会，上述工作策略是（    ）。

**选项：**
```
A.建立互助网络策略
B.激发老年人参与社会工作的热情
C.宣传在社区养老的优势
D.动员社区老年志愿者
```

> **正确答案：** A

**解析：**
老吴开设小组为老年人提供养老信息交流机会，重点在于搭建让老年人互相交流养老信息的平台，这符合建立互助网络策略，即通过小组成员间信息分享互助解决对养老安排及资源不了解的问题。答案：A

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_80_0.png) 社区工作者对所在社区的认识、认同与归属，社区书记组织他们进行社区漫步，活动结束后，他们把活动中了解到的社区地理位置、居民构成、、商业场所分布、社区工作人员情况等信息进行了总结与展示，新入职的社区工作者展示的这些内容属于（    ）。

**选项：**
```
A.社区类型分析
B.社区基本情况分析
C.社区问题分析
D.社区基本需求分析
```

> **正确答案：** B

**解析：**
地理位置、居民构成、商业场所分布、社区工作人员情况等都属于对社区最基本情况的描述。这不属于对社区类型的划分界定（A项不符），没提及社区存在的问题（C项不符），也不是对社区需求如设施需求等内容（D项不符）。答案：B

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_80_0.png) 阳节活动时，设计了多个方案，在计算每个方案的成本并评估预期成效后，下一步应该做的是（    ）。

**选项：**
```
A.再次确认问题
B.融合修订成新方案
C.确定评估指标
D.比较分析备选方案
```

> **正确答案：** D

**解析：**
已经计算了各方案成本并评估预期成效，接下来应比较分析这些备选方案，才能选出最优方案，而不是再次确认问题、直接融合修订新方案或重新确定评估指标。答案：D

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_80_0.png) 有点矛盾就挥拳，他觉得这样做不对，但也控制不住自己。为帮助小军改善这种状况，社会工作者与他一起制定了约束自己行为的目标。根据目标设定的原则，下列目标陈述中正确的是（    ）。

**选项：**
```
A.减少小军打人的次数
B.增强小军改变不良行为的信心
C.消除小军不良行为
D.增加小军与同学正向交流的次数
```

> **正确答案：** D

**解析：**
目标设定要具体、可测量、可实现、相关及有时限（SMART原则）。A选项“减少小军打人的次数”，相对不够具体和可测量标准不清晰；B选项“增强小军改变不良行为的信心”，信心难以直接量化测量；C选项“消除小军不良行为”过于绝对，较难实现；D选项“增加小军与同学正向交流的次数”明确具体，可测量，且与改善小军因矛盾就挥拳的状况相关，符合目标设定原则。答案：D

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_80_0.png) 发现，有些脑卒中患者难以接受突然偏瘫的事实，情绪低落，不愿说话，家属也不知道如何安抚，针对这一情况，小钟拟开设脑卒中病友家属沟通技巧训练小组。下列小钟的做法中，恰当的是（    ）。

**选项：**
```
A.模拟练习，以便统一组员的沟通方式
B.邀请组员分享表达内心感受和经验
C.邀请医生讲授脑卒中日常护理知识
D.组织组员观看脑卒中知识的宣传片
```

> **正确答案：** B

**解析：**
小组主题是“脑卒中病友家属沟通技巧训练”，重点应围绕家属间沟通技巧训练。A统一沟通方式过于刻板，不利于灵活沟通和个性化。C讲授日常护理知识偏离沟通技巧主题。D观看宣传片也是关于脑卒中知识而非沟通技巧。B邀请组员分享表达内心感受和经验，既有助于家属宣泄情绪，又能在分享中互相学习沟通技巧，较为恰当。答案：B

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_80_0.png) 在入户探访时发现，高大爷自老伴过世后，便不爱出门，闷闷不乐，总说“活着没意思"，小王担心高大爷有自杀倾向，便对其进行了自杀风险评估。下列自杀风险的线索中，属于间接线索的是（    ）

**选项：**
```
A.高大爷近期将名下的房产过户给独生女
B.高大爷最近突然频繁外出会见亲朋好友
C.高大爷觉得自己活着就是给家人添麻烦
D.高大爷跟独生女交代他的身后事要从简
```

> **正确答案：** C

**解析：**
这是一道关于识别自杀风险线索类型的判断题。我们需要根据题干描述和选项内容，确定哪一项属于间接线索。首先，明确间接线索的定义：间接线索通常指那些不直接表达自杀意愿，但通过言行举止反映出对生活的消极态度或对家人的担忧，可能暗示自杀倾向的表达。现在，分析各个选项：A项（高大爷近期将名下的房产过户给独生女）：这更像是一种实际行为，可能属于行为线索，因为它涉及到具体财产的处置，而非仅仅表达情感或态度。B项（高大爷最近突然频繁外出会见亲朋好友）：这描述了一种行为改变，但并不直接指向自杀倾向，可能与其他多种原因相关。C项（高大爷觉得自己活着就是给家人添麻烦）：这反映了高大爷对生活的消极态度和对家人的担忧，符合间接线索的定义，即不直接提及自杀，但表达出对生活的无意义感或对家人的负担感。D项（高大爷跟独生女交代他的身后事要从简）：这更接近直接线索或行为线索，因为它涉及到具体的身后事安排。综上所述，C项最符合间接线索的特征，即通过表达对生活的消极看法和对家人的担忧来间接反映自杀倾向。所以，正确答案是C项：“高大爷觉得自己活着就是给家人添麻烦”。
