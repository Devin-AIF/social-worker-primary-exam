# 2022年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_77_0.png) 在学校个案工作室接待了由老师转介过来的小明，与其进行初步的接案面谈。老李的下列工作中属于收集环境资料的是（）。

**选项：**
```
A.了解小明的行为方式
B.了解小明的成长背景
C.测试小明的认知能力
D.了解小明的兴趣爱好
```

> **正确答案：** B

**解析：**
服务对象所处的社会环境。包括服务对象的人际关系状况，与朋友和同事的关系：服务对象成长的背景（B正确），学习、工作和生活的环境，家庭的经济状况：家人之间的关系形态，父母的影响以及邻里关系等。AC属于服务对象的特点与能力，D属于个人资料。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_77_0.png) 从农村来到城市照顾孙子，由于人生地不熟，语言沟通困难，特别不适应。想回农村。但是她觉得儿子儿媳整天忙于工作，孙子无人照顾，又不忍向儿子开口。为了帮助赵奶奶识别问题的主观因素，社会工作者的下列做法中正确的是（）。

**选项：**
```
A.询问赵奶奶为解决这一问题做了什么努力
B.了解赵奶奶之前是如何应对类似问题的
C.与赵奶奶确认这一问题是什么时候开始的
D.引导赵奶奶说出她自己担心背后的原因
```

> **正确答案：** D

**解析：**
所谓主观因素是指服务对象对问题的实际感受。识别问题的主观因素即是要认识服务对象是如何看待自己的问题的，站在他的角度来理解这些问题对他的意义是什么、他为什么会有如此的主观理解、它们对服务对象现在的社会一心理影响是什么，即认识问题在服务对象处境中的意义。ABC属于识别服务对象问题的客观因素。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_77_0.png) 满父母对其上网和打游戏的监督，常反锁房门不理父母，亲子关系紧张，社会工作者小李发现社区里还有其他家庭也出现类似问题。针对这种情况，小李建立了家长和学生两个微信群。教授家长与孩子的相处之道及学生自我管理方法。小李的上述工作是社会工作介入类型中的（）。

**选项：**
```
A.直接介入
B.间接介入
C.综合介入
D.分层介入
```

> **正确答案：** C

**解析：**
直接介入是指以个人、家庭和小群体为关注对象，针对个人、家庭和小群体采取的直接行动。间接介入是指以个人、家庭、小组、组织和社区甚至更大的社会系统为关注对象，由社会工作者代表服务对象采取行动，通过介入服务对象以外的其他系统间接帮助他们。从人与环境两个环节进行介入的策略构成了将直接实务活动和间接实务活动结合在一起的综合介入行动，是对各种与服务对象系统有关的系统进行的介入，体现的是一种综合治理的理念。题干中社工小李建立了家长和学生两个微信群，既教授家长与孩子的相处之道，也教授学生自我管理方法。采取的是综合介入的方法，故选C。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_77_0.png) 在居民议事会上汇报了近年来老年人上下楼跌倒事件多发，一些独居老人因下楼不便而多日不出门等情况，以及部分居民对安装电梯的建议。老赵上述做法属于预估阶段的（）。

**选项：**
```
A.描述社区居民具有的资源
B.描述社区居民的生活环境
C.描述居民的问题与需要
D.深究问题不能解决的原因
```

> **正确答案：** C

**解析：**
描述服务对象的问题与需要。包括问题是什么、问题的范围、严重程度及持续的时间。根据题干，社工老赵主要是汇报了社区目前存在的问题以及居民需要，故选C。题干并没有涉及到资源和问题不能解决的原因。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_77_0.png) 心的社会工作者小刘计划请老年科医生到所在社区为老年人检查身体，举办慢性病管理和健康饮食讲座，以帮助老年人提高健康管理意识，提升老年群体健康水平。小刘为社区老年人制定的上述健康服务方案，主要描述了计划中的（）。

**选项：**
```
A.服务的对象
B.关注的问题
C.评估的方法
D.介入的行动
```

> **正确答案：** D

**解析：**
关注的问题是指介入工作要加以处理、改善的服务对象的社会功能的问题。故B错误。关注的对象是指介入行动要改变的人和系统，包括家庭、群体、组织和社区。故A错误。介入可以采用个人辅导、小组活动、社区介入、网络建构和政策倡导等多种方法，而介入行动则可以是危机干预、物质支持、心理辅导等多种行动。根据题干“请老年科医生到所在社区为老年人检查身体，举办慢性病管理和健康饮食讲座”可知为介入行动，故选D。题干中并未涉及评估方法，故C错误。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_77_0.png) 的父亲两年前病逝，母亲无法承受打击，一直情绪低落，无心照顾丁丁。丁丁性格内向，学习能力较弱，还经常不去上学，社会工作者接案后，对丁丁进行需求评估。下列属于丁丁发展性需要的是（    ）。

**选项：**
```
A.得到经济方面的帮助
B.获得基本的生活照料
C.与同伴相互交流沟通
D.获得良好的家庭生活
```

> **正确答案：** D

**解析：**
正确答案为D 获得良好的家庭生活。丁丁的案例中，家庭功能缺失是发展性需求的核心矛盾，需优先干预。
发展性需要的定义：根据社会工作理论，发展性需要指儿童在成长过程中所需的支持性环境与资源，以促进其潜能开发、社会适应和人格健全。核心包括：获得良好的家庭生活（父母关爱、适当管教、亲子关系）；受教育机会与学习环境；休闲娱乐与安全社交空间。
题干中丁丁的需求分析：①家庭环境：丁丁因父亲去世、母亲情绪低落导致家庭功能缺失，缺乏父母的关爱与稳定照料，直接影响其心理安全感和成长支持系统。②社会化需求（选项C）：与同伴交流属于儿童社会化需求，但题干中丁丁的核心矛盾是家庭功能崩溃导致的发展受阻（如学业困难、性格内向）。③经济与基本照料（选项A、B）：属于生存需要，保障基本生活条件，但未触及发展层面的深层问题。
选项D的合理性：家庭功能修复是丁丁恢复发展能力的前提。通过重建母亲的心理状态、改善亲子互动，才能为丁丁提供情感支持和成长环境，进而缓解其学业与社交问题。研究显示，丧亲儿童若长期缺乏家庭支持，更容易出现学业低成就、情绪障碍等后果，这与题干中丁丁的表现高度吻合。
【提示】选项C（与同伴交流）的误区：①社会化需求属于次级需求：同伴互动虽重要，但需以家庭功能为基础。丁丁因家庭问题导致自我封闭，单纯强调同伴交流无法解决根本矛盾。②发展性需要的优先级：家庭环境是儿童发展的核心场域，优先于社会化需求的满足。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_77_0.png) 发现社区内有一些准妈妈被即将面临的新生儿养育问题所困扰，为帮助她们学习育儿知识和技巧，小王组织了“新手妈妈训练营”，根据儿童社会工作服务的分类，上述服务属于（）。

**选项：**
```
A.支持性儿童福利服务
B.补充性儿童福利服务
C.替代性儿童福利服务
D.儿童保护服务
```

> **正确答案：** A

**解析：**
正确答案为 A 支持性儿童福利服务，因其完全符合题干中“通过提升父母能力满足儿童需求”的核心逻辑，且与其他三类服务的定义和适用场景无交集。
支持性儿童福利服务的对象是全体儿童及其家庭，尤其是家庭监护状况良好的家庭，其核心目标是支持父母履行教养职责，通过弥补父母亲职能力的不足，满足儿童成长需求。例如，为儿童及其家庭提供育儿咨询、协助办理新生儿户籍登记等。题目中“准妈妈”群体属于广义的儿童家庭支持范畴，符合支持性服务的对象定位。
教材明确将“为准妈妈提供咨询”列为支持性服务的典型内容，而补充性服务的案例通常是“四点半课堂”等临时托管服务。题干中“训练营”旨在提升准妈妈的育儿能力，属于支持父母履行亲职职责的直接干预，而非补充家庭监护功能的不足。
补充性儿童福利服务（B）：核心是弥补家庭监护功能的缺失，而训练营的服务属于优化家庭功能，属于支持性范畴，故排除。




替代性服务（C）：适用于亲职不当或缺失的情况（如家庭寄养、收养），需完全或部分替代父母职责。题干未提及儿童需要脱离原生家庭，故排除。
儿童保护服务（D）：针对遭受虐待、忽视等伤害的儿童，提供庇护或干预。题干中无此类风险描述，排除。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_77_0.png) 小张对本村留守儿童摸排后发现，留守儿童课后娱乐方式单一，沉迷手机游戏的情况很普遍，为改善这一现状，小张在本村儿童之家举办“凤凰花助飞成长营”，为留守儿童提供适宜的文学读物，定期开设科学探索课堂，激发他们的求知欲和探索精神，小张上述服务满足了留守儿童的（）需要。

**选项：**
```
A.生存
B.受保护
C.发展
D.社会化
```

> **正确答案：** C

**解析：**
发展的需要：儿童的发展需要也被称为儿童的成长需要，是指儿童为了身心发展，需要获得的关爱、教育和引导。它主要包括：（1）获得良好的家庭生活，得到父母的爱和适当管教，与父母建立良好的亲子关系。（2）拥有受教育的机会，有良好的教育和学习环境，满足其探索和认知世界的求知欲。（3）获得足够的休闲和娱乐，有适合儿童且安全的娱乐场所，为儿童形成良好的娱乐休闲态度、方式和行为提供教导和培养。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_77_0.png) 务的相关程序，社会工作者小陆为失依儿童乐乐提供收养服务，协助收养家庭在当地民政部门完成了登记，乐乐正式进入收养家庭生活，下一步小陆应提供的服务内容是（）。

**选项：**
```
A.评估结案
B.跟踪回访
C.家庭培训
D.信息发布
```

> **正确答案：** B

**解析：**
儿童收养服务包括如下一系列的服务环节和程序。1.送养儿童信息发布；2.收养人收养家庭招募；3.收养人收养家庭评估；4.收养人/收养家庭培训川5.送养儿童与收养人/收养家庭适配；6.办理收养手续；7.送养儿童进入收养家庭；8.收养后跟踪回访；9.评估结案根据题干：社工已经协助收养家庭在当地民政部门完成了登记，乐乐正式进入收养家庭生活，即已经完成了送养儿童进入收养家庭，下一步工作就是收养后跟踪回访，故选B。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_77_0.png) 轻父母与孩子存在亲子沟通问题。父母觉得孩子们的要求太多，太折腾，孩子们觉得父母不理解他们。针对这一状况，某社会工作服务机构利用双休日开设了亲子平行小组，帮助家庭增强亲子间的有效沟通和互动，上述）儿童社会工作服务内容属于家庭支持服务中的（）。

**选项：**
```
A.家庭辅导
B.婚姻辅导
C.儿童辅导
D.亲职辅导
```

> **正确答案：** A

**解析：**
正确答案为 A家庭辅导，因其直接针对亲子沟通问题，通过家庭成员共同参与改善家庭关系，符合家庭辅导的定义。
家庭辅导（A）：以家庭整体为服务对象，通过改善家庭成员间的关系（如沟通模式、互动方式），恢复健康的家庭功能。亲子沟通问题涉及父母与孩子的双向互动，服务通过“亲子平行小组”让全体家庭成员参与，直接改善亲子关系，属于家庭层面的干预。家庭辅导强调 家庭系统的调整，而非单方面提升父母的育儿技能。选项A恰当。


亲职辅导（D）：以儿童需求为中心，单向指导父母如何履行亲职角色（如育儿技巧、情绪管理），通常以家长为直接服务对象。亲职辅导仅适用于父母单方向能力缺失的场景（如家长不懂儿童发展规律），与本题双向互动调整的需求不符，因此选项D不恰当。

## 第 11 题 [单选题]

**题目：** 11. ![图](./images/q_11_77_0.png) 从小由奶奶照顾，母亲因工作单位较远且经常加班，与小军接触时间较少；父亲对小军十分严厉，经常责骂他。三个月前奶奶因为意外骨折卧病在床，无法照顾小军，小军出现逃学、打架等情况。针对上述情况，下列关于青少年社会工作的做法属于中观层面的是（）。

**选项：**
```
A.指出小军的逃学问题
B.制止小军父亲的不当行为
C.为小军提供学业辅导
D.了解小军一家的互动状态
```

> **正确答案：** D

**解析：**
青少年社会工作一般可分为微观、中观、宏观三个层面。微观层面一般指个人层面的社会工作，即针对青少年个体需求而开展的个案辅导服务：中观层面指家庭和小组层面的社会工作，即针对青少年家庭以及有相同需要或背景的青少年群体，而开展的社会工作服务：宏观层面指组织和社区社会工作，或者是针对法规或政策而进行的倡导或改变性工作，如建立青少年组织，开展社区工作，进行青少年法规、政策或制度的修改及倡导工作等。ABC为微观层面。D是从家庭层面进行介入，属于中观层面。

## 第 12 题 [单选题]

**题目：** 12. ![图](./images/q_12_77_0.png) 机构积极开展青少年红色教育主题活动，引导青少年厚植爱国、爱党、爱社会主义的情感，让红色基因、革命薪火代代传承。这体现了青少年社会工作的（）内容。

**选项：**
```
A.习惯养成
B.心理疏导
C.思想引导
D.权益保护
```

> **正确答案：** C

**解析：**
思想引导为青少年提供思想道德教育辅导，引导青少年积极践行社会主义核心价值体系，形成正确的世界观、人生观、价值观。根据题干“开展青少年红色教育主题活动，引导青少年厚植爱国、爱党、爱社会主义的情感”可知为思想引导。习惯养成为青少年提供正确的行为指导和良好的习惯训练，帮助青少年形成正确的生活、学习和行为习惯。

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_77_0.png) 一项针对困境青少年的社会工作服务项目。该项目围绕困境青少年自信心不足，自尊感较低等问题，开展了自我认知，自我探索和自我成长的户外探险团体体验活动，通过完成合作任务提升困境青少年的自信和自尊。上述项目中，社会工作者采用的服务方法是（）。

**选项：**
```
A.家庭治疗
B.朋辈辅导
C.历奇辅导
D.向导服务
```

## 第 30 题 [单选题]

**题目：** 30. ![图](./images/q_30_77_0.png) 先生向社会工作者倾诉，自己出狱后面对许多问题：有严重的心脏病，看病费用负担较重；几次求职被拒，生活靠得母亲微薄的退休金；几年前与妻子离异后，一直联系不上前妻与儿女。根据胡先生的表述，他呈现出明显特征是（）。

**选项：**
```
A.困难重重的生活特征
B.冲动好斗的人格特征
C.与社会严重脱节的社会特征
D.患得患失的心理特征
```

> **正确答案：** A

**解析：**
矫正服务对象的特点：具有困难重重的生活特征。罪犯一旦被判定有罪并处以刑罚，其自身和家庭遭受的冲击和改变是十分巨大的：家庭可能因此而解体；婚姻可能因此而结束：亲友可能因此而唾弃学业可能因此而中断：工作可能因此而丧失：健康可能因此而恶化：财产可能因此而化为乌有：社会地位可能因此而一落千丈。无论是在监狱服刑，还是在社区接受矫正，服务对象都会面临困难重重的生活压力。

## 第 32 题 [单选题]

**题目：** 32. ![图](./images/q_32_77_0.png) 小蒋接受委托，为未成年的社区服刑人员小芳开展服务，小芳向小蒋说自己已经怀孕，并打算找一个私人诊所去做人工流产，叮嘱小蒋不要告诉父母和老师，指下来小蒋在服务过程中，首先要妥善处理好（）。

**选项：**
```
A.为小芳保密和维护社会安全的关系
B.为小芳保密和维护学校纪律的关系
C.为小芳保密和维护其人身安全的关系
D.为小芳保密和维护其家庭稳定的关系
```

> **正确答案：** C

**解析：**
这道题的考点在于伦理抉择的优先级顺序。在社会工作伦理守则中，有一条清晰的底线：当保密原则与保护生命/防止严重伤害的原则冲突时，后者优先。
因此，小蒋最妥善的做法是：首先，与小芳进行深入沟通，耐心说明去私人诊所的极端风险，尝试说服她告知父母或寻求正规医疗帮助。如果小芳坚决不同意，小蒋应基于“保护生命”的最高原则，在履行必要的告知程序后（如向机构督导汇报），联系小芳的父母或法定监护人，共同介入以确保其安全。这虽然违背了保密承诺，但却是履行了更高层次的保护责任。
选项C准确地概括了小蒋面临的核心伦理困境和首要任务。
