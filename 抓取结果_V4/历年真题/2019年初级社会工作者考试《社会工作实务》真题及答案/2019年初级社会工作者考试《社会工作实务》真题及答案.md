# 2019年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_76_0.png) 城镇移到了大城市，丈夫常常出差，没有时间陪她，小陈在陌生的环境里没有朋友， 非常孤单， 几次应聘工作失败， 感到很沮丧，为此，小陈找到社会工作者大李寻求相助，大李与小陈一同就需要解决的问题定了期望达到的具体目标， 并一 一进行了讨论，确定了优先顺序。 大李此举的目的是为了（ ） 。

**选项：**
```
A.解释服务目标
B.选择服务方案
C.筛选服务目标
D.执行服务方案
```

> **正确答案：** C

**解析：**
筛选目标： 这步工作是指社会工作者与服务对象一同找出期望达到的初步目标并对各个目标进行讨论， 以决定具体的目标。 社会工作者可以与服务对象一起将可能的目标写出来，然后与服务对象逐个对目标进行筛选。 当不能确定目标时可以使用从 1～10 的量变给每个待定的目标打分， 挑选出服务对象最希望和最迫切的改变目标.

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_76_0.png) 机构拟在杨村开展精准扶贫项目。 该机构社会工作者在项目计划中详细列出了服务的目的及目标.关注的问题和对象。 按照服务计划构成的要求该项目计划的内容还应包括（） 。

**选项：**
```
A.服务要使用的方法
B.服务要改变的人
C.服务要影响的政策
D.服务要改变的系统
```

> **正确答案：** A

**解析：**
服务计划由以下几部分内容构成： (一)目的及目标； (二)关注的问题与对象； (三)介入的方法和介入行动。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_76_0.png) 中， 为激发服务对象的内在动力， 社会工作者大刘注重服务对象的参与，由服务对象对合作社的生产经营和管理进行自主决策， 学习如何承担责任。 大刘选择上述介入行动所遵循的原则是()

**选项：**
```
A.瞄准精准扶贫的目标
B.尊重服务对象自决权
C.关注服务对象的特点
D.注重服务的经济效益
```

> **正确答案：** B

**解析：**
介入行动的原则： (一)以人为本.服务对象自决； (二)个别化； (三)考虑服务对象的发展阶段和他们的特点； (四)与服务对象相互依赖； (五)瞄准服务目标； (六)考虑经济效益。题中社会工作者大刘遵循的原则属于服务对象自决。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_76_0.png) 里没钱， 社工采用以家庭为中心的方法提供服务， 依据该方法的实务原则， 首先应该采取的措施是（）

**选项：**
```
A.链接医疗资源， 查明晕倒的原因
B.组建家庭工作团队， 拟定具体帮扶计划
C.联系孩子的父亲， 建议他每周与孩子通话
D.向村委会报告， 协助孩子一家申请困难救助
```

> **正确答案：** A

**解析：**
本题考查的是以家庭为中心的社会工作服务方法的实务原则， 在服务过程中儿童的安全和健康永远是第一需要考虑的， 本题也是顺序题， “首先” 优先选择“保命保安全” 安抚情绪的选项。 其他选项可以慢慢开展。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_76_0.png) .亲子关系等方面， 小王提供的上述服务， 其内容属于（） 。

**选项：**
```
A.家庭辅导
B.儿童问题辅导
C.亲职辅导
D.家庭团队工作
```

> **正确答案：** C

**解析：**
本题考查儿童社会工作服务中家庭支持服务内容， 儿童的营养， 行为习惯， 学业， 亲子关系等方面的服务主要是提升家长的亲职能力， 因此是亲职辅导。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_76_0.png) 工作原因无法照看孩子， 开办了“四点半课堂” ， 开展学生临时托管，课后学习辅导等服务， 该机构的上述工作属于（） 儿童福利服务。

**选项：**
```
A.支持性
B.保护性
C.补充性
D.替代性
```

> **正确答案：** C

**解析：**
本题靠的是儿童社会工作类型， 补充性儿童福利服务包裹为时间和精力不足的父母提供托育服务

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_76_0.png) 暴力， 为此， 社工首先应当（）

**选项：**
```
A.报案
B.寻求法律援助
C.对家长进行法制教育
D.安置到儿童福利机构
```

> **正确答案：** A

**解析：**
社会工作者在发现儿童遭受家庭暴力时，必须优先履行报案义务（A），通过法定程序确保儿童安全，而非自行决定安置（D）。报案是启动司法保护程序的第一步，通过公安机关和法院的介入，能够更系统地为儿童提供长期安全保障（如禁止施暴者接触儿童）。若社工未经报案直接安置，可能因缺乏法律支持导致儿童后续权益无法持续保障。
