# 2019年初级社会工作者考试《社会工作实务》真题及答案

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_76_0.png) 城镇移到了大城市，丈夫常常出差，没有时间陪她，小陈在陌生的环境里没有朋友， 非常孤单， 几次应聘工作失败， 感到很沮丧，为此，小陈找到社会工作者大李寻求相助，大李与小陈一同就需要解决的问题定了期望达到的具体目标， 并一 一进行了讨论，确定了优先顺序。 大李此举的目的是为了（ ） 。

**选项：**
```
A.解释服务目标
B.选择服务方案
C.筛选服务目标
D.执行服务方案
```

> **正确答案：** C

**解析：**
筛选目标： 这步工作是指社会工作者与服务对象一同找出期望达到的初步目标并对各个目标进行讨论， 以决定具体的目标。 社会工作者可以与服务对象一起将可能的目标写出来，然后与服务对象逐个对目标进行筛选。 当不能确定目标时可以使用从 1～10 的量变给每个待定的目标打分， 挑选出服务对象最希望和最迫切的改变目标.

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_76_0.png) 暴力， 为此， 社工首先应当（）

**选项：**
```
A.报案
B.寻求法律援助
C.对家长进行法制教育
D.安置到儿童福利机构
```

> **正确答案：** A

**解析：**
社会工作者在发现儿童遭受家庭暴力时，必须优先履行报案义务（A），通过法定程序确保儿童安全，而非自行决定安置（D）。报案是启动司法保护程序的第一步，通过公安机关和法院的介入，能够更系统地为儿童提供长期安全保障（如禁止施暴者接触儿童）。若社工未经报案直接安置，可能因缺乏法律支持导致儿童后续权益无法持续保障。

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_76_0.png)  社工联络物业公司对全小区的道路.活动场所.公共厕所等进行了适老化改造， 并且组建了志愿者队伍在小区内进行巡查。 上述措施满足了老年人（） 的需要。

**选项：**
```
A.健康维护
B.居家安全
C.社会参与
D.就业休闲
```

> **正确答案：** B

**解析：**
社工通过改善小区公共区域的安全性，保障老年人在日常活动中的基本安全需求，属于 居家安全 的实务范畴。答案 B 正确。
核心需求：居家安全。（1）适老化改造的范畴：“居家安全”中的“居家”并非仅指家庭内部，而是指老年人日常生活的主要环境，包括家庭及周边社区（如小区道路、活动场所、公共厕所等）。这些区域的改造（如防滑处理、扶手安装、夜间照明）直接降低老人跌倒风险，属于居家安全的延伸范围。
（2）社工服务的实务界定：在社会工作领域，“居家安全服务”通常涵盖 居住环境的整体安全性。例如，《老年人权益保障法》明确要求社区为老年人提供安全便利的公共环境，适老化改造正是实现这一目标的直接手段。
A 健康维护：涉及疾病预防、健康管理等，与题目中的环境安全干预无直接关联。
C 社会参与：指向老年人参与社区活动或决策，而志愿者巡查队伍主要由外部人员组成，未直接促进服务对象（老人）的社会参与。
D 就业休闲：与题目中的安全防护措施无关。
