```
A.家庭重建与生计恢复
B.社区治理与融合
C.公共服务体系建设
D.社区互助网络搭建
```

> **正确答案：** A

**解析：**
本题考查灾后恢复重建阶段的社会工作任务，需结合选项核心特征判断：
选项A：家庭重建与生计恢复的核心是帮助受灾群众恢复生产能力、获得稳定的收入来源，保障基本生活。题干中社工培育农产品产销合作社、链接销售渠道、开展种植培训、建立供货合作，都是围绕提升村民的生产经营能力、恢复生计展开，直接对应生计恢复的任务，因此A正确。

**题目：** 32. ![图](./images/q_32_50_0.png) 事件总体应急预案》(2025年),下列不属于事故灾难类别的是（    ）。

**选项：**
```
A.工矿商贸生产安全事故
B.海上溢油事件
C.群体性不明原因疾病
D.网络安全事件
```

> **正确答案：** C

**解析：**
本题主要考查《国家突发事件总体应急预案》（2025年版）中对突发事件的分类。根据该预案，突发事件根据其性质、过程和机理，主要分为以下四类：
（1）自然灾害：主要包括水旱灾害、气象灾害、地震灾害、地质灾害、海洋灾害、生物灾害和森林草原火灾等。
（2）事故灾难：主要包括工矿商贸等企业的各类安全事故、交通运输事故、公共设施和设备事故、环境污染和生态破坏事件等。
（3）公共卫生事件：主要包括传染病疫情、群体性不明原因疾病、食品安全和职业危害、动物疫情，以及其他严重影响公众健康和生命安全的事件。
（4）社会安全事件：主要包括恐怖袭击事件、民族宗教事件、经济安全事件、涉外突发事件和群体性事件等。

A 工矿商贸生产安全事故（错误） ：此项明确属于"事故灾难"中"工矿商贸等企业的各类安全事故"的范畴，故不属于本题应选之项。
B 海上溢油事件（错误） ：该项属于"事故灾难"中"环境污染和生态破坏事件"的范畴，故不属于本题应选之项。
C 群体性不明原因疾病（正确） ：根据预案，此项明确属于"公共卫生事件"，而非"事故灾难"，故符合题意。
D 网络安全事件（错误） ：该项属于"事故灾难"中"公共设施和设备事故（网络安全基础设施）"或"社会安全事件"的范畴，故不属于本题应选之项。
