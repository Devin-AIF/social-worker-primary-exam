## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_52_0.png) 途中丢失包裹，被公司要求全额赔偿，小王为此而焦虑失眠，于是向社会工作者求助。社会工作者首先协助小王查看公司规章制度中关于包裹丢失赔偿的条款，再指导小王收集天气恶劣导致派送延误、包裹包装存在瑕疵等证据，最后协助小王与公司协商调整赔偿比例。这一服务过程最能体现新就业群体社会工作服务（    ）的专业特性。

```
A.公益优先，聚焦困难
B.助人自助，赋能为本
C.任务导向，快速解决
D.专业伦理，保障关系
```

**解析：**

通过协商过程增强小王的维权意识和能力，实现从“被动接受帮助”到“主动维护权益”的转变。

## 第 12 题 [单选题]

**题目：** 12. ![图](./images/q_12_52_0.png) 年龄偏大、夜间视力下降，担心代驾过程中出现交通意外，同时担心被平台辞退，近期频繁出现焦虑、失眠现象，影响了正常工作。社会工作者为其提供服务时，结合优势视角理论，下列做法正确的是（    ）。

**选项：**
```
A.建议老陈立即辞职，避免意外风险
B.协助老陈梳理多年代驾经验、熟悉本地路况的优势，链接短途代驾、白天陪练等适配岗位资源
C.帮助老陈疏导焦虑情绪，指导其学习放松技巧
D.与平台沟通，为老陈申请特殊派单保护政策
```

> **正确答案：** B

**解析：**
本题考查优势视角理论的核心内涵。优势视角理论的核心是聚焦服务对象自身的优势和资源，而不是只关注其问题和困境，强调通过挖掘、发挥服务对象的优势来解决问题、实现发展。
选项A：建议老陈辞职只看到了老陈年龄大、视力差的问题，没有考虑其自身优势，属于问题视角的做法，不符合优势视角。
选项B：协助老陈梳理多年代驾经验、熟悉路况的优势，是挖掘其自身的优势资源，链接适配的岗位资源是基于其优势匹配合适的发展路径，完全符合优势视角“聚焦优势、发挥优势”的核心要求，表述正确。
选项C：疏导焦虑情绪、教授放松技巧属于心理干预的做法，聚焦的是老陈“焦虑失眠”的问题，属于问题视角的应对方式，没有体现对优势的挖掘，因此不符合优势视角。
选项D：与平台沟通申请特殊保护政策是从外部环境层面解决问题，没有挖掘老陈自身的优势，不属于优势视角的做法。

## 第 14 题 [单选题]

**题目：** 14. ![图](./images/q_14_52_0.png) 餐途中发生交通事故，因未与平台签订劳动合同，平台拒绝承担医疗费用。社会工作者介入后，首先需要开展的工作是（    ）。

**选项：**
```
A.申请社区临时救助，解决生活困难
B.收集接单记录、沟通记录等证据，为维权奠定基础
C.组织其他外卖骑手集体抗议平台
D.帮助该外卖骑手寻找新的就业岗位，避免后续纠纷
```

> **正确答案：** B

**解析：**
本题考查新就业群体权益受损时的社会工作介入优先级，核心是优先解决核心矛盾的基础问题：
选项A：申请临时救助属于后续的生活保障措施，应在确认劳动关系、明确维权路径后，根据服务对象的实际生活困难情况酌情开展，不是首先需要开展的工作，因此A错误。
选项B：外卖骑手因未签订劳动合同导致平台拒赔，维权的核心前提是证明与平台存在事实劳动关系，收集接单记录、沟通记录等证据是开展维权工作的首要基础，只有固定证据才能后续通过劳动仲裁、诉讼等方式主张医疗费用等合法权益，因此B是首先需要开展的工作。
选项C：组织集体抗议属于非理性维权方式，违反社会工作专业伦理，可能导致矛盾升级，损害骑手的合法权益，因此C错误。
选项D：帮助寻找新工作属于长远发展层面的服务，应在当前的工伤维权问题基本解决后，根据服务对象的意愿开展，不是首要工作，因此D错误。

## 第 16 题 [单选题]

**题目：** 16. ![图](./images/q_16_52_0.png) 就业群体设计职业健康自主管理小组服务，拟邀请医生讲解职业疾病预防知识，组织组员分享健康管理经验，协助其制定个人健康计划。该小组服务设计主要依据的理论是（    ）。

**选项：**
```
A.社会学习理论
B.精神分析理论
C.行政命令理论
D.危机干预理论
```

> **正确答案：** A

**解析：**
本题考查小组工作的理论基础。
选项A：社会学习理论认为个体的行为可以通过观察他人、分享经验、正向强化等方式习得，强调“观察学习”“经验分享”在行为改变中的作用。本题中邀请医生讲解知识是让组员观察学习专业内容，组织组员分享健康管理经验是通过同伴经验的观察和交流学习健康管理方法，协助制定个人健康计划是将学习到的内容转化为自身行为，完全符合社会学习理论的核心逻辑，表述正确。
选项B：精神分析理论聚焦于通过探索早年经历、潜意识冲突解决心理问题，比如为存在童年创伤的服务对象提供心理疏导，探索其童年经历对当前行为的影响，本题未涉及潜意识或早年经历的探索，因此不适用。
选项C：行政命令理论不属于社会工作的专业理论，指依靠行政层级的指令开展工作，比如上级部门向下级发布行政命令要求完成某项任务，与本题的小组服务无关。
选项D：危机干预理论适用于个体遭遇突发危机事件（如自然灾害、亲人离世、重大事故等）时的紧急干预，目标是帮助个体恢复心理平衡、度过危机，比如为经历火灾的居民提供紧急心理支持，本题中没有突发危机场景，因此不适用。

## 第 52 题 [多选题]

**题目：** 18. ![图](./images/q_52_52_0.png) 就业群体开展小组社会工作服务，结合不同小组类型的核心功能设计服务方案。下列小组服务设计中，符合对应小组类型核心功能且适配新就业群体需求的有（    ）。

**选项：**
```
A.支持型小组：组织外卖骑手分享配送压力应对经验，构建同伴情感支持网络
B.教育型小组：为电商客服开展客户沟通技巧提升培训，提升职业技能水平
C.治疗型小组：为长期接触负面信息的内容审核员开展情绪疏导小组，缓解职业倦怠与情绪耗竭
D.任务型小组：组织新就业群体代表成立权益倡导小组，收集群体诉求并推动社区完善服务驿站功能
E.娱乐型小组：组织新就业群体开展趣味游戏，不关联职业困境解决或能力提升目标
```

> **正确答案：** ABC

**解析：**
根据《初级社会工作综合能力》教材，小组工作的核心类型为教育小组、成长小组、支持小组、治疗小组四类，“任务型小组”不属于初级考试范围内的标准小组类型。
各选项分析：
A选项正确：支持小组由具有共同问题的成员组成，通过经验分享构建情感支持网络，符合支持小组功能。
B选项正确：教育小组旨在帮助成员学习新知识、新技能，提升解决问题的能力，沟通技巧培训符合教育小组功能。
C选项正确：治疗小组针对行为或情绪存在问题的成员，通过专业干预缓解症状，情绪疏导缓解职业倦怠符合治疗小组功能。
D选项错误：“任务型小组”不属于初级综合能力规定的标准小组类型，该题考查教材规定的小组类型对应关系，因此D选项不符合要求。
E选项错误：娱乐型小组不属于核心小组类型，且小组服务应围绕服务对象需求设计，完全不关联需求的娱乐活动不符合专业服务要求。
