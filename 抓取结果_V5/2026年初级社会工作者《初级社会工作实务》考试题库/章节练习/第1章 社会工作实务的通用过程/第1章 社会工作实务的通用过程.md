# 第一章　社会工作实务的通用过程

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_176_0.png) 机构拟在杨村开展精准扶贫项目。该机构社会工作者在项目计划中详细列出了服务的目的及目标、关注的问题和对象。按照服务计划构成的要求，该项目计划的内容还应包括（）。

**选项：**
```
A.服务要使用的方法
B.服务要改变的人
C.服务要影响的政策
D.服务要改变的系统
```

> **正确答案：** A

**解析：**
（考点:服务计划的构成）社会工作服务计划的完整构成包括三部分：①目的及目标；②关注的问题与对象；③介入的方法和介入行动。题干中已经列出了目的及目标、关注的问题和对象，因此还缺少服务要使用的方法（A选项）。选项B服务要改变的人属于关注的对象范畴，题干已经涵盖；选项C影响政策、选项D改变的系统不属于服务计划的必备构成要素。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_176_0.png) 的身体健康，某养老院实行封闭管理，老人与外界的交流受到影响，部分老人情绪烦躁，心情低落。该养老院社会工作部计划为老人开展与外界线上交流的服务。负责此项工作的社会工作者需要识别开展线上服务可使用的资源，以满足老人与外界交流的需要。据此，社会工作者应该识别的老人及环境中的积极因素是（    ）。

**选项：**
```
A.老人的身体健康状况
B.老人的心理健康状况
C.老人家庭成员的意愿
D.社区志愿者组织资源
```

> **正确答案：** D

**解析：**
在优势视角和社会工作评估中，服务对象自身的能力和意愿本身就是最重要的资源。A和B是基础性评估内容，而D是需要主动开拓的关键性执行资源，因此D更符合本题在特定情境下的考查重点。
所以，这道题的答案选择D，是基于一种侧重于 “外部资源链接” 和 “项目可执行性” 的实务思维。

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_176_0.png) 你选择的目标去做，你觉得对你会有什么影响?有哪些因素能够帮助你实现这个目标?有没有什么障碍?”社会工作者老李对小明说这番话的目的是（    ）。

**选项：**
```
A.明确服务目标
B.确定目标的可操作性与现实性
C.向服务对象解释设定目标的目的
D.帮助小明思考目标的可行性和可能的利弊
```

> **正确答案：** D

**解析：**
本题考查个案工作中设定目标阶段的沟通目的，逐一分析选项：
选项A：明确服务目标是指社工和服务对象共同梳理具体要实现的目标内容，题干中社工的提问是在目标已经初步选定的前提下展开的，并非用于明确目标本身，因此A错误。
选项B：确定目标的可操作性与现实性通常是社工基于专业知识进行评估的工作，且核心是判断目标是否可落地、可实现，而题干中的提问是引导服务对象主动思考，并非社工单方面确定目标的可行性，因此B错误。
选项C：向服务对象解释设定目标的目的是指向服务对象说明为什么要设定目标、设定目标有什么作用，题干未涉及相关解释内容，因此C错误。

选项D：社工通过询问目标的影响、有利因素和障碍，核心是引导服务对象主动思考目标是否符合自身需求、是否具备实现条件，以及目标达成的利弊，帮助服务对象更清晰地认识自己选择的目标，符合题意，因此D正确。

## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_176_0.png) 助小曹所在的中学办理学籍证明材料：联系社区青少年中心，专门安排老师一对一地教他学习电脑，并担任中心临时电脑管理助理：还做了小曹爷爷和姑姑的工作，让他们一起来监督他的日常生活。这种介入属于（）。

**选项：**
```
A.直接介入
B.间接介入
C.整体介入
D.个别介入
```

> **正确答案：** B

**解析：**
（考点:介入的分类）直接介入：社会工作者直接面对服务对象本人开展的工作。例如，社工亲自为小曹做心理辅导、直接教他学习技巧、和他一起制定生活计划。这里的互动是“社工 ↔ 小曹”。
间接介入：社会工作者代表服务对象，去改变他周围的环境和系统，从而间接地帮助他。工作对象是 “环境系统”（如学校、社区、家庭），而不是服务对象本人。
题目中社工的所有行动，工作对象都不是小曹本人，而是他周围的系统：
1、帮助学校办理学籍证明 → 工作对象是 学校（外部系统）。
2、联系社区青少年中心安排老师 → 工作对象是 社区机构（外部系统），并连接了“老师”这个重要他人。社工并没有自己教。
3、做爷爷和姑姑的工作 → 工作对象是 家庭成员（支持系统）。
这些行动的共性：社工都在扮演一个 “资源链接者”和“系统改变者” 的角色。他通过改善小曹的就学环境（学校）、学习支持环境（社区中心）、家庭监督环境（家人）来间接地帮助小曹成长。社工和小曹之间没有发生直接的专业互动。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_176_0.png) ，社会工作者小李和小刘对是否需要继续为外来务工青年大明提供服务产生了分歧。小李认为大明已经返回原籍，个案就结案了。小刘则建议联络大明户籍所在地的社会工作服务机构，由他们继续为大明提供服务，以巩固大明改变的信心和成果。小刘上述建议依据的是对结案阶段（ ）的要求。

**选项：**
```
A.转介服务
B.跟进服务
C.时机选择
D.情绪处理
```

> **正确答案：** B

**解析：**
选项A，转介服务，是指在服务对象的需求超出当前服务机构的能力范围时，将他们转介到其他更适合的机构或资源。而题目中的情况是大明已经返回原籍，但小刘建议继续提供服务，这更像是为了巩固成果，而不是因为需求超出当前机构的能力。

选项B，跟进服务，是指在结案后，通过定期的联系和服务，帮助服务对象维持和巩固已经取得的改变。这与小刘的建议相符，即通过联络户籍所在地的社会工作服务机构继续提供服务，以巩固大明的改变成果。
选项C，时机选择，是指在结案阶段选择合适的时机结束服务，确保服务对象已经准备好接受结案。题目中的分歧在于是否需要继续提供服务，而不是关于结案时机的选择。
选项D，情绪处理，是指在结案阶段帮助服务对象处理可能的情绪反应，如失落、焦虑等。虽然结案可能会引起服务对象的情绪波动，但题目中的建议更侧重于服务的延续，而不是情绪处理。
综上所述，小刘的建议依据的是对结案阶段跟进服务的要求，因此正确答案是B.跟进服务

## 第 11 题 [单选题]

**题目：** 11. ![图](./images/q_11_176_0.png) 从农村来到城市照顾孙子，由于人生地不熟，语言沟通困难，特别不适应，想回农村。但是她觉得儿子儿媳整天忙于工作，孙子无人照顾，又不忍心向儿子开口。为了帮助赵奶奶识别问题的主观因素，社会工作者的下列做法中正确的是（    ）。

**选项：**
```
A.询问赵奶奶为解决这一问题作出了什么努力
B.了解赵奶奶之前是如何应对类似问题的
C.与赵奶奶确认这一问题是什么时候开始的
D.引导赵奶奶说出她自己的担心背后的原因
```

> **正确答案：** D

**解析：**
这道题的考点在于，社会工作者如何帮助服务对象区分问题的 “客观事实” 与 “主观认知和感受”。
赵奶奶面临的客观事实是：在城市不适应、语言不通、想回农村。但导致她陷入困境、无法解决问题的关键主观因素是：她内心“不忍心向儿子开口”的这种担心、顾虑和感受。在赵奶奶的案例中，她的主观因素（内心的不忍和担心）是导致客观问题（不适应却无法离开）持续存在的关键。因此，社会工作者要帮助她识别并厘清这些主观因素，才能找到解决问题的突破口。
客观因素：问题本身的事实、表现、时间、行动等（对应A、B、C）。
主观因素：服务对象对问题的看法、感受、情绪、价值观和内在冲突（对应D）。

## 第 13 题 [单选题]

**题目：** 13. ![图](./images/q_13_176_0.png) ，李女士与丈夫经常因是否接送孩子上下学发生冲突。李女士认为，孩子从小体质弱，比较敏感，即使已上初中，为安全起见还是应该接送，但丈夫不以为然，还以工作忙为由将接送孩子的事甩给了李女士。李女士对丈夫的行为很不解，怀疑丈夫有了“外心”，非常苦恼、困惑，遂求助社会工作者小齐。为了识别李女士问题产生的主观因素，小齐应该了解（ ）。

**选项：**
```
A.丈夫对夫妻问题的看法
B.李女士为处理夫妻问题所作的努力
C.李女士对夫妻问题的看法
D.丈夫为处理夫妻问题所作的努力
```

> **正确答案：** C

**解析：**
A选项关注的是丈夫的看法，这属于客观因素，因为它是从第三方的角度来了解问题，而不是求助者本人的主观感受。

B选项关注的是李女士为解决问题所作的努力，这可能涉及到她的行为和行动，但不一定直接反映她的主观感受。
C选项关注的是李女士对夫妻问题的看法，这直接反映了她的主观感受和内心想法，是识别主观因素的关键。
D选项关注的是丈夫为解决问题所作的努力，这同样属于客观因素，因为它是从丈夫的角度来看问题的。
因此，为了识别李女士问题产生的主观因素，社会工作者小齐应该了解李女士对夫妻问题的看法，即选项C。这有助于理解李女士的内心感受和她对问题的看法，从而更好地帮助她解决问题。
其他选项虽然可能有助于全面了解问题，但它们更多地关注客观行为或第三方的看法，而不是求助者的主观感受。因此，正确答案是C。

## 第 15 题 [单选题]

**题目：** 15. ![图](./images/q_15_176_0.png) 孩子去哪个学校上学发生争吵，情急之下还动手打了起来，为此两人闹得不可开交。沟通中社工问他们："根据刚才你们讲的情况，你们觉得问题出在哪里？"社会工作者这样是为了（）。

**选项：**
```
A.了解问题
B.了解需求
C.界定问题
D.界定需求
```

> **正确答案：** C

**解析：**
考点：界定问题。
本题考查的知识点是社会工作中“界定问题”的含义。在社会工作的通用过程模式中，“界定问题”是指社会工作者与服务对象一起找出并确认问题的本质、范围和成因。一个关键的技巧是，社会工作者需要从服务对象自己对问题的看法开始。题目中，社会工作者询问夫妻俩“你们觉得问题出在哪里？”，正是为了引导服务对象表达他们自身的感受和理解，从而共同界定（明确）他们需要共同面对和解决的核心问题是什么。这与简单“了解问题”（A选项，侧重于收集信息）或“（界定）需求”（B、D选项，侧重于明确需要什么样的改变或服务）在侧重点上有所不同。因此，社会工作者的提问是为了 C. 界定问题。

## 第 16 题 [单选题]

**题目：** 16. ![图](./images/q_16_176_0.png) 表服务对象采取行动，通过介入服务对象以外的其他系统间接帮助服务对象，属于介入行动中的（）介入。

**选项：**
```
A.直接介入
B.间接介入
C.综合介入
D.临时介入
```

> **正确答案：** B

**解析：**
考点：间接介入的含义。本题考查介入行动的分类：

直接介入：社工直接和服务对象本人开展工作，比如给服务对象做心理辅导、技能培训；间接介入：社工不直接和服务对象互动，而是通过改变服务对象周围的环境、系统来帮助他，比如对接学校给服务对象办理入学、和服务对象的家人沟通调整家庭氛围，也叫“环境介入”；综合介入：同时开展直接和间接介入。
题干中“通过介入服务对象以外的其他系统间接帮助”完全符合间接介入的定义，因此答案选B。

## 第 17 题 [单选题]

**题目：** 17. ![图](./images/q_17_176_0.png) 一对关系紧张的父子将他们的互动方式展示出来，社会工作者可借此了解父亲和儿子各自的需要和感受，以及他们的问题在哪里。

**选项：**
```
A.会谈
B.询问
C.角色扮演
D.问卷

## 第 22 题 [单选题]

**题目：** 22. ![图](./images/q_22_176_0.png) 种方法督促小强的日常学习，导致小强出现逆反心理，不仅成绩没有提高，而且父子关系紧张。为了协助他们改善父子关系，社会工作者小马拟为其开展家庭治疗。小马设定的下列具体目标中，适宜的是（    ）。

**选项：**
```
A.缓解小强与父亲之间的冲突
B.增进父亲与小强之间的相互理解
C.降低父亲对小强的学习要求
D.增加父亲与小强正向沟通的次数
```

> **正确答案：** D

**解析：**
最适宜的具体目标不是关系状态的描述（A、B），也不是替家庭做决定（C），而是像D选项那样，设计一个可以立刻着手去做的、新的互动行为。
A是一个终极目标或期望结果，而非具体目标。“缓解冲突”确实是治疗的最终目的。但作为介入目标，它太模糊了。社会工作者无法直接操作“缓解冲突”这个动作。冲突是通过一系列具体的改变（如沟通方式、互动频率）后自然而然减少的。因此，它不适合作为指导每次治疗会谈的具体行动目标。
B是一个理想的关系状态，同样不是可操作的具体目标。“相互理解”是一种内在的心理感受和认知状态，无法被直接测量和干预。在父子情绪对立、沟通完全被“学习”话题堵塞的情况下，直接要求双方“理解”对方是极为困难的，甚至可能引发新的指责（“你为什么不理解我？”）。在家庭治疗中，理解通常是行为改变之后产生的结果，而不是治疗的起点。
C是一个越界的实质性决定，不属于社会工作者的专业目标。社会工作者在家庭治疗中的角色是 “过程的促进者” ，而不是 “内容的决定者” 。父亲对孩子的学习有要求，其本身可能包含合理的期望。工作者的专业工作点在于 “改善实现要求的过程与方式” ，例如改变催促、批评的方式，而不是直接去评判或改变要求的内容。直接要求父亲降低标准，会侵犯家庭的自决权，也未必能真正解决问题。
D是一个具体、可观察、可测量、可操作的优质治疗目标。
具体：可以明确界定什么是“正向沟通”（例如：表达关心、谈论学习以外的兴趣、一起进行休闲活动时的对话）。
可测量：可以记录和统计这类沟通发生的次数或时长，从而清晰看到变化。
可操作：社会工作者可以据此设计具体的“家庭作业”，例如：“本周请每天至少安排10分钟，父子一起做一件与学习无关的事，并轻松地聊聊天。”
治疗逻辑：这个目标直接切入 “行为改变” 。通过强制性地增加良性互动，首先打破“督促学习→逆反对抗→关系紧张→更严厉督促”的恶性循环。当积极互动的经验积累起来，冲突自然会减少，相互理解才有了生长的基础。这是从最容易改变的行为层面入手，逐步影响认知和情感的稳健策略。

## 第 23 题 [单选题]

**题目：** 23. ![图](./images/q_23_176_0.png) 在走访拆迁安置小区时，发现居民获得大量拆迁安置款后安于享乐，不思进取，有的家庭因此关系紧张。小陈开办了劳动教育价值观小组，和居民委员会合作，对居民开展财产规划培训，还对小区居民开展了家风家教宣传教育。小陈运用了社会工作介入的哪种方式?（ ）

**选项：**
```
A.直接介入
B.间接介入
C.综合介入
D.系统介入
```

> **正确答案：** C

**解析：**
社会工作的介入方式主要包括直接介入、间接介入和综合介入。直接介入是指社会工作者直接与服务对象接触，提供直接的服务，如个案工作、小组工作等。间接介入则是通过改变环境、政策或系统来影响服务对象，如社区工作、政策倡导等。综合介入则是将直接介入和间接介入结合起来，既直接服务个体，也通过改变环境来支持个体。

**题目：** 112. ![图](./images/q_112_176_0.png) 通过一节课的观察，记录了君君27次干扰同学、不听讲的行为。经过1个月的行为干预，再次观察记录，君君只有5次干扰同学的行为。这里小李运用了（ ）的评估方法。

A.服务对象满意度测量
B.目标核对
C.任务完成测量
D.基线测量
```

> **正确答案：** D

**解析：**
这道题目考察的是社会工作评估方法中的基线测量。基线测量是指在干预开始之前，对服务对象的行为或状况进行测量，以确定干预前的基线水平。在这个案例中，社会工作者小李首先记录了君君在干预前（一节课）的干扰行为次数，即27次。经过一个月的干预后，再次观察记录，发现君君的干扰行为减少到5次。这里，小李通过比较干预前后的行为变化来评估干预效果，这正是基线测量的应用。基线测量有助于确定干预措施的有效性，因为它提供了干预前后的对比数据。其他选项如服务对象满意度测量、目标核对和任务完成测量，虽然也是评估方法，但它们更侧重于服务对象的主观感受、目标达成情况或任务完成情况，而不是直接比较干预前后的行为变化。因此，正确答案是D.基线测量。
