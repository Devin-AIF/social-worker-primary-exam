## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_50_0.png) 梯引发居民集体信访，部分低层居民反对，高层居民则普遍强烈要求加装。街道信访部门组织听证，邀请了3名社区居民代表、1名电梯工程专家、1名律师组成听证员团队。针对上述听证表述正确的是（    ）。

**选项：**
```
A.正式听证，人数符合5人及以上单数要求
B.正式听证，人数未达到7人及以上单数要求
C.非正式听证，人数符合灵活设置要求
D.非正式听证，人数不符合法定标准
```

> **正确答案：** A

**解析：**
本题考查信访正式听证的法定要求，核心依据为《信访工作条例》及各地信访听证办法的统一规定。

选项A正确：首先，本次听证由街道信访部门依职权组织，属于有权处理信访事项的国家机关主动发起的正式听证，符合正式听证的启动程序要求；其次，正式听证的听证员人数要求为5至11人的单数，本次听证员共5人，符合法定人数标准。选项B错误：信访正式听证的法定人数下限为5人单数，不存在7人的最低要求，该表述不符合规定。选项C错误：非正式听证通常适用于简单信访事项，无严格的听证员组成要求，本次听证邀请了多方代表、专家和律师参与，程序严谨，属于正式听证范畴。选项D错误：本次听证属于正式听证，且人数符合法定标准，该表述完全错误。
补充说明：正式听证的启动方式包括两种：一是信访人提出听证申请后由有权机关决定受理，二是有权处理信访事项的国家机关根据工作需要主动组织听证，两种方式均属于正式听证的合法启动程序，因此用户提出的“正式听证需要申请和受理程序”的表述不全面，国家机关依职权主动组织的听证同样属于正式听证。

```
B.处理结果是否充分回应信访人的合法合理诉求

```

**解析：**
信访事项处理结果的合规性审核属于内部质量控制环节，核心审核内容聚焦于处理过程和结果的合法性、合理性，具体包括：

选项A属于核心内容：处理程序是否符合《信访工作条例》规定的时限、告知、调查等程序要求，是合规性审核的首要内容，确保程序正义。选项B属于核心内容：处理结果是否充分回应信访人的合法合理诉求，是审核处理决定是否全面、恰当的重要标准。选项C不属于核心内容：信访人满意度是服务效果评价指标，而非合规性审核的强制要求。根据信访工作规定，处理结果只要符合法律法规和政策要求、程序合规、回应了合法诉求，即使信访人不满意，也属于合规处理结果。满意度可作为工作改进的参考，但不影响合规性判定。选项D属于核心内容：处理结果是否符合相关法律法规和政策要求，是合规性审核的核心标准，确保处理决定的合法性。

## 第 12 题 [单选题]

**题目：** 12. ![图](./images/q_12_50_0.png) 产前曾承诺将商铺分配给职工王某抵付工资，但一直未兑现这一承诺且无书面证据，王某为此持续上访25年，其间与妻子争吵不断，与亲友逐渐疏远。社会工作者介入后，通过多次沟通发现王某内心深处更渴望家庭和睦，而非单纯追求拥有商铺。社会工作者随后协调王某与妻子进行家庭治疗，并邀请王某加入社区跑友圈，最终王某与政府签订息访协议。该案例中，社会工作者主要运用的策略是（    ）。

**选项：**
```
A.矛盾纠纷判断型调解方法
B.动机式会谈法
C.偏执型信访人综合干预策略
D.分裂型人格障碍信访人干预策略
```

> **正确答案：** C

**解析：**
偏执型信访人是指长期持续上访，对信访诉求存在执念，社会支持系统受损，且往往忽略自身其他合理需求的信访人员，对其的综合干预策略核心是“需求重构+社会支持重建+多元调解”。各选项分析如下：
选项A：矛盾纠纷判断型调解方法侧重于对纠纷事实的认定、责任的划分，本题中社会工作者的核心工作不是判断商铺纠纷的对错，而是关注王某的深层需求，因此A不选。
选项B：动机式会谈法主要用于激发服务对象改变的内在动机，常用于成瘾行为、行为矫正等领域，本题中社会工作者不仅采用了会谈方法，还开展了家庭治疗、链接社区资源等综合服务，因此B不选。
选项C：偏执型信访人综合干预策略针对长期上访、诉求固化、社会关系受损的信访人员，具体包括：挖掘服务对象的深层需求（本题中发现王某更渴望家庭和睦而非商铺）、修复受损的社会支持系统（开展家庭治疗改善夫妻关系）、重建社会连接（邀请加入跑友圈丰富社交生活）、推动多元协商达成息访共识，题干中的做法完全符合该策略的实施路径，因此C正确。
选项D：分裂型人格障碍属于精神障碍范畴，核心特征是认知扭曲、情感冷漠、行为怪异，题干中王某的表现是长期上访导致的社会关系受损，不属于人格障碍，因此D不选。

## 第 14 题 [单选题]

**题目：** 14. ![图](./images/q_14_50_0.png) 信访事项的复核意见不服，以同一事实和理由向信访部门再次上访。信访部门经甄别后，应作出的处理是（    ）。

**选项：**
```
A.予以受理并重新核查
B.认定为涉法涉诉事项转法院处理
C.属于“两个不再受理”情形，不予受理
D.责令孙某按逐级上访规定重新申请
```

> **正确答案：** C

**解析：**
本题考查信访工作的“两个不再受理”规定，根据《信访工作条例》，“两个不再受理”的具体情形是：

信访事项已经受理或者正在办理的，信访人在规定期限内向受理、办理机关、单位的上级机关、单位又提出同一信访事项的，上级机关、单位不予受理；信访人对复核意见不服，仍然以同一事实和理由提出投诉请求的，各级党委和政府信访部门和其他机关、单位不再受理。
本题中孙某对复核意见不服，以同一事实和理由再次上访，属于第二种情形，因此信访部门不予受理。
对各选项分析如下：选项A：对于复核终结的信访事项不再重新受理，因此错误。选项B：题干未提及该信访事项属于涉法涉诉事项，因此错误。选项C：符合“两个不再受理”的情形，应当不予受理，因此正确。选项D：复核意见为信访事项的终结意见，不能重新申请，因此错误。
综上本题选C。

## 第 35 题 [多选题]

**题目：** 2. ![图](./images/q_35_50_0.png) 信访事项时，需严格区分“三个不予受理”和“两个不再受理”。下列情形中，最易被误判的有（    ）。

**选项：**
```
A.信访人李某反映的劳动争议，已向法院提起诉讼且法院已立案
B.信访人王某对信访事项处理结果不服，未申请复核直接向省级信访部门上访
C.信访人赵某就同一拆迁补偿问题，在市信访部门办理期限内再次上访
D.信访人孙某对复核意见不服，以“政策存在漏洞”为由重新提出同一诉求
E.信访人周某未向乡镇信访部门反映，直接到市信访部门反映村集体资产侵占问题
```

> **正确答案：** BDE

**解析：**
本题考查信访事项的“三个不予受理”和“两个不再受理”规则，首先明确两类情形的定义：
三个不予受理（针对不符合受理条件的初次信访）：
属于人民代表大会及其常务委员会、人民法院、人民检察院职权范围的，或已经通过诉讼、仲裁、行政复议等法定途径解决的涉法涉诉事项，不予受理。
属于下级机关职权范围的越级信访事项，不予受理（应当告知信访人向有权处理的机关提出）。
同一信访事项已经受理或者正在办理过程中，信访人再次提出同一诉求的，不予受理。
两个不再受理（针对已经走完信访程序的重复信访）：
信访事项已经复查、复核终结，信访人对复核意见不服，仍然以同一事实和理由提出诉求的，不再受理。
涉法涉诉事项已经依法终结的，不再受理。
对各选项的分析如下：
选项A：李某的劳动争议已经法院立案，属于涉法涉诉事项，符合“三个不予受理”的第一种情形，归类明确，不易误判。
选项B：王某对处理结果不服，未申请复核直接到省级上访，不属于“三个不予受理”（因为已经过了办理期限，不属于正在办理的情形），也不属于“两个不再受理”（因为还没有走完复核程序），很容易被误判为越级上访而不予受理，属于易误判情形。
选项C：赵某在市信访部门办理期限内再次上访，属于“三个不予受理”的第三种情形，归类明确，不易误判。
选项D：孙某对复核意见不服仍以同一理由上访，属于“两个不再受理”的第一种情形，容易被误判为“三个不予受理”的越级或正在办理情形，属于易误判情形。
选项E：周某直接到市信访部门反映村集体问题，属于越级上访，符合“三个不予受理”的第二种情形，容易被误判为“两个不再受理”的重复信访情形，属于易误判情形。
因此最易被误判的是B、D、E选项。
