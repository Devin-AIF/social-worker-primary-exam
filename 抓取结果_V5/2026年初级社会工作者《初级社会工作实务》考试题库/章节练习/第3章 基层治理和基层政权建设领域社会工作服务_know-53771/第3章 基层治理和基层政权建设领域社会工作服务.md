## 第 6 题 [单选题]

**题目：** 6. ![图](./images/q_6_50_0.png) 机构对某老旧社区的适老化改造服务项目进行评估，社会工作者通过与居民深度访谈、参与式观察记录服务过程，撰写了包含服务成效、居民反馈、改进建议的评估报告。该评估方法的核心特征是（    ）。

**选项：**
```
A.定量评估
B.定性评估
C.过程评估
D.成效评估
```

> **正确答案：** B

**解析：**
本题考查社区服务评估方法的分类，核心关键词区分如下：
定量评估：关键词是“数据、统计、问卷、数值分析”，通过可量化的指标进行评估，比如统计服务人数、满意度评分的平均分等，本题中没有涉及量化数据，因此不选A。
定性评估：关键词是“访谈、观察、文字描述、主观感受”，通过非量化的资料（如访谈记录、观察笔记）分析服务效果，本题中通过深度访谈、参与式观察收集资料，撰写文字性的评估报告，符合定性评估的核心特征，因此B选项正确。
过程评估：关键词是“服务流程、执行步骤、是否按计划开展”，聚焦于服务实施的过程是否规范，本题中评估报告既包含过程也包含成效，不是单纯的过程评估，因此不选C。
成效评估：关键词是“服务效果、目标达成率、问题解决程度”，聚焦于服务最终带来的改变，本题中评估报告包含成效但不仅限于成效，因此不选D。

**题目：** 12. ![图](./images/q_12_50_0.png) 绿地被私占、楼道堆杂物等问题，虽然居民对此不满但参与治理的积极性不高。对此，社会工作者举办社区环境治理主题展览，展示私占绿地的危害、整洁社区的案例，并发放参与治理的倡议书。该做法的核心目的是（    ）。

```
B.提升居民的参与意愿
C.提高居民的参与能力
D.优化社区资源配置
```

> **正确答案：** A

**解析：**

选项A：促进居民对参与价值的肯定，核心是帮助居民建立“参与是有用的、有意义的”认知，属于认知层面的引导。题干中社工展示私占绿地的危害、整洁社区的案例，本质是通过事实传递“参与治理能够改善居住环境”的认知，帮助居民认识到参与的价值，因此符合该策略。选项B：提升居民的参与意愿，是在居民已经认可参与价值的基础上，通过降低参与成本、提供激励等方式，激发居民主动参与的动力，属于动机层面的引导，题干中社工的做法未涉及动力激发的内容，因此不符合。选项C：提高居民的参与能力是帮助居民掌握议事、沟通、问题解决等参与技能，题干未开展相关技能培训，因此不符合。选项D：优化社区资源配置是对社区人、财、物等资源的重新调配，题干中的展览属于宣传引导活动，不属于资源配置，因此不符合。

## 第 14 题 [单选题]

**题目：** 14. ![图](./images/q_14_50_0.png) 因宅基地边界问题产生纠纷，甲认为乙占用了自家10厘米地界，乙坚决否认。村委会介入前，双方曾多次私下沟通但未达成一致。此时最基础的调解方式是（    ）。

**选项：**
```
A.村委会调解
B.自行协商
C.人民调解委员会调解
D.法律诉讼
```

> **正确答案：** A

**解析：**
本题考查民间纠纷的解决方式，正确答案为A。
首先明确两个核心概念：纠纷解决方式包括自行协商、调解、行政裁决、诉讼等，其中“调解”特指由中立第三方居中斡旋，促成双方达成一致的纠纷解决方式，自行协商是双方自行沟通，没有第三方参与，因此不属于调解方式。
 选项A正确：题干中明确说明双方已经多次私下沟通未达成一致，说明自行协商已经失败，此时进入第三方调解环节。村委会是基层群众自治组织，承担民间纠纷调解的法定职责，属于最基础的第三方调解方式，因此为正确答案。
选项B错误：自行协商不属于“调解方式”的范畴，且已经失败，因此不选。
选项C错误：人民调解委员会是专门的人民调解组织，但村委会调解是人民调解的基层形式，对于村民之间的宅基地纠纷，村委会作为最贴近村民的基层组织，是更基础的调解主体，因此C不是最基础的。
选项D错误：法律诉讼是纠纷解决的最终途径，成本最高、程序最复杂，不属于基础调解方式。

**题目：** 17. ![图](./images/q_17_50_0.png) 机构入驻某乡村开展乡村振兴服务，协助村集体梳理特色农产品资源，链接电商平台拓宽销售渠道，培育村民合作社。在此过程中，社会工作者不承担的角色是（    ）。

## 第 20 题 [单选题]

**题目：** 20. ![图](./images/q_20_50_0.png) 对辖区内无物业小区的环境整治问题，牵头组织居民议事会，协调环卫、城管等部门提供资源支持，最终推动成立居民自治管理小组。关于街道办事处的角色定位，下列说法正确的是（    ）。

**选项：**
```
A.街道办事处是基层群众性自治组织
B.街道办事处是城市基层政权的重要组成部分
C.街道办事处是县级政府的派出机关
D.街道办事处是独立的行政机关
```

> **正确答案：** C

**解析：**
本题考查街道办事处的法定性质。

选项A：基层群众性自治组织是指城市居民委员会和农村村民委员会，由居民/村民选举产生，不属于行政机关体系，街道办事处是政府派出的行政性机构，因此该选项错误。选项B：城市基层政权是指市辖区、不设区的市的人民代表大会和人民政府，街道办事处是基层政权的派出机构，并非政权本身的组成部分，因此该选项表述不准确。选项C：根据《地方各级人民代表大会和地方各级人民政府组织法》规定，街道办事处是市辖区、不设区的市的人民政府的派出机关，属于县级政府的派出机关，该选项表述符合法律定义，因此正确。选项D：独立的行政机关能够以自己的名义独立行使行政权力、承担法律责任，街道办事处的权力来源于派出政府的委托，不具备独立的行政主体资格，因此不属于独立的行政机关，该选项错误。
综上本题选C。

选项A：参与价值认知是居民对“参与是否有意义”的判断。题干中“部分居民认为参不参与都一样，解决不了实际问题”，是对参与价值的否定，属于该影响因素。选项B：参与意愿是居民参与活动的主观动力。题干中“部分居民因工作忙没时间也不愿参与”，“不愿”直接体现了参与意愿不足，属于该影响因素。选项C：参与能力包括居民的表达能力、参与技能、时间精力等。题干中“居民因不会表达意见不敢参与”属于表达能力不足，“工作忙没时间”属于时间资源不足，均属于参与能力的范畴，因此该选项符合要求。选项D：制度保障是指是否有完善的参与规则、反馈机制等外部制度支持，题干未提及制度相关问题，因此不符合。选项E：天气条件属于偶发的外部环境因素，并非影响居民参与的核心因素，因此不符合。

## 第 41 题 [多选题]

**题目：** 8. ![图](./images/q_41_50_0.png) 量外来务工人员，社会工作者开展外来人口融入社区服务。下列服务措施中，能同时提升外来人口适应能力和促进社区融合的有（    ）。

**选项：**
```
A.开展权益保障讲座，提高外来人口维权能力
B.组织外来人口与本地居民共同参与社区志愿服务
C.成立外来人口互助小组，分享生活和工作经验
D.推动外来人口参与社区议事协商，表达合理诉求
E.为外来人口提供子女入学咨询服务
```

> **正确答案：** ABCD

**解析：**
本题要求同时满足两个条件：提升外来人口适应能力 + 促进社区融合。
对各选项分析如下：
选项A：开展权益保障讲座，提高外来人口维权能力。直接提升外来人口在城市生活中的适应能力（了解自身权益），但讲座形式主要面向外来人口自身，对“促进社区融合”（与本地居民的互动）的直接作用有限。本题要求“同时提升”两者，A在这方面的作用较弱，但仍有间接促进融合的效应。
选项B：组织外来人口与本地居民共同参与社区志愿服务。外来人口与本地居民在志愿服务中建立互动和联系，既能提升外来人口的社区归属感和适应能力，又能直接促进社区融合，正确。
选项C：成立外来人口互助小组，分享生活和工作经验。首先能够提升外来人口的适应能力（通过经验分享解决生活工作问题）；其次，互助小组能够增强外来人口的社会支持网络，为其后续参与社区生活奠定基础，是促进社区融合的重要中间环节，因此符合“同时提升”的要求，应当入选。
选项D：推动外来人口参与社区议事协商，表达合理诉求。参与社区公共事务能够提升外来人口的社区主人翁意识（适应能力提升），同时也是与社区原有治理体系互动的重要途径，有助于促进融合，正确。
选项E：为外来人口提供子女入学咨询服务。仅能解决外来人口的具体生活困难，属于单项服务，未体现促进社区融合的作用，不符合要求。
因此，正确答案为A、B、C、D。
