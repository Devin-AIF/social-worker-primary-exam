# 第五章 志愿服务的法规与政策

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_47_tit_0.png) 务相关规定，关于我国志愿者参与境外志愿服务的表述，正确的是（ &nbsp;&nbsp;&nbsp;）。

**选项：**
```
A.政府开展的境外志愿服务无须遵守专门法规规范
B.非政府组织开展的境外志愿服务仅需遵守我国法律
C.境外志愿服务在中国境内的志愿者招募需遵照《志愿服务条例》
D.国内非政府组织开展的境外志愿服务仅涉及紧急援助类活动
```

> **正确答案：** C

**解析：**
境内志愿服务组织、志愿者到境外服务
