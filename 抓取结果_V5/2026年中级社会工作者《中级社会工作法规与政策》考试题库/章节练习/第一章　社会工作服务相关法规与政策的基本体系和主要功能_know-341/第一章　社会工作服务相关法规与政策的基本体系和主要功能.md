# 第一章　社会工作服务相关法规与政策的基本体系和主要功能

## 第 50 题 [单选题]

**题目：** 50. ![图](./images/q_50_125_tit_0.png) 社会保障部门在制定本地社会工作专业人才队伍建设规划时，参考了《社会工作专业人才队伍建设中长期规划(2011-2020年)》,该规划中提出的社会工作人才队伍建设的根本是（ &nbsp;&nbsp;&nbsp;）。

**选项：**
```
A.人才培养
B.人才使用
C.人才评价激励
D.政策制度建设
```

> **正确答案：** 未知

**解析：**
《社会工作专业人才队伍建设中长期规划(2011-2020年)》明确提出，社会工作人才队伍建设的总体要求是：以人才培养为基础，以人才使用为根本，以人才评价激励为重点，以政策制度建设为保障，着力建设一支规模宏大、结构合理、素质优良的社会工作专业人才队伍。 其中“人才使用是根本”是因为培养人才的最终目的是合理使用人才，让专业人才在服务实践中发挥作用，人才只有得到有效使用，其价值才能得到体现，队伍建设的目标才能实现。
