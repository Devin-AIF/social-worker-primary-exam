# 第一章　社会工作服务相关法规与政策的基本体系和主要功能

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_125_tit_0.png) 有“专家”和（ ）的双重角色，使得他们在相关法规和政策的制定、完善和发展过 程中能发挥重要作用。

**选项：**
```
A.“倡导者”
B. “基层合作者”
C.“基层实践者”
D. “基层创新者”
```

> **正确答案：** 未知

**解析：**
（考点：社会工作实践对社会工作法规与政策的作用）本题考查社会工作实践对社会工作法规与政策的作用。社会工作者在此过程中扮演双重角色： “专家”角色：指社会工作者基于专业理论、实务经验和对社会问题的深入研究，能够为法规政策的制定、修订提供专业的建议和依据。 “基层实践者”角色（C选项）：指社会工作者身处服务一线，直接面对服务对象和各类社会问题，是法规政策的具体执行者和效果体验者。他们能最直接、最真实地反馈政策在基层的执行情况、存在的问题以及服务对象的新需求，从而为政策的完善提供鲜活的实践依据。

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_125_tit_0.png) 从各个方面具体论述了加强和改善民生的政策主张。在教育政策方面，提出了“努力办好人民满意的教育”,并重点强调了要“大力促进教育公平”,具体措施不包括（ &nbsp;&nbsp;&nbsp;）。

**选项：**
```
A.合理配置教育资源，重点向农村、边远、贫困、民族地区倾斜
B.支持特殊教育，提高家庭经济困难学生资助水平
C.积极推动农民工子女平等接受教育
D.实行公办学校标准化建设和校长教师交流轮岗，不设重点学校、重点班
```

> **正确答案：** 未知

**解析：**
党的十八大报告在“努力办好人民满意的教育”部分，对“大力促进教育公平”提出了明确、具体的要求。其原文表述主要包括以下三点：①合理配置教育资源，重点向农村、边远、贫困、民族地区倾斜。②加快缩小教育差距。③健全家庭经济困难学生资助体系，积极推动农民工子女平等接受教育。 A与报告原文的第一点完全一致，是报告中明确提出的措施。因此，此项属于。 B“提高家庭经济困难学生资助水平”与报告原文中“健全家庭经济困难学生资助体系”的精神完全一致。“支持特殊教育”也是国家一贯的教育政策，并在此背景下得到强调。因此，此项属于。 C与报告原文的第三点完全一致。因此，此项属于。 D“实行公办学校标准化建设和校长教师交流轮岗”是后续国家为了促进教育公平而推出的非常重要且具体的政策，但它是在党的十八届三中全会通过的《中共中央关于全面深化改革若干重大问题的决定》中首次明确提出的。 “不设重点学校、重点班”是我国《义务教育法》中的规定，旨在促进教育均衡。 因此，虽然D选项描述的内容本身是正确的，并且是促进教育公平的有效手段，但它并非党的十八大报告在论述“大力促进教育公平”时直接列出的具体措施。命题人正是用这些后来推出的、广为人知的正确政策来检验考生对原始报告文本掌握的精确度。

## 第 3 题 [单选题]

**题目：** 3. ![图](./images/q_3_125_tit_0.png) 根据法律和国务院的行政法规、决定、命令在部门的职权范围内依法按照《规章制定程序条例》制定的规章被称为（ ）。

**选项：**
```
A.国家法律
B.行政法规
C.国务院部门规章
D.地方性法规
```

> **正确答案：** 未知

**解析：**
国家法律是由全国人大及其常委会制定的基本法律和一般法律，A选项错误。行政法规是国务院制定的，不是国务院有关部门，B选项错误。国务院有关部门根据法律和国务院的行政法规、决定、命令在部门的职权范围内依法按照《规章制定程序条例》制定的规幸被称为国务院部门规幸，C选项正确。地方性法规是地方人大及其常委会制定的，和题干主体不符，D选项错误。故选C。
