# 第一章　社会工作服务相关法规与政策的基本体系和主要功能

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_125_tit_0.png) 有“专家”和（ ）的双重角色，使得他们在相关法规和政策的制定、完善和发展过 程中能发挥重要作用。

**选项：**
```
A.“倡导者”
B. “基层合作者”
C.“基层实践者”
D. “基层创新者”
```

> **正确答案：** C

**解析：**
（考点：社会工作实践对社会工作法规与政策的作用）本题考查社会工作实践对社会工作法规与政策的作用。社会工作者在此过程中扮演双重角色：“专家”角色：指社会工作者基于专业理论、实务经验和对社会问题的深入研究，能够为法规政策的制定、修订提供专业的建议和依据。“基层实践者”角色（C选项）：指社会工作者身处服务一线，直接面对服务对象和各类社会问题，是法规政策的具体执行者和效果体验者。他们能最直接、最真实地反馈政策在基层的执行情况、存在的问题以及服务对象的新需求，从而为政策的完善提供鲜活的实践依据。
