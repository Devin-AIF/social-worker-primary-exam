## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_125_tit_0.png) 从各个方面具体论述了加强和改善民生的政策主张。在教育政策方面，提出了“努力办好人民满意的教育”,并重点强调了要“大力促进教育公平”,具体措施不包括（ &nbsp;&nbsp;&nbsp;）。

**选项：**
```
A.合理配置教育资源，重点向农村、边远、贫困、民族地区倾斜
B.支持特殊教育，提高家庭经济困难学生资助水平
C.积极推动农民工子女平等接受教育
D.实行公办学校标准化建设和校长教师交流轮岗，不设重点学校、重点班
```

> **正确答案：** 未知

**解析：**
党的十八大报告在“努力办好人民满意的教育”部分，对“大力促进教育公平”提出了明确、具体的要求。其原文表述主要包括以下三点：①合理配置教育资源，重点向农村、边远、贫困、民族地区倾斜。②加快缩小教育差距。③健全家庭经济困难学生资助体系，积极推动农民工子女平等接受教育。 A与报告原文的第一点完全一致，是报告中明确提出的措施。因此，此项属于。 B“提高家庭经济困难学生资助水平”与报告原文中“健全家庭经济困难学生资助体系”的精神完全一致。“支持特殊教育”也是国家一贯的教育政策，并在此背景下得到强调。因此，此项属于。 C与报告原文的第三点完全一致。因此，此项属于。 D“实行公办学校标准化建设和校长教师交流轮岗”是后续国家为了促进教育公平而推出的非常重要且具体的政策，但它是在党的十八届三中全会通过的《中共中央关于全面深化改革若干重大问题的决定》中首次明确提出的。 “不设重点学校、重点班”是我国《义务教育法》中的规定，旨在促进教育均衡。 因此，虽然D选项描述的内容本身是正确的，并且是促进教育公平的有效手段，但它并非党的十八大报告在论述“大力促进教育公平”时直接列出的具体措施。命题人正是用这些后来推出的、广为人知的正确政策来检验考生对原始报告文本掌握的精确度。

## 第 4 题 [单选题]

**题目：** 4. ![图](./images/q_4_125_tit_0.png) 、法规和规章的效力的说法不正确的是（ )

**选项：**
```
A. 各种法律 法规和规章具有的效力是不一样的
B.根据立法法规定，宪法具有最高的法律效力，一切法律、行政法规、地方性法规、自治条例和单行条例、规章都不得同宪法相抵触
C. 法律的效力高于行政法规 地方性法规、规章
D. 地方性法规、规章高于所有行政法规的效力
```

> **正确答案：** 为D。

**解析：**
（考点：法律、法规和规章的效力层级） 根据《中华人民共和国立法法》的规定： A项“各种法律、法规和规章具有的效力是不一样的”——正确。宪法、法律、行政法规、地方性法规、规章等不同层级的规范性文件，其法律效力存在差异。 B项“根据立法法规定，宪法具有最高的法律效力，一切法律、行政法规、地方性法规、自治条例和单行条例、规章都不得同宪法相抵触”——正确。这是《立法法》的明确规定。 C项“法律的效力高于行政法规、地方性法规、规章”——正确。《立法法》第九十九条规定：“法律的效力高于行政法规、地方性法规、规章。” D项“地方性法规、规章高于所有行政法规的效力”——错误。《立法法》第九十九条规定：“行政法规的效力高于地方性法规、规章。”行政法规的效力高于地方性法规和规章，而不是相反。 本题要求选择“说法不正确的”，故正确答案为D。

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_125_tit_0.png) 和法律制定的有关行政管理等方面的规范性文件被称为（ ）。

**选项：**
```
A.国家法律
B.行政法规
C.国务院部门规章
D.地方性法规
```

> **正确答案：** 为

**解析：**
根据《中华人民共和国宪法》和《中华人民共和国立法法》： 国家法律（A）：由全国人民代表大会及其常务委员会制定。 行政法规（B）：由国务院根据宪法和法律制定，属于行政管理领域的规范性文件，效力低于法律，但高于地方性法规和规章。 国务院部门规章（C）：由国务院各部委制定，仅在本部门权限内适用。 地方性法规（D）：由省、自治区、直辖市及设区的市人民代表大会及其常务委员会制定，适用于本行政区域。 题目中明确提到“国务院制定的规范性文件”，根据《宪法》第89条，国务院有权制定行政法规。因此正确答案为 B。

**题目：** 9. ![图](./images/q_9_125_tit_0.png) 共和国宪法》和《中华人民共和国立法法》等有关法律的规定，由省、自治区、直辖市和较大的市的人民代表大会及其常务委员会，根据本行政区域的具体情况和实际需要，在不与宪法、法律、行政法规相抵触的前提下制定的规范性文件，以及自治州、自治县的人民代表大会及其常务委员会依照法定职权和程序制定的自治条例和单行条例属（ ）。

**选项：**
```
A.国家法律
B.行政法规
C.国务院部门规章
D.地方性法规
```

> **正确答案：** 未知

**解析：**
根据《中华人民共和国宪法》和《中华人民共和国立法法》的规定： 地方性法规：由省、自治区、直辖市和设区的市（即“较大的市”）人民代表大会及其常务委员会制定，需符合本行政区域的具体情况，且不得与宪法、法律、行政法规相抵触。 自治条例和单行条例：由自治州、自治县人民代表大会依照当地民族的政治、经济和文化特点制定，属于广义的“地方性法规”，但需经省级人大常委会批准后生效。 A国家法律：由全国人大及其常委会制定，效力最高，与题目中地方立法主体无关。B行政法规：由国务院制定，属于中央层面的规范性文件。C国务院部门规章：由国务院各部委制定，仅适用于特定行政管理领域。D地方性法规：完全符合题目中“省、市人大及其常委会制定的规范性文件”及“自治条例、单行条例”的定义。

**选项：**
```
A.仅参加高级社会工作师考试
B.仅参加高级社会工作师评审
C.先通过考试，再参加评审

```

> **正确答案：** 为C。

**解析：**
根据《高级社会工作师评价办法》规定：“高级社会工作师实行考试和评审相结合的评价制度。参加考试合格并通过评审，方可取得高级社会工作师资格。” 这意味着申报高级社会工作师的评价流程分为两个步骤： 第一步：参加并通过全国统一的高级社会工作师考试。 第二步：在考试合格有效期内，参加并通过高级社会工作师评审。 各选项分析： A选项（仅参加高级社会工作师考试） ：错误。仅通过考试不能取得资格，必须通过评审。 B选项（仅参加高级社会工作师评审） ：错误。必须先通过考试才有资格参加评审。 C选项（先通过考试，再参加评审） ：正确。符合“考试+评审”相结合的制度安排。 D选项（先参加评审，再通过考试） ：错误。评审资格以考试合格为前提，顺序不可颠倒。 因此，正确答案为C。

## 第 22 题 [单选题]

**题目：** 22. ![图](./images/q_22_125_tit_0.png) 李在开展未成年人服务时，发现服务对象小明(15岁)因家庭矛盾离家出走，且存在被不良社会人员引诱的风险。根据《中华人民共和国未成年人保护法》,小李最恰当的做法是（ &nbsp;&nbsp;&nbsp;）。

```
A.联系小明的父母，要求其尽快接回孩子
B.直接将小明送往当地未成年人救助保护机构
C.为小明提供临时庇护场所，并及时向相关部门报告
D.对小明进行心理疏导后，让其自行回家
```

> **正确答案：** 未知

**解析：**
本题考查《未成年人保护法》中关于发现未成年人处于危险状态时的处置规定。根据法律规定，任何组织或者个人发现未成年人身心健康受到侵害、疑似受到侵害或者面临其他危险情形的，应当立即向公安、民政、教育等有关部门报告，并根据情况采取必要的临时庇护措施。各选项分析如下： A选项错误：仅联系父母接回孩子，无法消除小明被不良社会人员引诱的风险，且小明是因家庭矛盾离家出走，直接让父母接回可能激化家庭矛盾，无法保障其安全，不符合法律规定的“保障未成年人安全”的首要原则。 B选项错误：直接送往救助保护机构，未履行报告程序，且未先提供临时庇护，不符合处置流程。 C选项正确：为小明提供临时庇护场所，保障其当前安全，同时及时向相关部门报告，由相关部门后续介入处理家庭矛盾和风险问题，符合法律规定的处置要求。 D选项错误：让其自行回家，完全没有消除其面临的被引诱风险，也未解决家庭矛盾，可能导致其再次离家出走，不符合未成年人保护的要求。

## 第 24 题 [单选题]

**题目：** 24. ![图](./images/q_24_125_tit_0.png) 政策与人的需要的说法不正确的是（ ）。

**选项：**
```
A. 在当代社会中，政府制定各种社会政策的基本目标之一是为了满足社会需要
B. 社会需要是指满足社会运行及发展所需要的各种条件，它不包括人的需要
C. 当代社会中，政府制定和实施各项社会政策首先是为了满足人们的基本需要
D.人的需要是指人作为生物有机体和社会存在物为维持生存、发展、享受和从事各种社会活动所产生的对各种外部条件的依赖及获得这些条件的主观愿望
```

> **正确答案：** 未知

**解析：**
（考点：社会政策体系一社会政策）社会政策的基本目标首先是满足人们的基本需要。 A正确。社会政策的重要目标包括解决社会问题、促进社会公平、满足社会整体运行和发展的需要。 B不正确。社会需要与人的需要并非完全割裂。社会需要通常包含了社会成员普遍的基本需要（如安全、教育、健康），这些需要的满足是社会稳定和发展的基础。因此，社会需要与人的需要是相互关联的，社会需要本身就源于并包含了人的基本需要。该选项将二者完全对立，是错误的。 C正确。满足人们的基本生活需要（如衣食住行、教育、医疗等）是社会政策最直接和首要的目标。 D是对“人的需要”这一概念的准确定义，正确。
