# 第一章　社会工作服务相关法规与政策的基本体系和主要功能

## 第 7 题 [单选题]

**题目：** 7. ![图](./images/q_7_125_tit_0.png) 和法律制定的有关行政管理等方面的规范性文件被称为（ ）。

**选项：**
```
A.国家法律
B.行政法规
C.国务院部门规章
D.地方性法规
```

> **正确答案：** 为

**解析：**
根据《中华人民共和国宪法》和《中华人民共和国立法法》： 国家法律（A）：由全国人民代表大会及其常务委员会制定。 行政法规（B）：由国务院根据宪法和法律制定，属于行政管理领域的规范性文件，效力低于法律，但高于地方性法规和规章。 国务院部门规章（C）：由国务院各部委制定，仅在本部门权限内适用。 地方性法规（D）：由省、自治区、直辖市及设区的市人民代表大会及其常务委员会制定，适用于本行政区域。 题目中明确提到“国务院制定的规范性文件”，根据《宪法》第89条，国务院有权制定行政法规。因此正确答案为 B。

## 第 8 题 [单选题]

**题目：** 8. ![图](./images/q_8_125_tit_0.png)  ）的行动体系

**选项：**
```
A.规范化
B. 制度化
C. 原则化
D. 系统化
```

> **正确答案：** B

**解析：**
社会政策是制度化的行动体系。这是其最核心的特征。“制度化”意味着社会政策不是临时、随意的措施，而是通过法律法规、政府规章、组织程序等固定下来的一套长期、稳定、有约束力的行动规则和资源分配方式。它确保了政策的持续性、权威性和可预期性。

## 第 9 题 [单选题]

**题目：** 9. ![图](./images/q_9_125_tit_0.png) 共和国宪法》和《中华人民共和国立法法》等有关法律的规定，由省、自治区、直辖市和较大的市的人民代表大会及其常务委员会，根据本行政区域的具体情况和实际需要，在不与宪法、法律、行政法规相抵触的前提下制定的规范性文件，以及自治州、自治县的人民代表大会及其常务委员会依照法定职权和程序制定的自治条例和单行条例属（ ）。

**选项：**
```
A.国家法律
B.行政法规
C.国务院部门规章
D.地方性法规
```

> **正确答案：** D

**解析：**
根据《中华人民共和国宪法》和《中华人民共和国立法法》的规定： 地方性法规：由省、自治区、直辖市和设区的市（即“较大的市”）人民代表大会及其常务委员会制定，需符合本行政区域的具体情况，且不得与宪法、法律、行政法规相抵触。 自治条例和单行条例：由自治州、自治县人民代表大会依照当地民族的政治、经济和文化特点制定，属于广义的“地方性法规”，但需经省级人大常委会批准后生效。 A国家法律：由全国人大及其常委会制定，效力最高，与题目中地方立法主体无关。B行政法规：由国务院制定，属于中央层面的规范性文件。C国务院部门规章：由国务院各部委制定，仅适用于特定行政管理领域。D地方性法规：完全符合题目中“省、市人大及其常委会制定的规范性文件”及“自治条例、单行条例”的定义。

## 第 10 题 [单选题]

**题目：** 10. ![图](./images/q_10_125_tit_0.png) 效力的排列正确的是（ ）

**选项：**
```
A. 宪法＞法律＞行政法规＞地方性法规
B. 宪法＞行政法规〉地方性法规＞法律
C. 宪法＞地方性法规＞行政法规＞法律
D. 法律＞宪法＞行政法规＞地方性法规
```

> **正确答案：** A

**解析：**
（考点：社会工作法规体系一一一法规的制定过程）宪法是国家根本大法，具有最高法律效力；其次是全国人民代表大会及其常委会制定的法律；然后是国务院制定的行政法规；最后是地方人民代表大会及其常委会制定的地方性法规。您的总结混淆了顺序，正确排序下A选项符合法律规定。
