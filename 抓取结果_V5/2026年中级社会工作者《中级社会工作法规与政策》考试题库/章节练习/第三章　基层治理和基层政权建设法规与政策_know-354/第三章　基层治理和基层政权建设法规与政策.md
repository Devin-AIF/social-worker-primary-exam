# 第三章　基层治理和基层政权建设法规与政策

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_130_tit_0.png) 参加本区居民委员会的是（）。

**选项：**
```
A.居委会所在地企业
B.户籍不在本居委会，但居住在本区内的外地人
C.户籍在本居委会，但常住地不在本区的人
D.在本地居住的军人和随军家属
```

> **正确答案：** A

**解析：**
（考点：居民委员会与社区社会组织的关系）《居民委员会组织法》第十九条规定：“机关、团体、部队、企业事业组织，不参加所在地的居民委员会…”

## 第 2 题 [单选题]

**题目：** 2. ![图](./images/q_2_130_tit_0.png) 的决策机构的是（）。

**选项：**
```
A.村民（代表）会议
B.村民委员会
C.居民会议
D.村党支部委员会
```

> **正确答案：** A

**解析：**
根据《中华人民共和国村民委员会组织法》第二条，村民委员会是村民自我管理、自我教育、自我服务的基层群众性自治组织。而村民会议（或村民代表会议）是村民自治的最高决策形式，村民委员会向村民会议、村民代表会议负责并报告工作。在法规考试中，通常将村民（代表）会议定性为村民自治的决策机构或决策组织形式。 A选项（村民（代表）会议）：属于村民自治的决策机构，正确。 B选项（村民委员会）：属于村民自治的执行机构，负责日常管理和执行村民会议的决策，不是决策机构，错误。 C选项（居民会议）：属于城市居民自治范畴，与村民自治决策机构无关，错误。 D选项（村党支部委员会）：属于基层党组织，领导村民自治工作，但不是村民自治的决策机构，错误。 因此，正确答案为A。
