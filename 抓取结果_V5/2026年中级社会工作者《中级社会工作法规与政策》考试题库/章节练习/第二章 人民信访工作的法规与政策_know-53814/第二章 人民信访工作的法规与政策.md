# 第二章 人民信访工作的法规与政策

## 第 1 题 [单选题]

**题目：** 1. ![图](./images/q_1_47_tit_0.png) 到可能造成社会影响的重大、紧急信访信息，根据规定，该部门应首先（ &nbsp;&nbsp;&nbsp;）。

**选项：**
```
A.仅报告省委和政府
B.仅报告国家信访局
C.向上一级信访部门报告，同时报告国家信访局
D.先自行处理再报告
```

> **正确答案：** C

**解析：**
根据《信访工作条例》第二十五条规定，各级机关、单位发现可能造成社会影响的重大、紧急信访事项和信访信息时，应当及时报告本级党委和政府，通报相关主管部门和本级信访工作联席会议办公室，在职责范围内依法及时采取措施，防止不良影响的产生、扩大。地方各级党委和政府信访部门接到重大、紧急信访事项和信访信息，应当向上一级信访部门报告，同时报告国家信访局。 各选项分析： 选项A：仅报告省委和政府不符合规定，还需向上一级信访部门及国家信访局报告，因此A错误。 选项B：仅报告国家信访局不符合层级报告要求，还需向上一级信访部门报告，因此B错误。 选项C：省级信访部门接到重大紧急信访信息后，向上一级信访部门报告同时报国家信访局，完全符合条例规定，因此C正确。 选项D：先自行处理再报告违反重大紧急信息“先报告、边处置边报告”的原则，可能导致风险扩大，因此D错误。
