# 题库自动化抓取工具 V5 (全能增强版)

该工具基于 Playwright 开发，能够全自动遍历题库系统并生成高质量的 Markdown 题库。
**当前默认版本：`crawler_v5.js`**，该版本融合了控制台脚本的高效抓取逻辑，并支持完整的图片本地化。

## 使用步骤

1. **安装 Node.js**: 确保您的电脑上已安装 Node.js。
2. **安装依赖**:
   在当前目录下打开终端，运行：
   ```bash
   npm install
   npx playwright install chromium
   ```
3. **运行脚本**:
   ```bash
   npm start
   ```
4. **手动登录**:
   - 脚本会自动打开一个 Chromium 浏览器窗口。
   - 请在该窗口中手动完成网站登录。
   - **登录完成后**，回到终端按一下 **回车键**，脚本将开始全自动抓取。

## V5 版本核心特性
- **深度内容清洗**: 自动移除页面上的干扰按钮（纠错、讨论等）和垃圾文案。
- **图片全自动本地化**: 自动下载所有题目图片（支持 Base64 格式），保存至本地 `images` 目录并使用相对路径引用。
- **智能解析拆分**: 精准识别答案与解析混合的容器，自动拆分为独立的“正确答案”和“解析”字段。
- **排版格式对标**: 输出的 Markdown 格式经过优化，与官方参考文件保持高度一致。
- **断点续爬与监控**: 支持自动跳过已完成题目，并具备实时健康状态监控逻辑。

## 注意事项
- 抓取结果保存在 `抓取结果_V5` 目录下。
- 脚本内含防屏蔽延迟，请勿随意大幅缩短等待时间。
- 抓取过程中请勿关闭浏览器窗口。
