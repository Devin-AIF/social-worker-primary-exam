# 题库自动化抓取工具 V4

该工具基于 Playwright 开发，可以自动遍历题库的多个层级（题型 -> 章节/试卷 -> 题目），并按照文件夹结构自动保存为 Markdown 文件。
当前默认入口为 `crawler_v4.js`，输出目录为 `抓取结果_V4`。

## 使用步骤

1. **安装 Node.js**: 确保您的电脑上已安装 Node.js。
2. **安装依赖**:
   在当前目录下打开终端，运行：
   ```bash
   npm install
   npx playwright install chromium
   ```
3. **运行脚本**:
   ```bash
   npm start
   ```
4. **手动登录**:
   - 脚本会自动打开一个 Chromium 浏览器窗口。
   - 请在该窗口中手动完成网站登录。
   - **登录完成后**，回到终端按一下 **回车键**，脚本将开始全自动抓取。

## 功能特点
- **全自动导航**: 自动从第一级（题型选择）开始，遍历所有链接。
- **自动分类**: 按照 `抓取结果_V4/题型/章节名/章节名.md` 的结构保存。
- **模拟交互**: 自动点击选项以触发显示正确答案和解析。
- **断点续爬**: 已抓取题目会自动续跑，不会在重启后从头覆盖。

## 注意事项
- 请保持网络通畅。
- 脚本中设置了合理的延迟（sleep），以避免被服务器封禁。如果觉得太慢或太快，可以修改 `crawler_v4.js` 中的等待参数。
- 抓取过程中请勿关闭浏览器窗口。
