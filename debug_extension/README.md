# Scraper Debugger Helper 插件安装指南

为了帮助我更精准地定位抓取脚本在网站（xs507.com）上遇到的问题，请按照以下步骤安装并使用这个调试插件：

## 安装步骤

1. **打开 Chrome 浏览器**。
2. 在地址栏输入 `chrome://extensions/` 并回车。
3. 在页面右上角，找到并**开启“开发者模式”** (Developer mode) 开关。
4. 点击左上角的**“加载已解压的扩展程序”** (Load unpacked)。
5. 在弹出的文件选择框中，定位到以下目录并点击“选择”：
   `/Users/devin_aif/Downloads/抓取题目/debug_extension`
6. 看到“Scraper Debugger Helper”出现在插件列表中，即表示安装成功。

## 使用说明

1. **进入目标网页**：在 Chrome 中打开你正在抓取的题目页面（例如 `xs507.com` 的做题界面）。
2. **刷新页面**：安装插件后，请刷新一次当前页面以确保调试脚本已加载。
3. **捕获状态**：
   - 点击浏览器工具栏上的插件图标（可能在拼图图标里）。
   - 点击**“捕获当前页面状态并复制报告”**按钮（用于快速粘贴）。
   - 或者点击**“导出调试报告 (JSON)”**按钮（用于生成文件）。
4. **发送报告**：
   - 如果是复制的报告，请直接在此粘贴发给我。
   - 如果导出了 JSON 文件，请将该文件上传给我，我会基于文件内容进行深度排查。

## 报告包含的内容
- **Page Data**: 包含题目标题、选项、解析等关键 DOM 元素的文本、HTML 片段及可见性。
- **Logs**: 包含页面最近的 `console.log`、`console.error` 以及 JavaScript 运行错误。
- **Masks**: 检查是否存在阻碍抓取的遮罩层。
