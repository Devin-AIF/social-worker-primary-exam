# 社会工作模拟练习系统 - 使用与维护指南

本系统是基于 AIT3 架构定制开发的单机版模拟练习平台。

## 1. 快速开始
- **打开系统**：直接双击 `index.html` 即可在浏览器中打开。
- **推荐浏览器**：Chrome 或 Edge。

## 2. 核心功能
- **理论练习**：支持顺序出题、随机出题和模拟组卷。
- **章节筛选**：顶部的“章节筛选”下拉框可以帮助您进行专项练习。
- **模拟考试**：模拟组卷模式下，单选题 1分/题，多选题 2分/题，满分 100分。
- **错题/收藏**：系统会自动记录您的错题并支持收藏重点题目。

## 3. 如何同步与更新题库（重点）
为了实现“修改 MD 即刷新”的零成本体验，我为您提供了一个**实时模拟服务器**：

1. **启动实时服务器**：
   在终端（Terminal）中进入项目文件夹，运行：
   ```bash
   python3 run_simulator.py
   ```
2. **访问系统**：
   打开浏览器，访问：`http://localhost:8000`
3. **实现实时同步**：
   - 您现在可以随时修改 `source_data/` 下的 MD 文件或图片。
   - 修改保存后，**只需刷新浏览器页面**，服务器就会自动重新解析 MD 并加载最新题目。您不再需要手动执行任何转换脚本。

*(如果您不想运行服务器，依然可以双击 `index.html`，但每次修改后需手动运行 `python3 build_data.py`。建议使用服务器模式。)*

## 4. 文件夹结构说明
- `source_data/`: 存放原始 Markdown 题目文件（您的“源头”文件）。
- `js/questions.js`: 自动生成的题库文件（请勿手动修改）。
- `data/images/`: 自动归集的题目图片库。
- `build_data.py`: 负责将 Markdown 转换为系统数据的“心脏”。
