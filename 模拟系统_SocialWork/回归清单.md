## 2. 理论模块

### 2.1 顺序练习

- [ ] 打开“理论练习”，确认默认可正常显示题目、选项和题号面板
- [ ] 连续作答 3 题，确认进度、题号颜色、正确/错误统计、正确率同步变化
- [ ] 在多选题中测试“选择 -> 取消 -> 重新选择”，确认按钮状态正常

### 2.2 随机出题

- [ ] 切换到“随机出题”，确认已答题不会再次进入当前未答池
- [ ] 若多数题已答，确认随机池变小属于预期，不是空白崩溃
- [ ] 随机模式切回顺序模式后，理论视图滚动位置可恢复

### 2.3 模拟组卷

- [ ] 切换到“模拟组卷（100题）”，确认自动进入考试态且相关控件被正确约束
- [ ] 作答若干题后提前交卷，确认分数、已答数、正确数、错误数正常
- [ ] 关闭成绩弹窗后，确认可以继续复盘本卷，不会错误回退为普通随机练习
- [ ] 再次重新组卷，确认能生成新一轮完整题卷

### 2.4 错题本与收藏夹

- [ ] 制造至少 1 道错题，确认错题本新增记录
- [ ] 收藏至少 1 道题，确认收藏夹新增记录
- [ ] 从错题本进入重练，确认进度条显示当前练习池口径
- [ ] 从收藏夹进入专项练习，确认进度条显示当前练习池口径
- [ ] 在“练习模式 / 错题本 / 收藏夹”之间切换，确认各自滚动位置都能恢复

### 2.5 导入导出

- [ ] 点击“导出”，确认可正常生成 JSON 备份
- [ ] 重新导入备份，确认错题、收藏、理论已答记录、实操记录都能恢复
- [ ] 导入后随机题池与进度统计仍然正常

## 3. 实操模块

### 3.1 题号跳转与高亮

- [ ] 进入“实操考核”，点击左侧任意小题，确认页面直接跳转，无蓝色平滑移动痕迹
- [ ] 连续点击多个不同题号，确认高亮瞬时更新且不会残留旧状态
- [ ] 滚动浏览实操内容，确认侧栏高亮不会在相邻题号间反复横跳

### 3.2 位置记忆

- [ ] 在实操套卷 1 中滚动到中后段，切回“理论练习”再切回来，确认恢复到原来的精确滚动位置
- [ ] 在实操套卷 1 与实操套卷 2 之间切换，确认各自都能恢复上次位置
- [ ] 在“理论练习 / 错题本 / 收藏夹 / 实操考核”之间来回切换，确认各视图滚动位置互不覆盖

### 3.3 文本、截图、填空

- [ ] 在 `text` 题中输入内容，失焦后确认预览正常，再次点击可回到编辑态
- [ ] 在 `upload` 或 `text_upload` 题中上传截图，确认预览可见并能保留
- [ ] 在代码填空题输入答案后刷新页面，确认填空内容能恢复
- [ ] 点击“核对填空与查看解答”，确认展开期间当前题高亮不会乱跳

### 3.4 实操评估

- [ ] 完成若干题后点击“交卷并进行实操评估”，确认成绩弹窗可正常显示
- [ ] 关闭评估弹窗后，页面位置与题号高亮不会异常跳转
