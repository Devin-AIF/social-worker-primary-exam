# AIT3 项目结构文档

---

## 1. 项目概览

- **项目定位**：基于浏览器本地存储的 AI 智能训练模拟系统，包含理论练习与实操考核双模块。
- **技术栈**：HTML5 + CSS3 + Vanilla JavaScript。
- **数据持久化方式**：浏览器 `localStorage`。
- **当前版本**：**v1.21**（2026-04-22）。
- **缓存标记**：资源版本号已同步升级为 `?v=21`。

---

## 3. 关键文件职责

### index.html

- 页面骨架、左右分栏结构、帮助弹窗和内嵌更新日志。
- 引用 `css/` 与 `js/` 下核心资源。
- 负责页面版本号和资源缓存版本同步。

### js/app.js

- 管理理论模块的核心状态：
  - 当前题号
  - 错题本
  - 收藏夹
  - 已答映射
  - 模拟组卷
  - 交卷统计
- 当前版本重点逻辑：
  - 随机模式未答优先
  - 模拟组卷未答优先且自动补足
  - 交卷结果回写 `answeredMap`
  - 全局进度与当前练习池进度分口径显示

### js/handson.js

- 负责实操题渲染与交互：
  - 套卷切换
  - 题号导航
  - 文本作答
  - 截图上传
  - 代码填空
  - 成绩评估
- 当前版本重点逻辑：
  - 套卷级位置记忆
  - 金轴判定式侧栏高亮追踪
  - 展开答案时的临时锁定
  - 直接跳转而非平滑轨迹滚动

### js/questions.js / js/handsonData.js

- 静态题库数据源。
- 属于内容资产文件，维护时应优先关注：
  - ID 唯一性
  - 字段结构完整性
  - 图片路径有效性

## 4. 模块关系

### 理论模块

- 视图：练习、错题本、收藏夹
- 关键能力：顺序/随机/模拟、答题统计、导入导出

### 实操模块

- 视图：套卷内容区 + 左侧题号导航
- 关键能力：位置记忆、文本与截图保存、代码填空核对、实操评估

### 全局模块

- 主题切换
- 倒计时
- 帮助弹窗
- 更新日志弹窗
- 本地数据导入导出
