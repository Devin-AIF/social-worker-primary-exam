**选项：**
```
A.司法行政部门
B.人力资源和社会保障部门
C.公安部门
D.教育部门
```

> **正确答案：** D

**解析：**

## 第 5 题 [单选题]

**题目：** 5. ![图](./images/q_5_48_0.png) 提供专业咨询服务是社会工作者的一项重要工作，下列选项中不属于这项工作范畴的是（    ）。

**选项：**
```
A.家庭走访
B.心理、情绪辅导
C.职业技能训练
D.公民教育
```

> **正确答案：** A

**解析：**
本题考查司法社会工作中监狱服务的内容范畴，为监狱服刑人员提供的专业咨询服务是指在监狱场所内，针对服刑人员的认知、心理、能力提升等需求开展的专业服务。
选项A：家庭走访属于出狱后的社会融入服务或矫正前期的社会调查工作，不属于在监狱内为服刑人员提供的咨询服务范畴，符合题意，正确。
选项B：心理、情绪辅导是监狱服刑人员咨询服务的核心内容，帮助服刑人员疏导负面情绪、矫正认知偏差，属于服务范畴，排除。
选项C：职业技能训练是提升服刑人员出狱后就业能力的重要咨询服务内容，属于服务范畴，排除。
选项D：公民教育是帮助服刑人员树立法治观念、培养公民意识的重要咨询服务内容，属于服务范畴，排除。

## 第 10 题 [单选题]

**题目：** 10.16 ![图](./images/q_10_48_0.png) 拿父母的钱用于游戏充值，父母发现后求助于社会工作者，希望社会工作者能够介入提供帮助。社会工作者在与明明父母交谈后，制定了一系列帮助明明改正不良行为的计划。上述工作体现了司法社会工作服务的（    ）功能。

**选项：**
```
A.服务
B.监管
C.矫正
D.预防
```

> **正确答案：** D

**解析：**
本题考查司法社会工作的功能，正确答案为D。

选项A错误：服务功能主要是指为服务对象提供生活帮扶、心理支持、资源链接等基础性服务，本题的核心目标是帮助明明改正偷拿钱财的不良行为，避免后续出现更严重的违法犯罪行为，不属于一般性服务功能。选项B错误：监管功能主要针对涉法涉诉的服务对象，比如社区矫正对象、刑满释放人员等，通过监督、管理规范其行为，避免其再次违法，明明尚未涉及司法程序，因此不适用监管功能。选项C错误：矫正功能针对的是已经存在违法犯罪行为，或者已经形成严重偏差行为的服务对象，通过干预纠正其已经存在的违法或偏差行为。本题中明明虽然多次偷拿父母钱财，但属于未成年人的不良行为，尚未达到需要司法矫正的程度，社工介入的核心是避免其行为进一步恶化，而非矫正已有的违法犯罪行为。选项D正确：预防功能是指通过提前干预，识别服务对象的潜在风险，避免其从一般不良行为发展为违法犯罪行为。明明的偷钱行为属于不良行为，社工及时介入制定改正计划，目的是防止其后续出现更严重的违法问题，符合预防功能的定义。

## 第 16 题 [单选题]

**题目：** 16. ![图](./images/q_16_48_0.png) 受人民法院委托撰写服务对象李某的审前调查报告。该报告的主要内容如下：“李某，2018年6月18日以欠债不还为由，将被害人张某扣留于一辆商务车内。两天后民警介入，张某获救。李某以前没有犯罪记录，此案件主要因经济纠纷而致。”结合审前调查报告的相关要求，上述报告还应补充的信息是（    ）。

**选项：**
```
A.小王的工作简历
B.非法拘禁的认定标准
C.李某的现实生活状况
D.社区矫正的帮教建议
```

> **正确答案：** C

**解析：**
审前调查报告（社区矫正调查评估报告）的核心内容包括三个部分：一是犯罪嫌疑人/被告人的个人情况，包括基本信息、成长经历、家庭环境、社会关系、一贯表现、犯罪行为的后果和影响、居住地村（居）民委员会和被害人意见等；二是对其再犯罪风险的评估；三是是否适合社区矫正的建议。题干中已有的内容仅包括犯罪事实和前科情况，缺少对李某现实生活状况的调查，这是评估其社会危险性和矫正条件的重要依据。

选项A与报告内容无关，选项B是法律适用问题不属于报告内容，选项D社区矫正帮教建议是在判决后制定的，不属于审前调查报告的必需内容。

## 第 29 题 [单选题]

**题目：** 29. ![图](./images/q_29_48_0.png) 在与社区矫正对象小夏的例行会谈中得知，小夏过去的“兄弟”来找他，拉他参与对一个团伙的报复行动。尽管小夏不想再次违法，但他迫于压力不得已答应了“兄弟”的要求。对此，大江恰当的做法是（    ）。

**选项：**
```
A.要求小夏反思自己的错误行为
B.妥善处理保密与社会安全的关系
C.鼓励小夏自行决定是否参与行动
D.建议小夏改变犹豫不决的个性
```

> **正确答案：** B

**解析：**
本题考查社区矫正社会工作中的伦理困境处理，核心考点为保密原则的例外情形。面对社区矫正的特殊工作对象，开展个案工作时需要注意的一个核心问题，就是妥善处理为案主保密的原则与维护社会安全的关系。社区矫正工作的对象是被判处刑罚的罪犯，他们的思想和行为及社会交往往往隐含着对他人和社会造成新的危害的可能性。
选项A：要求小夏反思自己的错误行为。依据非评判原则，社会工作者不应随意评价、指责和批判服务对象的行为和价值观。该做法违背了非评判原则，错误。
选项B：妥善处理保密与社会安全的关系。社会工作者本应对小夏的信息保守秘密，但小夏已答应参与报复行动，面临再次违法的重大风险，这直接涉及社会安全问题。当服务对象的行为有可能伤及其自己和他人的生命安全，或服务对象的行为触犯法律红线时，社会工作者要打破保密原则。常见的保密例外情况包括服务对象想杀害他人、服务对象做了违法的事情等。此时社工不能借口保密而不闻不问，必须妥善处理两者关系。正确。
选项C：鼓励小夏自行决定是否参与行动。依据服务对象自决原则，社会工作者在介入过程中一般需要由服务对象自己做出行动选择。但教材明确指出，除非有明显的证据证明，服务对象的行动选择会给自己或者他人带来危险，此时社会工作者才可以不由服务对象自决。 小夏准备参与的是违法犯罪行动，已对他人和社会构成现实威胁，不属于服务对象自决原则的适用场景，不能鼓励其自行决定。错误。
选项D：建议小夏改变犹豫不决的个性。这属于对服务对象个人的评价和标签化，违背了社会工作的非评判原则。同时，该做法也未触及小夏面临的核心伦理困境。错误。

## 第 45 题 [多选题]

**题目：** 13. ![图](./images/q_45_48_0.png) 因一次意外构成了犯罪行为，入狱后他感觉不适应，觉得十分丢脸，同时难以接受自己入狱这个事实，每日郁郁寡欢，不与人交流。社会工作者小齐了解这件事后对小旭进行了心理疏导，并且帮助他重新建立与他人的联系，每天也会与小旭一起读报纸，使小旭能更多了解监狱外面的世界。本案例中，小齐主要开展了哪些工作?（    ）

**选项：**
```
A.帮助服务对象熟悉监禁环境
B.协助服务对象戒除不健康的生活习惯
C.进行心理和情绪辅导
D.帮助服务对象了解外面社会的变化
E.帮助服务对象构建支持性社会网络
```

> **正确答案：** ACD

**解析：**
本题考查监禁场所中的社会工作服务内容，答案为ACD。

选项A正确：小旭刚入狱时感觉不适应，社工通过开展心理疏导、帮助他建立与他人的联系、共同读报纸等服务，本质上是帮助他逐步适应监禁场所的生活环境，减少陌生感和抵触情绪，因此体现了帮助其熟悉监禁环境的工作。选项B错误：题干中未提及小旭存在酗酒、赌博、吸毒等不健康的生活习惯，也没有提到社工帮助其戒除相关习惯的内容，属于无中生有。选项C正确：社工针对小旭郁郁寡欢、难以接受入狱事实的情绪状态进行心理疏导，直接体现了心理和情绪辅导的工作内容。选项D正确：社工每天与小旭一起读报纸，让他了解监狱外面的世界，直接体现了帮助服务对象了解外面社会变化的工作内容。选项E错误：题干中仅提到社工帮助小旭重新建立与他人的联系，未涉及链接其家人、朋友、社区等外部支持资源，构建支持性社会网络的范围更广泛，题干内容不足以体现该工作。
