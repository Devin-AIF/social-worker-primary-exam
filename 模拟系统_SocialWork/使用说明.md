# AI智能训练师·三级模拟系统 · 使用说明
**当前版本：v1.22（2026-04-25）**

## 一、快速开始

双击 [index.html](/Users/devin_aif/Downloads/22-提升/02-AI智训/AIT3/index.html) 即可在浏览器中打开系统。  
推荐使用 Chrome 或 Edge，以获得更稳定的本地缓存与交互体验。

---

## 二、系统概览

系统由两部分组成：

1. **理论练习（390 题）**
   支持顺序、随机、模拟组卷、错题重练、收藏专项、考试交卷与统计复盘。
2. **实操考核（多套套卷）**
   支持文本作答、截图上传、代码填空核对、参考答案展开和实操评估。

### v1.22 本次重点变化

- **题库扩充至 390 题**：新增 27 道题目（17 道单选题 + 10 道多选题），全部附带详细解析。

## 三、理论练习功能

| 功能 | 说明 |
|------|------|
| 练习模式 | 即时反馈对错，题目答完后自动锁定 |
| 考试模式 | 不显示即时反馈，交卷后统一评分 |
| 顺序出题 | 按题库顺序逐题练习 |
| 随机出题 | 优先抽取未答题进行练习 |
| 模拟组卷 | 按真实比例抽取 100 题并进入考试态 |
| 题型筛选 | 按单选/多选/判断专项练习 |
| 错题本 | 自动沉淀错误题目并支持重练 |
| 收藏夹 | 收藏重点题并单独练习 |
| 进度统计 | 显示当前进度、全库正确/错误数、正确率 |
| 导入/导出 | 备份与恢复错题、收藏、实操进度 |

### 题号面板颜色说明

| 颜色 | 含义 |
|------|------|
| 灰色 | 未作答 |
| 蓝色 | 当前题 |
| 绿色 | 已答且正确 |
| 红色 | 已答且错误 |

### 理论模块使用建议

- 想系统刷完一遍时，用“顺序出题 + 练习模式”。
- 想专门查漏补缺时，用“随机出题”，此时系统会优先避开已答题。
- 想模拟考试时，用“模拟组卷”，交卷后可直接查看本卷复盘结果。

## 四、实操考核功能

进入“💻 实操考核”后，你会看到独立的套卷和题号导航。

### 支持的交互

- **套卷切换**：可在“实操练习题 1 / 2”之间切换。
- **位置记忆**：切走再回来时，会尽量恢复到上次停留的小题。
- **文本作答**：输入内容自动保存，失焦后会转为 Markdown 预览。
- **截图上传**：可为需要证明材料的题目上传并预览本地截图。
- **代码填空**：在代码块中直接输入答案，点击“核对填空与查看解答”后校验正误。
- **参考答案**：支持展开答案说明、图片参考和填空标准答案。

### 实操模块使用建议

- 点击左侧题号可直接跳到对应小题。
- 若在某道题上展开答案或核对填空，侧栏高亮会锁定在当前题，避免跳位。
- 完成整套题后，可点击底部“交卷并进行实操评估”生成成绩单。

## 五、数据保存与备份

系统数据保存在当前浏览器的 `localStorage` 中，包括：

- 理论题作答记录
- 错题本
- 收藏夹
- 实操作答文本
- 实操截图
- 实操填空输入

如果需要迁移到其他电脑：

1. 点击“💾 导出”生成备份 JSON。
2. 在新环境打开系统后点击“📂 导入”恢复数据。

如果只是想重新开始，可使用“🗑 重置”清空本地记录。

## 六、常见问题

### Q1：为什么随机模式有时题目数量变少？
因为 v1.21 开始，随机模式默认优先只出未答题。如果你已经做了很多题，可随机池本来就会变小，这是预期行为。

### Q2：模拟卷为什么还能抽到旧题？
模拟组卷会优先抽未答题，但如果某一题型未答题数量不足，系统会自动从旧题中回补，以保证卷面完整。

### Q3：为什么交卷后统计和之前不一样了？
因为现在交卷结果会正式写入学习记录，所以进度、正确率和随机题池过滤都以真实作答结果为准，统计更一致。

### Q4：关闭浏览器后数据还在吗？
在。只要不手动清空浏览器缓存或点“重置”，数据会保留在本地。
